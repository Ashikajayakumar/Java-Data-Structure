## 📌 What is Space Complexity?

Space complexity means the **total memory used by an algorithm** based on input size **n**.

👉 It includes:

* Input space
* Auxiliary space (extra memory used in code)

---

# 📊 Common Space Complexities with Java Examples

---

## 1️⃣ O(1) → Constant Space

* Uses fixed memory, no extra space

📌 Example:

```java
public class ConstantSpace {
    public static void main(String[] args) {
        int a = 10;
        int b = 20;

        int sum = a + b; // only fixed variables used

        System.out.println(sum);
    }
}
```

---

## 2️⃣ O(n) → Linear Space

* Extra memory increases with input size

📌 Example: Using array

```java
public class LinearSpace {
    public static void main(String[] args) {
        int n = 5;
        int[] arr = new int[n]; // space grows with n

        for (int i = 0; i < n; i++) {
            arr[i] = i * 2;
        }

        System.out.println(arr[3]);
    }
}
```

---

## 3️⃣ O(n) → Recursion Stack Space

* Each recursive call uses stack memory

📌 Example: Factorial

```java
public class RecursionSpace {

    public static int fact(int n) {
        if (n == 0)
            return 1;

        return n * fact(n - 1); // stack space O(n)
    }

    public static void main(String[] args) {
        System.out.println(fact(5));
    }
}
```

---

## 4️⃣ O(log n) → Logarithmic Space

* Recursive calls reduce problem size

📌 Example: Binary Search (recursive)

```java
public class BinarySearchSpace {

    public static int search(int[] arr, int low, int high, int key) {
        if (low > high)
            return -1;

        int mid = (low + high) / 2;

        if (arr[mid] == key)
            return mid;
        else if (arr[mid] > key)
            return search(arr, low, mid - 1, key);
        else
            return search(arr, mid + 1, high, key);
    }
}
```

---

## 5️⃣ O(n²) Space (Rare but possible)

* 2D structures

📌 Example:

```java
public class MatrixSpace {
    public static void main(String[] args) {
        int n = 3;
        int[][] matrix = new int[n][n]; // n² space

        matrix[0][0] = 1;
        System.out.println(matrix[0][0]);
    }
}
```
