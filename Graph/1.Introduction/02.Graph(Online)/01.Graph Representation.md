# 1. Adjacency List (Undirected Graph)

### Input

```text
5 4
0 1
0 2
1 3
3 4
```

* 5 vertices
* 4 edges

### Code

```java
import java.util.*;

public class Main {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        int e = sc.nextInt();

        List<List<Integer>> graph = new ArrayList<>();

        for(int i = 0; i < n; i++) {
            graph.add(new ArrayList<>());
        }

        for(int i = 0; i < e; i++) {

            int u = sc.nextInt();
            int v = sc.nextInt();

            graph.get(u).add(v);
            graph.get(v).add(u);
        }

        System.out.println(graph);
    }
}
```

### Output

```text
[[1, 2], [0, 3], [0], [1, 4], [3]]
```

# 3. Adjacency Matrix (Undirected)

### Input

```text
5 4
0 1
0 2
1 3
3 4
```

### Code

```java
import java.util.*;

public class Main {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        int e = sc.nextInt();

        int[][] graph = new int[n][n];

        for(int i = 0; i < e; i++) {

            int u = sc.nextInt();
            int v = sc.nextInt();

            graph[u][v] = 1;
            graph[v][u] = 1;
        }

        for(int[] row : graph) {
            System.out.println(Arrays.toString(row));
        }
    }
}
```
