
# 1. Adjacency List

```java
int n = 5;

List<List<Integer>> graph = new ArrayList<>();

for(int i = 0; i < n; i++) {
    graph.add(new ArrayList<>());
}

graph.get(0).add(1);
graph.get(1).add(0);

graph.get(0).add(2);
graph.get(2).add(0);
```

---

# 2. Adjacency Matrix

```java
int n = 5;

int[][] graph = new int[n][n];

graph[0][1] = 1;
graph[1][0] = 1;

graph[0][2] = 1;
graph[2][0] = 1;
```
