# Minimum Spanning Tree (MST)

There are two important MST algorithms: 
     
1. **Kruskal's Algorithm**        
2. **Prim's Algorithm**         
     
---  

# 1. Kruskal's Algorithm (Using DSU) 
 
### Input

```text
4 5
0 1 10
0 2 6
0 3 5
1 3 15
2 3 4
```

Format:

```text
Vertices Edges
u v weight
u v weight
...
```

---

### Complete Java Code

```java
import java.util.*;

class Edge {
    int u, v, weight;

    Edge(int u, int v, int weight) {
        this.u = u;
        this.v = v;
        this.weight = weight;
    }
}

public class Main {

    static int[] parent;
    static int[] rank;

    static int find(int x) {

        if (parent[x] == x) {
            return x;
        }

        return parent[x] = find(parent[x]);
    }

    static void union(int x, int y) {

        int px = find(x);
        int py = find(y);

        if (px == py) {
            return;
        }

        if (rank[px] < rank[py]) {
            parent[px] = py;
        } else if (rank[px] > rank[py]) {
            parent[py] = px;
        } else {
            parent[py] = px;
            rank[px]++;
        }
    }

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        int V = sc.nextInt();
        int E = sc.nextInt();

        Edge[] edges = new Edge[E];

        for (int i = 0; i < E; i++) {

            int u = sc.nextInt();
            int v = sc.nextInt();
            int w = sc.nextInt();

            edges[i] = new Edge(u, v, w);
        }

        parent = new int[V];
        rank = new int[V];

        for (int i = 0; i < V; i++) {
            parent[i] = i;
        }

        Arrays.sort(edges,
                (a, b) -> a.weight - b.weight);

        int mstCost = 0;

        System.out.println("MST Edges:");

        for (Edge edge : edges) {

            if (find(edge.u) != find(edge.v)) {

                union(edge.u, edge.v);

                mstCost += edge.weight;

                System.out.println(
                        edge.u + " - " +
                        edge.v + " : " +
                        edge.weight);
            }
        }

        System.out.println("MST Cost = " + mstCost);
    }
}
```

---

### Output

```text
MST Edges:
2 - 3 : 4
0 - 3 : 5
0 - 1 : 10

MST Cost = 19
```

# 2. Prim's Algorithm

### Input

```text
4 5
0 1 10
0 2 6
0 3 5
1 3 15
2 3 4
```

---

### Complete Java Code

```java
import java.util.*;

class Pair {
    int node;
    int weight;

    Pair(int node, int weight) {
        this.node = node;
        this.weight = weight;
    }
}

public class Main {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        int V = sc.nextInt();
        int E = sc.nextInt();

        List<List<Pair>> graph = new ArrayList<>();

        for (int i = 0; i < V; i++) {
            graph.add(new ArrayList<>());
        }

        for (int i = 0; i < E; i++) {

            int u = sc.nextInt();
            int v = sc.nextInt();
            int w = sc.nextInt();

            graph.get(u).add(new Pair(v, w));
            graph.get(v).add(new Pair(u, w));
        }

        boolean[] visited = new boolean[V];

        PriorityQueue<Pair> pq =
                new PriorityQueue<>(
                        (a, b) -> a.weight - b.weight);

        pq.offer(new Pair(0, 0));

        int mstCost = 0;

        while (!pq.isEmpty()) {

            Pair curr = pq.poll();

            int node = curr.node;
            int weight = curr.weight;

            if (visited[node]) {
                continue;
            }

            visited[node] = true;

            mstCost += weight;

            for (Pair nbr : graph.get(node)) {

                if (!visited[nbr.node]) {
                    pq.offer(
                            new Pair(
                                    nbr.node,
                                    nbr.weight));
                }
            }
        }

        System.out.println("MST Cost = " + mstCost);
    }
}
```

---

### Output

```text
MST Cost = 19
```

# Interview Focus

### Kruskal

Uses:

```text
Sorting
+
DSU (Union Find)
```

Complexity:

```text
O(E log E)
```

Questions:

* Min Cost to Connect All Points
* Redundant Connection
* Connecting Cities

---

### Prim

Uses:

```text
Priority Queue
+
Greedy
```

Complexity:

```text
O(E log V)
```

Questions:

* Minimum Spanning Tree
* Network Design

For most coding interviews and online assessments, the most important MST template is:

```text
Kruskal + DSU ⭐⭐⭐⭐⭐
```

because it simultaneously tests:

* Graphs
* Greedy
* Sorting
* Union Find (DSU)

which are frequently asked together.
