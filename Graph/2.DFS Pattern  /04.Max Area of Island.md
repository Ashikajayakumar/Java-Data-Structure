# Problem 
   
You are given a binary matrix.       
     
* `1` → Land  
* `0` → Water 
  
Return the **maximum area** of an island.

The **area** is the number of connected land cells.

Only **Up, Down, Left, Right** are considered connected.

---

# Difference from Number of Islands

| Number of Islands                                   | Max Area of Island         |
| --------------------------------------------------- | -------------------------- |
| Count islands                                       | Find largest island        |
| Answer starts from 0 and increments for each island | Answer stores maximum area |
| DFS returns nothing                                 | DFS returns area           |

---

# Key Idea

Whenever we find a land cell (`1`):

* Run DFS.
* Count every connected land cell.
* DFS returns the area of that island.
* Keep the maximum area.

---

# Algorithm

```text
maxArea = 0

for every cell

      if land

            area = DFS()

            maxArea = max(maxArea, area)

return maxArea
```

---

# DFS Idea

Every land contributes **1** to the island area.

Current Area

```text
1
+
Up
+
Down
+
Left
+
Right
```

So,

```text
Area =
1
+ dfs(up)
+ dfs(down)
+ dfs(left)
+ dfs(right)
```

---

# Java Code (DFS)

```java
class Solution {

    public int maxAreaOfIsland(int[][] grid) {

        int rows = grid.length;
        int cols = grid[0].length;

        int maxArea = 0;

        for (int r = 0; r < rows; r++) {

            for (int c = 0; c < cols; c++) {

                if (grid[r][c] == 1) {

                    int area = dfs(grid, r, c);

                    maxArea = Math.max(maxArea, area);
                }
            }
        }

        return maxArea;
    }

    private int dfs(int[][] grid, int r, int c) {

        if (r < 0 || c < 0 ||
            r >= grid.length ||
            c >= grid[0].length ||
            grid[r][c] == 0) {

            return 0;
        }

        // Mark as visited
        grid[r][c] = 0;

        return 1
                + dfs(grid, r - 1, c)   // Up
                + dfs(grid, r + 1, c)   // Down
                + dfs(grid, r, c - 1)   // Left
                + dfs(grid, r, c + 1);  // Right
    }
}
```

# How DFS Returns the Area

Suppose

```text
1 1
1 0
```

Call

```text
dfs(0,0)
```

Returns

```text
1
+
dfs(-1,0)=0
+
dfs(1,0)=1
+
dfs(0,-1)=0
+
dfs(0,1)=1
```

Now,

```text
dfs(1,0)

=

1

+0+0+0+0

=1
```

Similarly,

```text
dfs(0,1)

=

1

+0+0+0+0

=1
```

Finally,

```text
dfs(0,0)

=

1+1+1

=3
```

```
dfs(0,0)

       returns 1

     dfs(0,1)

       returns 1

```

# Why Return `0`?

If

* Outside grid
* Water
* Already visited

Then

```java
return 0;
```

Because these cells do **not** contribute to the island area.
