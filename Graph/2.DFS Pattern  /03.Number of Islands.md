# LeetCode 200. Number of Islands

 
# Problem

You are given a 2D grid of `'1'` (land) and `'0'` (water).

Return the number of **islands**.

An island is formed by connecting adjacent lands **horizontally or vertically**.

Diagonal cells are **not connected**.

---

```text
[

```

# Key Idea

Think of every island as a **connected component** in a grid.

Whenever we find an unvisited land cell (`'1'`):

1. We have found a **new island**.
2. Increase the island count.
3. Run DFS to visit every connected land cell.
4. Mark all visited cells so they are not counted again.

---

# Algorithm

```text
count = 0

for every cell

    if cell == '1'

         count++

         DFS(row, col)

return count
```

---

# DFS Logic

For every land cell,

Visit

↓

Go

* Up
* Down
* Left
* Right

Ignore

* Water
* Outside grid
* Already visited land

---

# Java Code (DFS)

```java
class Solution {

    public int numIslands(char[][] grid) {

        int rows = grid.length;
        int cols = grid[0].length;

        int islands = 0;

        for (int r = 0; r < rows; r++) {

            for (int c = 0; c < cols; c++) {

                if (grid[r][c] == '1') {

                    islands++;

                    dfs(grid, r, c);
                }
            }
        }

        return islands;
    }

    private void dfs(char[][] grid, int r, int c) {

        if (r < 0 || c < 0 ||
            r >= grid.length ||
            c >= grid[0].length ||
            grid[r][c] == '0') {

            return;
        }

        // Mark as visited
        grid[r][c] = '0';

        dfs(grid, r - 1, c); // Up
        dfs(grid, r + 1, c); // Down
        dfs(grid, r, c - 1); // Left
        dfs(grid, r, c + 1); // Right
    }
}
```

# Why change `'1'` to `'0'`?

Instead of using a separate `visited[][]` array, we directly mark visited land as water.

Before DFS

```text
1
```

After DFS

```text
0
```

So next time we scan,

```text
if(grid[r][c]=='1')
```

will be false, preventing us from counting the same island again.

This saves extra memory.
