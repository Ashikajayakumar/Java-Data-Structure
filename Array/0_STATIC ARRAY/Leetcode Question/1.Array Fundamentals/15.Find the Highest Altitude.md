## Problem

You are given an array `gain[]` where:

```text id="a1"
gain[i] = altitude change from point i to i+1
```

Start altitude = `0`

Return the **highest altitude reached**.

---

# Example

```text id="a2"
Input:
gain = [-5,1,5,0,-7]
```

---

# Java Solution

```java id="c1"
class Solution {

    public int largestAltitude(int[] gain) {

        int altitude = 0;
        int max = 0;

        for(int g : gain) {

            altitude += g;

            max = Math.max(max, altitude);
        }

        return max;
    }
}
```
