# Build Array from Permutation (LeetCode 1920)

## Problem

Given a **0-indexed array** `nums` of length `n`.

Build a new array `ans` where:

[
ans[i] = nums[nums[i]]
]

Return `ans`.

---

## Example 1

### Input

```text
nums = [0,2,1,5,3,4]
```

### Construction

```text
ans[0] = nums[nums[0]]
       = nums[0]
       = 0

ans[1] = nums[nums[1]]
       = nums[2]
       = 1

ans[2] = nums[nums[2]]
       = nums[1]
       = 2

ans[3] = nums[nums[3]]
       = nums[5]
       = 4

ans[4] = nums[nums[4]]
       = nums[3]
       = 5

ans[5] = nums[nums[5]]
       = nums[4]
       = 3
```

### Output

```text
[0,1,2,4,5,3]
```

## Approach

1. Create a new array `ans`
2. Traverse `nums`
3. Store:

```java
ans[i] = nums[nums[i]];
```

4. Return `ans`

---

## Java Solution

```java
class Solution {
    public int[] buildArray(int[] nums) {

        int n = nums.length;

        int[] ans = new int[n];

        for(int i = 0; i < n; i++) {
            ans[i] = nums[nums[i]];
        }

        return ans;
    }
}
```
