# Number of Good Pairs (LeetCode 1512)

## Problem

Given an array `nums`, return the number of **good pairs**.

A pair `(i, j)` is good if:

```text
nums[i] == nums[j]  and  i < j
```

---

# Example 1

```text
Input:
nums = [1,2,3,1,1,3]
```

Good Pairs:

```text
(0,3) → 1 = 1
(0,4) → 1 = 1
(3,4) → 1 = 1
(2,5) → 3 = 3
```

Output:

```text
4
```

---

# Main Idea

Check every pair:

```text
i < j
```

If:

```text
nums[i] == nums[j]
```

increase count.

---

# Easy Java Solution (Brute Force)

```java
class Solution {

    public int numIdenticalPairs(int[] nums) {

        int count = 0;

        for(int i = 0; i < nums.length; i++) {

            for(int j = i + 1; j < nums.length; j++) {

                if(nums[i] == nums[j]) {
                    count++;
                }
            }
        }

        return count;
    }
}
```

# Optimized HashMap Solution

```java
class Solution {

    public int numIdenticalPairs(int[] nums) {

        HashMap<Integer, Integer> map = new HashMap<>();

        int count = 0;

        for(int num : nums) {

            count += map.getOrDefault(num, 0);

            map.put(num, map.getOrDefault(num, 0) + 1);
        }

        return count;
    }
}
```

---

# Why HashMap Works?

```text
nums = [1,1,1]
```

### First 1

```text

map = {1=1}
```

### Second 1

```text
count += 1
map = {1=2}
```

### Third 1

```text
count += 2
map = {1=3}
```

Total:

```text
3
```
