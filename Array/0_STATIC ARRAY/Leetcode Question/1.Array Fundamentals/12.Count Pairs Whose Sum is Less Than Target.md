## Problem

Given an array `nums` and an integer `target`, count the number of pairs `(i, j)` such that:

```text id="a1"
i < j  and  nums[i] + nums[j] < target
```

---

# Example

```text id="a2"
Input:
nums = [-1,1,2,3,1]
target = 2
```

# Approach 1: Brute Force

Check all pairs.

---

## Java Code (Brute Force)

```java id="b1"
class Solution {

    public int countPairs(List<Integer> nums, int target) {

        int count = 0;

        for(int i = 0; i < nums.size(); i++) {

            for(int j = i + 1; j < nums.size(); j++) {

                if(nums.get(i) + nums.get(j) < target) {
                    count++;
                }
            }
        }

        return count;
    }
}
```

# Approach 2: Two Pointer (Optimal)

### Step Idea:

1. Sort array
2. Use two pointers (left, right)

---

## Visualization

```text id="b2"
[-1, 1, 1, 2, 3]
  L          R
```

If sum < target:

* count all pairs with `R`
* move `L++`

Else:

* move `R--`

---

## Java Code (Optimal)

```java id="b3"
class Solution {

    public int countPairs(List<Integer> nums, int target) {

        Collections.sort(nums);

        int left = 0;
        int right = nums.size() - 1;
        int count = 0;

        while(left < right) {

            if(nums.get(left) + nums.get(right) < target) {

                count += (right - left);
                left++;
            } else {
                right--;
            }
        }

        return count;
    }
}
```
