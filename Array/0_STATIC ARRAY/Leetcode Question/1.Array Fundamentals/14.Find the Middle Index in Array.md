## Problem

You are given an integer array `nums`.

A **middle index** is an index where:

```text id="a1"
sum of elements on the left == sum of elements on the right
```

Return the **first middle index**, otherwise return `-1`.

---

# Example

```text id="a2"
Input:
nums = [2,3,-1,8,4]
```

---

# Visualization

```text id="a3"
Index:  0  1  2  3  4
Nums:   2  3 -1  8  4
```

---

## Check index 3

### Left sum

```text id="a4"
2 + 3 + (-1) = 4
```

### Right sum

```text id="a5"
4 = 4
```

✔ Equal → middle index found

---

# Output

```text id="a6"
3
```


---

# Formula

leftSum = totalSum - leftSum - nums[i]

---

# Java Solution

```java id="b1"
class Solution {

    public int findMiddleIndex(int[] nums) {

        int total = 0;

        for(int num : nums){
            total += num;
        }

        int leftSum = 0;

        for(int i = 0; i < nums.length; i++){

            int rightSum = total - leftSum - nums[i];

            if(leftSum == rightSum){
                return i;
            }

            leftSum += nums[i];
        } 

        return -1;
    }
}
```
