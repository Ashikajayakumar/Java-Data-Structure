# Count Items Matching a Rule (LeetCode 1773)

## Problem

You are given a list of items. Each item has:

```text id="a1"
[itemType, color, name]
```

And a rule:

```text id="a2"
ruleKey = "type" | "color" | "name"
ruleValue = some string
```

You need to count how many items match the rule.

---

# Example

```text id="b1"
items =
[
 ["phone","blue","pixel"],
 ["computer","silver","lenovo"],
 ["phone","gold","iphone"]
]

ruleKey = "color"
ruleValue = "silver"
```

---

## Step 1: Understand rule

```text id="b2"
We check column based on ruleKey
```

Mapping:

```text id="b3"
type  → index 0
color → index 1
name  → index 2
```

---

## Step 2: Check each item

### Item 1

```text id="c1"
blue ≠ silver → false
```

### Item 2

```text id="c2"
silver = silver → true
```

### Item 3

```text id="c3"
gold ≠ silver → false
```

---

## Output

```text id="c4"
1
```

# Main Idea

1. Convert ruleKey → index
2. Compare items[i][index] with ruleValue
3. Count matches

---

# Formula

count=\sum_{i=0}^{n-1} [items[i][index]==ruleValue]

---

# Java Solution

```java id="e1"
class Solution {

    public int countMatches(List<List<String>> items, String ruleKey, String ruleValue) {

        int index = 0;

        if(ruleKey.equals("type")) {
            index = 0;
        } 
        else if(ruleKey.equals("color")) {
            index = 1;
        } 
        else {
            index = 2;
        }

        int count = 0;

        for(List<String> item : items) {

            if(item.get(index).equals(ruleValue)) {
                count++;
            }
        }

        return count;
    }
}
```
