## Problem

You are given a 2D array `accounts`.

* Each row = one customer
* Each column = money in a bank account

Return the **maximum wealth** among all customers.

Wealth of a customer:

```text
wealth = sum of all accounts in that row
```

---

# Example

```text
Input:

accounts =
[
 [1,2,3],
 [3,2,1]
]
```

## Customer 1

```text
1 + 2 + 3 = 6
```

## Customer 2

```text
3 + 2 + 1 = 6
```

Output:

```text
6
```

---

# Main Idea

1. Calculate sum of each row.
2. Keep track of maximum sum.
3. Return maximum.

---

# Formula

Wealth=\sum_{j=0}^{n-1}accounts[i][j]

---

# Easy Java Solution

```java
class Solution {

    public int maximumWealth(int[][] accounts) {

        int maxWealth = 0;

        for(int i = 0; i < accounts.length; i++) {

            int wealth = 0;

            for(int j = 0; j < accounts[i].length; j++) {
                wealth += accounts[i][j];
            }

            maxWealth = Math.max(maxWealth, wealth);
        }

        return maxWealth;
    }
}
```
