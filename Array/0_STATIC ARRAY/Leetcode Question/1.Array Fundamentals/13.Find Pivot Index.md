## Problem
 
Find the **pivot index** in an array. 
  
A pivot index is an index where:  

```text id="a1" 
sum of left side == sum of right side
```

If no such index exists → return `-1`.

---

# Example

```text id="a2"
Input: 
nums = [1,7,3,6,5,6]
```

---

# Step-by-step

## Check index 3

```text id="a3"
Left: 1 + 7 + 3 = 11
Right: 5 + 6 = 11
```

✔ Equal → pivot found

---

# Output

```text id="a4"
3
```

# Optimal Approach

### Step 1:

Find total sum

```text id="a6"
total = sum(nums)
```

### Step 2:

Traverse array and maintain left sum

```text id="a7"
right = total - left - nums[i]
```

---

# Formula

leftSum = rightSum \Rightarrow leftSum = totalSum - leftSum - nums[i]

---

# Java Solution

```java id="b1"
class Solution {

    public int pivotIndex(int[] nums) {

        int total = 0;

        for(int num : nums){
            total += num;
        }

        int leftSum = 0;

        for(int i = 0; i < nums.length; i++){

            int rightSum = total - leftSum - nums[i];

            if(leftSum == rightSum){
                return i;
            }

            leftSum += nums[i];
        }

        return -1;
    }
}
```
