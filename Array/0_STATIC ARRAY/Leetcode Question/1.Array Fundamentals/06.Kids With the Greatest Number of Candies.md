# Kids With the Greatest Number of Candies (LeetCode 1431)

## Problem

You are given an array `candies` and an integer `extraCandies`.

Each kid has some candies. We want to check:

> If we give a kid all `extraCandies`, will they have the **greatest or equal greatest candies** among all kids?

Return a boolean array.

---

# Example

```text id="c1"
Input:
candies = [2,3,5,1,3]
extraCandies = 3
```

---

# Main Idea

1. Find maximum in array
2. Check if:

```text id="c10"
candies[i] + extraCandies >= max
```

---

# Formula

candies[i]+extraCandies \ge max(candies)

---

# Java Solution

```java id="c11"
class Solution {

    public List<Boolean> kidsWithCandies(int[] candies, int extraCandies) {

        int max = 0;

        // Find maximum
        for(int candy : candies){
            max = Math.max(max, candy);
        }

        List<Boolean> result = new ArrayList<>();

        // Check each kid
        for(int candy : candies){

            if(candy + extraCandies >= max){
                result.add(true);
            } else {
                result.add(false);
            }
        }

        return result;
    }
}
```
