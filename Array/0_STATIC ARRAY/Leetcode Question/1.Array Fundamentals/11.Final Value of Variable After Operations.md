## Problem

You are given an array of operations on a variable `X` starting from `0`.

Each operation is one of:

```text id="a1"
"++X", "X++", "--X", "X--"
```

Return the final value of `X`.

---

# Example

```text id="a2"
Input:
operations = ["--X","X++","X++"]
```

---

## Step-by-step

### Start

```text id="a3"
X = 0
```

### Operation 1

```text id="a4"
--X → X = -1
```

### Operation 2

```text id="a5"
X++ → X = 0
```

### Operation 3

```text id="a6"
X++ → X = 1
```

---

## Output

```text id="a7"
1
```

# Java Solution

```java id="b1"
class Solution {

    public int finalValueAfterOperations(String[] operations) {

        int x = 0;

        for(String op : operations) {

            if(op.contains("+")) {
                x++;
            } else {
                x--;
            }
        }

        return x;
    }
}
```
