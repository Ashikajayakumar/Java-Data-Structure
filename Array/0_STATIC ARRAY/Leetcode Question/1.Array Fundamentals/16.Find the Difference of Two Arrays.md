## Problem

Given two integer arrays `nums1` and `nums2`, return:

```text id="a1"
[
  elements in nums1 not in nums2,
  elements in nums2 not in nums1
]
```

Remove duplicates in each result.

---

# Example

```text id="a2"
Input:
nums1 = [1,2,3]
nums2 = [2,4,6]
```

---

## Step-by-step

### Elements in nums1 not in nums2

```text id="a3"
1 ✔
2 ✘
3 ✔
```

Result:

```text id="a4"
[1,3]
```

---

### Elements in nums2 not in nums1

```text id="a5"
2 ✘
4 ✔
6 ✔
```

Result:

```text id="a6"
[4,6]
```

---

## Output

```text id="a7"
[[1,3],[4,6]]
```

# Java Solution

```java id="b1"
class Solution {

    public List<List<Integer>> findDifference(int[] nums1, int[] nums2) {

        HashSet<Integer> set1 = new HashSet<>();
        HashSet<Integer> set2 = new HashSet<>();

        for(int num : nums1){
            set1.add(num);
        }

        for(int num : nums2){
            set2.add(num);
        }

        List<Integer> res1 = new ArrayList<>();
        List<Integer> res2 = new ArrayList<>();

        // nums1 - nums2
        for(int num : set1){
            if(!set2.contains(num)){
                res1.add(num);
            }
        }

        // nums2 - nums1
        for(int num : set2){
            if(!set1.contains(num)){
                res2.add(num);
            }
        }

        List<List<Integer>> ans = new ArrayList<>();
        ans.add(res1);
        ans.add(res2);

        return ans;
    }
}
```
