# Shuffle the Array (LeetCode 1470)

## Problem

You are given an array:

```text id="a1"
nums = [x1,x2,...,xn,y1,y2,...,yn]
```

and an integer `n`.

You need to return:

```text id="a2"
[x1,y1,x2,y2,...,xn,yn]
```

---

# Example

```text id="b1"
Input:
nums = [2,5,1,3,4,7]
n = 3
```

---

## Step 1: Split array

```text id="b2"
x = [2,5,1]
y = [3,4,7]
```

---

## Step 2: Shuffle

```text id="b3"
x1=2, y1=3
x2=5, y2=4
x3=1, y3=7
```

---

## Output

```text id="b4"
[2,3,5,4,1,7]
```

---

# Visualization

# Main Idea

We use two pointers:

* one for first half
* one for second half

---

# Formula

result[2i]=x_i,\quad result[2i+1]=y_i

---

# Java Solution

```java id="d1"
class Solution {

    public int[] shuffle(int[] nums, int n) {

        int[] res = new int[2 * n];

        int index = 0;

        for(int i = 0; i < n; i++) {

            res[index++] = nums[i];       // xi
            res[index++] = nums[i + n];   // yi
        }

        return res;
    }
}
```
