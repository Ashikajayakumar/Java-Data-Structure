# Find Numbers with Even Number of Digits (LeetCode 1295)

## Problem

Given an array `nums`, count how many numbers have an **even number of digits**.

---

# Example

```text id="a1"
Input:
nums = [12, 345, 2, 6, 7896]
```

---

## Step 1: Check digits

```text id="a2"
12    → 2 digits → even ✔
345   → 3 digits → odd  ✘
2     → 1 digit  → odd  ✘
6     → 1 digit  → odd  ✘
7896  → 4 digits → even ✔
```

---

## Output

```text id="a3"
2
```

# Java Solution

### Simple (String method)

```java id="b1"
class Solution {

    public int findNumbers(int[] nums) {

        int count = 0;

        for(int num : nums) {

            String s = String.valueOf(num);

            if(s.length() % 2 == 0) {
                count++;
            }
        }

        return count;
    }
}
```

---

### Optimized (Math method)

```java id="b2"
class Solution {

    public int findNumbers(int[] nums) {

        int count = 0;

        for(int num : nums) {

            int digits = 0;

            while(num > 0) {
                num /= 10;
                digits++;
            }

            if(digits % 2 == 0) {
                count++;
            }
        }

        return count;
    }
}
```
