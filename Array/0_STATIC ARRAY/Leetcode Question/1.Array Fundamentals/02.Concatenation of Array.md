Given an integer array `nums` of length `n`.

Create an array `ans` of length `2n` where:

```text
ans[i] = nums[i]
ans[i + n] = nums[i]
```

Return `ans`.

---

# Example

## Input

```text
nums = [1,2,1]
```

## Output

```text
[1,2,1,1,2,1]
```

# Main Idea

Copy the array twice.

```text
ans = nums + nums
```

---

# Easy Java Solution

```java
class Solution {

    public int[] getConcatenation(int[] nums) {

        int n = nums.length;

        int[] ans = new int[2 * n];

        for(int i = 0; i < n; i++) {

            ans[i] = nums[i];

            ans[i + n] = nums[i];
        }

        return ans;
    }
}
```
