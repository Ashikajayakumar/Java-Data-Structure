# LeetCode 1464: Maximum Product of Two Elements in an Array

## Problem Statement

Given an integer array `nums`, choose two different indices `i` and `j`.

Return the maximum value of:

(nums[i]-1)\times(nums[j]-1)

---

## Example 1

```java
Input: nums = [3,4,5,2]

Output: 12
```

### Explanation

Choose:

```text
5 and 4
```

Calculate:

```text
(5 - 1) × (4 - 1)
= 4 × 3
= 12
```

Answer:

```text
12
```

## Approach 1: Sorting

### Algorithm

1. Sort the array.
2. Take the largest two elements.
3. Apply the formula.

---

## Java Solution (Sorting)

```java
import java.util.Arrays;

class Solution {
    public int maxProduct(int[] nums) {

        Arrays.sort(nums);

        int n = nums.length;

        return (nums[n - 1] - 1) * (nums[n - 2] - 1);
    }
}
```

---

## Approach 2: Optimal (One Pass)

### Idea

Track:

* `max1` → largest number
* `max2` → second largest number

Then return:

```text
(max1 - 1) × (max2 - 1)
```

## Java Solution (Optimal)

```java
class Solution {
    public int maxProduct(int[] nums) {

        int max1 = 0;
        int max2 = 0;

        for (int num : nums) {

            if (num > max1) {
                max2 = max1;
                max1 = num;
            } else if (num > max2) {
                max2 = num;
            }
        }

        return (max1 - 1) * (max2 - 1);
    }
}
```
