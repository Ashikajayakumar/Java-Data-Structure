## Problem Statement

Given an integer array `nums` consisting of `n` elements and an integer `k`.

Find the contiguous subarray of length `k` that has the maximum average value and return that value.

Any answer with an error less than `10⁻⁵` will be accepted.

---

## Example 1

```java
Input: nums = [1,12,-5,-6,50,3], k = 4

Output: 12.75
```

### Explanation

Subarrays of size `4`:

```text
[1,12,-5,-6]  → Sum = 2
Average = 2 / 4 = 0.5

[12,-5,-6,50] → Sum = 51
Average = 51 / 4 = 12.75

[-5,-6,50,3]  → Sum = 42
Average = 42 / 4 = 10.5
```

Maximum Average:

51/4 =12.75

Output:

```text
12.75
```

# Brute Force Approach

## Idea

Generate every subarray of size `k`.

Calculate its sum and average.

Keep track of the maximum average.

---

## Java Solution (Brute Force)

```java
class Solution {
    public double findMaxAverage(int[] nums, int k) {

        double maxAvg = Double.NEGATIVE_INFINITY;

        for (int i = 0; i <= nums.length - k; i++) {

            int sum = 0;

            for (int j = i; j < i + k; j++) {
                sum += nums[j];
            }

            maxAvg = Math.max(maxAvg, (double) sum / k);
        }

        return maxAvg;
    }
}
```


# Optimal Approach (Sliding Window)


## Sliding Window Formula

When moving from one window to the next:

NewSum = OldSum - OutgoingElement + IncomingElement

This reduces the complexity to **O(n)**.

# Java Solution (Optimal)

```java
class Solution {
    public double findMaxAverage(int[] nums, int k) {

        int windowSum = 0;

        for (int i = 0; i < k; i++) {
            windowSum += nums[i];
        }

        int maxSum = windowSum;

        for (int i = k; i < nums.length; i++) {

            windowSum += nums[i];
            windowSum -= nums[i - k];

            maxSum = Math.max(maxSum, windowSum);
        }

        return (double) maxSum / k;
    }
}
```
