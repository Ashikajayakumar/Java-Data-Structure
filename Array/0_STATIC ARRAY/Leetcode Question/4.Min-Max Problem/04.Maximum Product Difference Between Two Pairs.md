# LeetCode 1913: Maximum Product Difference Between Two Pairs

## Problem Statement

Given an integer array `nums`, choose four distinct indices `w`, `x`, `y`, and `z`.

Return the maximum value of:

(nums[w] * nums[x]) - (nums[y] * nums[z])

---

## Example 1

```java
Input: nums = [5,6,2,7,4]

Output: 34
```

### Explanation

Choose:

```text
7 and 6
```

Their product:

```text
7 × 6 = 42
```

Choose:

```text
2 and 4
```

Their product:

```text
2 × 4 = 8
```

Product Difference:

```text
42 - 8 = 34
```

Output:

```text
34
```

## Approach 1: Sorting

### Algorithm

1. Sort the array.
2. Take:

   * Two largest elements
   * Two smallest elements
3. Compute the product difference.

---

## Java Solution (Sorting)

```java
import java.util.Arrays;

class Solution {
    public int maxProductDifference(int[] nums) {

        Arrays.sort(nums);

        int n = nums.length;

        return (nums[n - 1] * nums[n - 2])
                - (nums[0] * nums[1]);
    }
}
```

## Approach 2: Optimal (One Pass)

### Idea

Track:

* Largest number → `max1`
* Second largest → `max2`
* Smallest number → `min1`
* Second smallest → `min2`

Then compute:

```text
(max1 × max2) − (min1 × min2)
```

---

## Dry Run

```java
nums = [5,6,2,7,4]
```

After traversal:

```text
max1 = 7
max2 = 6

min1 = 2
min2 = 4
```

Answer:

```text
(7 × 6) − (2 × 4)

42 − 8

34
```

## Java Solution (Optimal)

```java
class Solution {
    public int maxProductDifference(int[] nums) {

        int max1 = Integer.MIN_VALUE;
        int max2 = Integer.MIN_VALUE;

        int min1 = Integer.MAX_VALUE;
        int min2 = Integer.MAX_VALUE;

        for (int num : nums) {

            // Update largest numbers
            if (num > max1) {
                max2 = max1;
                max1 = num;
            } else if (num > max2) {
                max2 = num;
            }

            // Update smallest numbers
            if (num < min1) {
                min2 = min1;
                min1 = num;
            } else if (num < min2) {
                min2 = num;
            }
        }

        return (max1 * max2) - (min1 * min2);
    }
}
```


**Time Complexity:** O(n) (Optimal)
**Space Complexity:** O(1)
