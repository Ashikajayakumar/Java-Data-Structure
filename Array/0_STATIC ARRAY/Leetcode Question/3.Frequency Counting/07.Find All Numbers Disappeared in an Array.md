## Problem Statement

Given an array `nums` of length `n` where:

```text
1 ≤ nums[i] ≤ n
```

Some numbers appear twice and others appear once.

Return all numbers in the range:

```text
1 to n
```

that do **not appear** in the array.

---

## Example 1

```java
Input: nums = [4,3,2,7,8,2,3,1]

Output: [5,6]
```

### Explanation

Numbers from 1 to 8:

```text
1 2 3 4 5 6 7 8
```

Present in array:

```text
1 2 3 4 7 8
```

Missing:

```text
5 6
```

Answer:

```java
[5,6]
```

---

## Example 2

```java
Input: nums = [1,1]

Output: [2]
```

### Explanation

Numbers from 1 to 2:

```text
1 2
```

Present:

```text
1
```

Missing:

```text
2
```

Answer:

```java
[2]
```

# Approach 1: HashSet



---

## Java Solution

```java
import java.util.*;

class Solution {
    public List<Integer> findDisappearedNumbers(int[] nums) {

        HashSet<Integer> set = new HashSet<>();

        for (int num : nums) {
            set.add(num);
        }

        List<Integer> result = new ArrayList<>();

        for (int i = 1; i <= nums.length; i++) {

            if (!set.contains(i)) {
                result.add(i);
            }
        }

        return result;
    }
}
```

# Approach 2: Frequency Array

```java
import java.util.*;

class Solution {
    public List<Integer> findDisappearedNumbers(int[] nums) {

        int n = nums.length;

        int[] freq = new int[n + 1];

        for (int num : nums) {
            freq[num]++;
        }

        List<Integer> result = new ArrayList<>();

        for (int i = 1; i <= n; i++) {

            if (freq[i] == 0) {
                result.add(i);
            }
        }

        return result;
    }
}
```

# Optimal Approach: Negative Marking



## Optimal Code

```java
import java.util.*;

class Solution {
    public List<Integer> findDisappearedNumbers(int[] nums) {

        List<Integer> result = new ArrayList<>();

        for (int i = 0; i < nums.length; i++) {

            int index = Math.abs(nums[i]) - 1;

            if (nums[index] > 0) {
                nums[index] = -nums[index];
            }
        }

        for (int i = 0; i < nums.length; i++) {

            if (nums[i] > 0) {
                result.add(i + 1);
            }
        }

        return result;
    }
}
```

