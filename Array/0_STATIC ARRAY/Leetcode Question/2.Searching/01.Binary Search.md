## Problem

Given a **sorted array** `nums` and a `target` value:

* Return the index of `target` if found.
* Otherwise return `-1`.

---

# Example

```text
Input:
nums = [-1,0,3,5,9,12]
target = 9
```

Output:

```text
4
```

Because:

```text
nums[4] = 9
```

---

## Step 1

```text
left = 0
right = 5

mid = (0+5)/2 = 2
nums[mid] = 3
```

Since:

```text
3 < 9
```

Search right half.

---

## Step 2

```text
left = 3
right = 5

mid = (3+5)/2 = 4
nums[mid] = 9
```

Found target.

---

## Output

```text
4
```

---





# Formula

mid = left + right-left/2

---

# Java Solution

```java
class Solution {

    public int search(int[] nums, int target) {

        int left = 0;
        int right = nums.length - 1;

        while(left <= right){

            int mid = left + (right - left) / 2;

            if(nums[mid] == target){
                return mid;
            }

            else if(nums[mid] < target){
                left = mid + 1;
            }

            else{
                right = mid - 1;
            }
        }

        return -1;
    }
}
```
