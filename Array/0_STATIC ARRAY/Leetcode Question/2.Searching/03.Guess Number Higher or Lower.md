## Problem Statement

You are playing a guessing game.

I pick a number from **1 to n**.

You have to guess the number.

There is a predefined API:

```java
int guess(int num)
```

It returns:

* `-1` → My number is lower than your guess.
* `1` → My number is higher than your guess.
* `0` → Correct guess.

Return the number that I picked.

---

## Example 1

**Input**

```text
n = 10, pick = 6
```

**Output**

```text
6
```

### Explanation

```text
Guess 5 → Higher (1)
Guess 8 → Lower (-1)
Guess 6 → Correct (0)
```

Answer = 6

## Constraints

```text
1 <= n <= 2^31 - 1
1 <= pick <= n
```

---

## Hint

Since the numbers are sorted from `1` to `n`, use **Binary Search**.

* Find middle number.
* If guess is too high, search left half.
* If guess is too low, search right half.
* Repeat until found.

### Time Complexity

```text
O(log n)
```

### Space Complexity

```text
O(1)
```

---

## Java Solution

```java
public class Solution extends GuessGame {

    public int guessNumber(int n) {

        int left = 1;
        int right = n;

        while (left <= right) {

            int mid = left + (right - left) / 2;

            int result = guess(mid);

            if (result == 0) {
                return mid;
            }
            else if (result == -1) {
                right = mid - 1;
            }
            else {
                left = mid + 1;
            }
        }

        return -1;
    }
}
```
