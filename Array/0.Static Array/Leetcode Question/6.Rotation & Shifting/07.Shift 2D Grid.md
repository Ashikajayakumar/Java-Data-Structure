# LeetCode 1260: Shift 2D Grid

## Problem Statement

Given a 2D grid of size `m × n` and an integer `k`, shift the grid `k` times.

One shift operation:

```text
grid[i][j]         → grid[i][j + 1]
grid[i][n - 1]     → grid[i + 1][0]
grid[m - 1][n - 1] → grid[0][0]
```

Return the resulting grid.

---

## Example 1

```text
Input:

grid = [
 [1,2,3],
 [4,5,6],
 [7,8,9]
]

k = 1

Output:

[
 [9,1,2],
 [3,4,5],
 [6,7,8]
]
```

---

# Simple Idea

Treat the 2D grid as a **1D array**.

```text
1 2 3
4 5 6
7 8 9

↓

[1,2,3,4,5,6,7,8,9]
```

After shifting:

```text
[9,1,2,3,4,5,6,7,8]
```

Convert back to 2D.

---

# Formula

For every element:

```text
oldIndex = i * n + j
```

New position:

```text
newIndex = (oldIndex + k) % (m * n)
```

Convert back:

```text
newRow = newIndex / n
newCol = newIndex % n
```

# Java Solution

```java
class Solution {

    public List<List<Integer>> shiftGrid(int[][] grid, int k) {

        int m = grid.length;
        int n = grid[0].length;

        int total = m * n;

        int[][] result = new int[m][n];

        for (int i = 0; i < m; i++) {

            for (int j = 0; j < n; j++) {

                int oldIndex = i * n + j;

                int newIndex = (oldIndex + k) % total;

                int newRow = newIndex / n;
                int newCol = newIndex % n;

                result[newRow][newCol] = grid[i][j];
            }
        }

        List<List<Integer>> answer = new ArrayList<>();

        for (int[] row : result) {

            List<Integer> list = new ArrayList<>();

            for (int num : row) {
                list.add(num);
            }

            answer.add(list);
        }

        return answer;
    }
}
```
