## Problem Statement

You have a set containing numbers from `1` to `n`.

Due to an error:

* One number appears **twice** (duplicate).
* One number is **missing**.

Return an array:

```java
[duplicate, missing]
```

---

## Example 1

```java
Input: nums = [1,2,2,4]

Output: [2,3]
```

### Explanation

Numbers should be:

```text
1 2 3 4
```

Given:

```text
1 2 2 4
```

* Duplicate = 2
* Missing = 3

Answer:

```java
[2,3]
```

---

## Example 2

```java
Input: nums = [1,1]

Output: [1,2]
```

### Explanation

Numbers should be:

```text
1 2
```

Given:

```text
1 1
```

* Duplicate = 1
* Missing = 2

Answer:

```java
[1,2]
```

# Approach 1: HashSet

### Idea

1. Traverse the array.
2. Use a `HashSet` to detect the duplicate.
3. Find the missing number by checking numbers from `1` to `n`.

---

## Java Solution

```java
import java.util.*;

class Solution {
    public int[] findErrorNums(int[] nums) {

        HashSet<Integer> set = new HashSet<>();

        int duplicate = -1;
        int missing = -1;

        for (int num : nums) {

            if (set.contains(num)) {
                duplicate = num;
            }

            set.add(num);
        }

        for (int i = 1; i <= nums.length; i++) {

            if (!set.contains(i)) {
                missing = i;
                break;
            }
        }

        return new int[]{duplicate, missing};
    }
}
```

# Approach 2: Frequency Array

```java
class Solution {
    public int[] findErrorNums(int[] nums) {

        int n = nums.length;

        int[] freq = new int[n + 1];

        for (int num : nums) {
            freq[num]++;
        }

        int duplicate = -1;
        int missing = -1;

        for (int i = 1; i <= n; i++) {

            if (freq[i] == 2) {
                duplicate = i;
            }

            if (freq[i] == 0) {
                missing = i;
            }
        }

        return new int[]{duplicate, missing};
    }
}
