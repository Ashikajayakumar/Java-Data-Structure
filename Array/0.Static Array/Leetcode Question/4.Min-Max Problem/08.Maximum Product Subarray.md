## Problem Statement

Given an integer array `nums`, find the **contiguous subarray**
that has the **largest product**, and return that product.

> A subarray is a contiguous non-empty sequence of elements within an array.

---

## Example 1

### Input

```text
nums = [2,3,-2,4]
```

### Output

```text
6
```

### Explanation

```text
Subarray = [2,3]

Product = 2 × 3 = 6
```

Largest product:

```text
6
```

---

## Example 2

```text
nums = [-2,0,-1]
```

### Output

```text
0
```

### Explanation

```text
[-2] = -2
[0]  = 0
[-1] = -1
```

Maximum product:


# Brute Force Approach

## Idea

1. Generate every subarray.
2. Calculate product of each subarray.
3. Keep track of maximum product.

---

## Complexity

```text
Time Complexity  : O(n²)
Space Complexity : O(1)
```

---

## Java Solution (Brute Force)

```java
class Solution {
    public int maxProduct(int[] nums) {

        int answer = Integer.MIN_VALUE;

        for (int i = 0; i < nums.length; i++) {

            int product = 1;

            for (int j = i; j < nums.length; j++) {

                product *= nums[j];

                answer = Math.max(answer, product);
            }
        }

        return answer;
    }
}
```

# Optimal Approach (Dynamic Programming)

## Key Idea

At every position maintain:

```text
maxProd = Maximum product ending here
minProd = Minimum product ending here
```

Why?

Because:

```text
Negative × Negative = Positive
```

A minimum product may become the next maximum product.

---

# State Transition

For current number:

```text
num
```

Possible values:

```text
num

num × previousMax

num × previousMin
```

New Maximum:

```text
max(
    num,
    num × previousMax,
    num × previousMin
)
```

New Minimum:

```text
min(
    num,
    num × previousMax,
    num × previousMin
)
```

# Java Solution (Optimal)

```java
class Solution {

    public int maxProduct(int[] nums) {

        int maxProd = nums[0];
        int minProd = nums[0];
        int answer = nums[0];

        for (int i = 1; i < nums.length; i++) {

            int num = nums[i];

            int tempMax = maxProd;

            maxProd = Math.max(
                    num,
                    Math.max(num * maxProd,
                             num * minProd)
            );

            minProd = Math.min(
                    num,
                    Math.min(num * tempMax,
                             num * minProd)
            );

            answer = Math.max(answer, maxProd);
        }

        return answer;
    }
}
```



This allows a small negative product to become a large positive product later.
