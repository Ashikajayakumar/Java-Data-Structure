# LeetCode 628: Maximum Product of Three Numbers

## Problem Statement

Given an integer array `nums`, find three numbers whose product is maximum and return that product.

---

## Example 1

```java
Input: nums = [1,2,3]

Output: 6
```

### Explanation

```text
1 × 2 × 3 = 6
```

Maximum product is:

```text
6
```

---

## Example 2

```java
Input: nums = [1,2,3,4]

Output: 24
```

### Explanation

```text
2 × 3 × 4 = 24
```

Maximum product is:

```text
24
```

---

## Example 3

```java
Input: nums = [-1,-2,-3]

Output: -6
```

### Explanation

```text
(-1) × (-2) × (-3) = -6
```

## Approach 1: Sorting

### Algorithm

1. Sort the array.
2. Calculate:

```text
Product1 = largest × second largest × third largest
```

3. Calculate:

```text
Product2 = smallest × second smallest × largest
```

4. Return the maximum of both.

---

## Dry Run

```java
nums = [-10,-10,5,2]
```

After sorting:

```text
[-10,-10,2,5]
```

### Product 1

```text
5 × 2 × (-10)
= -100
```

### Product 2

```text
(-10) × (-10) × 5
= 500
```

Answer:

```text
500
```

---

## Java Solution (Sorting)

```java
import java.util.Arrays;

class Solution {
    public int maximumProduct(int[] nums) {

        Arrays.sort(nums);

        int n = nums.length;

        int product1 = nums[n - 1] * nums[n - 2] * nums[n - 3];

        int product2 = nums[0] * nums[1] * nums[n - 1];

        return Math.max(product1, product2);
    }
}
```

## Approach 2: Optimal (Without Sorting)

### Idea

Track:

* Largest numbers:

  * max1
  * max2
  * max3

* Smallest numbers:

  * min1
  * min2

Then compute:

```text
max1 × max2 × max3
```

and

```text
min1 × min2 × max1
```

Return the maximum.

---

## Java Solution (Optimal)

```java
class Solution {
    public int maximumProduct(int[] nums) {

        int max1 = Integer.MIN_VALUE;
        int max2 = Integer.MIN_VALUE;
        int max3 = Integer.MIN_VALUE;

        int min1 = Integer.MAX_VALUE;
        int min2 = Integer.MAX_VALUE;

        for (int num : nums) {

            // Update largest values
            if (num > max1) {
                max3 = max2;
                max2 = max1;
                max1 = num;
            } else if (num > max2) {
                max3 = max2;
                max2 = num;
            } else if (num > max3) {
                max3 = num;
            }

            // Update smallest values
            if (num < min1) {
                min2 = min1;
                min1 = num;
            } else if (num < min2) {
                min2 = num;
            }
        }

        return Math.max(
            max1 * max2 * max3,
            min1 * min2 * max1
        );
    }
}
```
