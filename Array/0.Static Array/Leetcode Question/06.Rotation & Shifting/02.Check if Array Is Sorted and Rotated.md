# LeetCode 1752: Check if Array Is Sorted and Rotated
<img width="1536" height="1024" alt="image" src="https://github.com/user-attachments/assets/153cd523-f768-4458-b806-06c6dc723519" />

## Problem Statement

Given an array `nums`, return `true` if the array was originally sorted in non-decreasing order and then rotated some number of positions (including zero). Otherwise, return `false`.

There may be duplicate elements in the array.

---

## Example 1

```text
Input:
nums = [3,4,5,1,2]

Output:
true
```

### Explanation

```text
Original Sorted Array:
[1,2,3,4,5]

Rotate 3 times:

[3,4,5,1,2]
```

---

## Example 2

```text
Input:
nums = [2,1,3,4]

Output:
false
```

### Explanation

```text
More than one break in sorted order.
```

---

## Example 3

```text
Input:
nums = [1,2,3]

Output:
true
```

### Explanation

```text
Already sorted.
Rotation count = 0
```

# Java Solution

```java
class Solution {

    public boolean check(int[] nums) {

        int count = 0;
        int n = nums.length;

        for (int i = 0; i < n; i++) {

            if (nums[i] > nums[(i + 1) % n]) {
                count++;
            }
        }

        return count <= 1;
    }
}
```
