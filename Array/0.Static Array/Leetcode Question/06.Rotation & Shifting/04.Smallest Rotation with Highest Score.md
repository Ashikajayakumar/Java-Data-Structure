# LeetCode 798: Smallest Rotation with Highest Score

## Problem Statement

You are given an array `nums` of length `n`.

For each rotation `k` (`0 ≤ k < n`), rotate the array to the left by `k` positions.

A value `nums[i]` contributes **1 point** if after rotation it is less than or equal to its new index.

Return the **smallest rotation index `k`** that achieves the highest score.

---

## Example 1

```text
Input:
nums = [2,3,1,4,0]

Output:
3
```

# Java Solution

```java
class Solution {

    public int bestRotation(int[] nums) {

        int n = nums.length;

        int[] diff = new int[n];

        for (int i = 0; i < n; i++) {

            int start = (i - nums[i] + 1 + n) % n;
            int end = (i + 1) % n;

            diff[start]--;

            diff[end]++;

            if (start > end) {
                diff[0]--;
            }
        }

        int bestK = 0;
        int score = 0;
        int maxScore = Integer.MIN_VALUE;

        for (int k = 0; k < n; k++) {

            score += diff[k];

            if (score > maxScore) {
                maxScore = score;
                bestK = k;
            }
        }

        return bestK;
    }
}
```
