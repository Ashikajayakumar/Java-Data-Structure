# LeetCode 1051: Height Checker

## Problem Statement

A school is trying to take an annual photo of all students.

The students should stand in **non-decreasing order of height**.

Given an integer array `heights`, return the number of indices where the current heights do not match the expected sorted order.

---

## Example

```text
Input:
heights = [1,1,4,2,1,3]

Output:
3
```

### Explanation

```text
Original:
[1,1,4,2,1,3]

Expected (Sorted):
[1,1,1,2,3,4]
```

Compare:

```text
Index 0 : 1 = 1 ✓
Index 1 : 1 = 1 ✓
Index 2 : 4 ≠ 1 ✗
Index 3 : 2 = 2 ✓
Index 4 : 1 ≠ 3 ✗
Index 5 : 3 ≠ 4 ✗
```

Count:

```text
3 mismatches
```

Answer:

```text
3
```

---

# Simple Java Code

```java
class Solution {

    public int heightChecker(int[] heights) {

        int[] expected = heights.clone();

        Arrays.sort(expected);

        int count = 0;

        for (int i = 0; i < heights.length; i++) {

            if (heights[i] != expected[i]) {
                count++;
            }
        }

        return count;
    }
}
```
