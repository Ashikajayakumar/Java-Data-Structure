# LeetCode 324: Wiggle Sort II

## Problem Statement

Given an integer array `nums`, reorder it such that:

```text
nums[0] < nums[1] > nums[2] < nums[3] > nums[4] ...
```

This is called a **wiggle order**.

---

## Example 1

```text
Input:
nums = [1,5,1,1,6,4]

Output:
[1,6,1,5,1,4]
```

Check:

```text
1 < 6 > 1 < 5 > 1 < 4
✓
```

---

## Example 2

```text
Input:
nums = [1,3,2,2,3,1]

Output:
[2,3,1,3,1,2]
```

Check:

```text
2 < 3 > 1 < 3 > 1 < 2
✓
```

---

# Simple Idea

1. Sort the array.
2. Put the smaller half at even indices.
3. Put the larger half at odd indices.

This avoids adjacent equal values and creates the wiggle pattern.

---

# Simple Java Code

```java
class Solution {

    public void wiggleSort(int[] nums) {

        int n = nums.length;

        int[] sorted = nums.clone();
        Arrays.sort(sorted);

        int left = (n - 1) / 2; // middle
        int right = n - 1;      // largest

        for (int i = 0; i < n; i++) {

            if (i % 2 == 0) {
                nums[i] = sorted[left--];
            } else {
                nums[i] = sorted[right--];
            }
        }
    }
}
```

```text

│

│
├── Find Middle
│
├── Fill Even Indices
│   with smaller numbers
│
├── Fill Odd Indices

│

```

# Interview Tip

A very simple and accepted approach is:

```text
1. Sort array
2. Split into two halves
3. Fill:
   Even positions ← smaller half
   Odd positions  ← larger half
```

### Shortcut

```text
Sort
↓
Take numbers from middle and end
↓
Place alternately
↓
nums[0] < nums[1] > nums[2] < nums[3] ...
```

This is the easiest interview-friendly solution for **LeetCode 324 – Wiggle Sort II**.
