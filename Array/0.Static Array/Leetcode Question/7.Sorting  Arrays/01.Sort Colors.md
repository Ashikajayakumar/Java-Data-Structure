# LeetCode 75: Sort Colors

## Problem Statement

Given an array `nums` containing only:

```text
0 → Red
1 → White
2 → Blue
```

Sort the array in-place.

---

## Example

```text
Input:
[2,0,2,1,1,0]

Output:
[0,0,1,1,2,2]
```

---

# Simple Code (Counting Method)

```java
class Solution {

    public void sortColors(int[] nums) {

        int zero = 0;
        int one = 0;
        int two = 0;

        for (int num : nums) {

            if (num == 0) zero++;
            else if (num == 1) one++;
            else two++;
        }

        int i = 0;

        while (zero-- > 0) nums[i++] = 0;
        while (one-- > 0) nums[i++] = 1;
        while (two-- > 0) nums[i++] = 2;
    }
}
```

# Optimal Code (Dutch National Flag)

```java
class Solution {

    public void sortColors(int[] nums) {

        int low = 0;
        int mid = 0;
        int high = nums.length - 1;

        while (mid <= high) {

            if (nums[mid] == 0) {

                int temp = nums[low];
                nums[low] = nums[mid];
                nums[mid] = temp;

                low++;
                mid++;

            } else if (nums[mid] == 1) {

                mid++;

            } else {

                int temp = nums[mid];
                nums[mid] = nums[high];
                nums[high] = temp;

                high--;
            }
        }
    }
}
```
