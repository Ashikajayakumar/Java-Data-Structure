# LeetCode 1636: Sort Array by Increasing Frequency

## Problem Statement

Given an integer array `nums`, sort the array:

1. By increasing frequency.
2. If two numbers have the same frequency, sort them in decreasing order.

Return the sorted array.

---

## Example 1

```text
Input:
nums = [1,1,2,2,2,3]

Output:
[3,1,1,2,2,2]
```

### Explanation

```text
Frequency:

1 → 2 times
2 → 3 times
3 → 1 time

Sort by frequency:

3, 1, 2
```

---

## Example 2

```text
Input:
nums = [2,3,1,3,2]

Output:
[1,3,3,2,2]
```

---

# Simple Idea

1. Count frequency using HashMap.
2. Convert array to Integer[].
3. Sort using custom comparator:

   * Smaller frequency first.
   * If same frequency, larger number first.

---

# Simple Java Code

```java
class Solution {

    public int[] frequencySort(int[] nums) {

        HashMap<Integer, Integer> map = new HashMap<>();

        for (int num : nums) {
            map.put(num, map.getOrDefault(num, 0) + 1);
        }

        Integer[] arr = new Integer[nums.length];

        for (int i = 0; i < nums.length; i++) {
            arr[i] = nums[i];
        }

        Arrays.sort(arr, (a, b) -> {

            if (map.get(a).equals(map.get(b))) {
                return b - a; // larger value first
            }

            return map.get(a) - map.get(b); // smaller frequency first
        });

        for (int i = 0; i < nums.length; i++) {
            nums[i] = arr[i];
        }

        return nums;
    }
}
```
