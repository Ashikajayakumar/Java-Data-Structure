# 🔷 Daily Temperatures (LeetCode 739)

## 📌 Problem Statement

You are given an integer array `temperatures`.

For each day, find **how many days you have to wait until a warmer temperature**.

If there is **no warmer temperature** in the future, store `0`.

Return the answer array.

---

### Input

```text
temperatures = [73,74,75,71,69,72,76,73]
```

### Output

```text
[1,1,4,2,1,1,0,0]
```

# ✅ Java Implementation

```java
import java.util.Stack;

class Solution {

    public int[] dailyTemperatures(int[] temperatures) {

        int n = temperatures.length;

        int[] result = new int[n];

        Stack<Integer> st = new Stack<>();

        for(int i = 0; i < n; i++) {

            while(!st.isEmpty() &&
                  temperatures[i] > temperatures[st.peek()]) {

                int previous = st.pop();

                result[previous] = i - previous;

            }

            st.push(i);

        }

        return result;
    }
}
```



# ⏱ Complexity Analysis

## Time Complexity

```
O(N)
```
