# 🔷 Final Prices With a Special Discount in a Shop (LeetCode 1475)

## 📌 Problem Statement

You are given an integer array `prices`.

For every item, find the **first item to its right** whose price is **less than or equal to the current item's price**.

That value becomes the **discount**.

The final price is:

```
Final Price = Current Price - Discount
```

If there is **no such item**, the price remains unchanged.

Return the updated price array.

---

# 💡 Why Use a Stack?

A brute-force solution checks every element's right side.

```
Time Complexity

O(N²)
```

Instead, use a **Monotonic Increasing Stack**.

The stack stores the **indices** of prices that are still waiting to find their discount.

When a smaller or equal price appears, it becomes the discount for previous items.

This reduces the complexity to:

```
O(N)
```

---

# 💡 Main Idea

Instead of asking:

> "What is my discount?"

We ask:

> "Can today's price become the discount for previous items?"

That's why we scan from **left to right**.

---

# 🔄 Algorithm

1. Create an empty stack.
2. Traverse the array from left to right.
3. While the stack is not empty and the current price is less than or equal to the price at the stack's top index:
   - Pop the previous index.
   - Apply the current price as the discount.
4. Push the current index into the stack.
5. Remaining indices in the stack have no discount.

# ✅ Java Implementation

```java
import java.util.Stack;

class Solution {

    public int[] finalPrices(int[] prices) {

        int n = prices.length;

        Stack<Integer> st = new Stack<>();

        for(int i = 0; i < n; i++) {

            while(!st.isEmpty() &&
                  prices[i] <= prices[st.peek()]) {

                int previous = st.pop();

                prices[previous] =
                        prices[previous] - prices[i];

            }

            st.push(i);

        }

        return prices;

    }
}
```
