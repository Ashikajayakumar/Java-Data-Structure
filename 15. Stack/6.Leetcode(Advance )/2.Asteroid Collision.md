# LeetCode 735: Asteroid Collision

## Problem Statement

We are given an array of asteroids.

Example:

```java
asteroids = [5, 10, -5]
```

Each number represents:

* Positive number → asteroid moving **right**
* Negative number → asteroid moving **left**

The absolute value represents the asteroid size.

Example:

```
5  --->

-5 <---
```

When two asteroids collide:

### Case 1: Different sizes

Bigger asteroid survives.

Example:

```
10  --->   -5 <---

10 wins

Result:
10
```

---

### Case 2: Same size

Both destroy each other.

Example:

```
5 --->  <---5

Both disappear

Result:
empty
```

---

### Case 3: Same direction

No collision.

Example:

```
5 ---> 10 --->

Both survive
```

# Step 1: Remove Smaller Right Asteroids

Code:

```java
while(!st.isEmpty() 
      && st.peek()>0 
      && st.peek() < -a)
{
    st.pop();
}
```

Condition:

```
st.peek() = 10

-a = -(-5)

= 5
```

Check:

```
10 > 0       true

10 < 5       false
```

Loop does not execute.

Why?

Because:

```
10 is bigger than 5
```

So `-5` will be destroyed.

Stack remains:

```
[5,10]
```

---

# Step 2: Check If Left Asteroid Survives

Code:

```java
if(st.isEmpty() || st.peek()<0)
{
    st.push(a);
}
```

Meaning:

Push `-5` only if:

1. No asteroid exists
2. Top asteroid is moving left

Here:

```
stack.peek() = 10
```

Check:

```
10 < 0 ❌
```

So:

Do not push.

Stack:

```
[5,10]
```

# Step 3: Check Equal Size Collision

if(st.peek()==-a)

# Complete Code Explanation

```java
class Solution {
    public int[] asteroidCollision(int[] asteroids) {

        Stack<Integer> st = new Stack<>();


        for(int a : asteroids){


            // Right moving asteroid
            if(a > 0){

                st.push(a);

            }


            // Left moving asteroid
            else{


                // Collision happens
                while(!st.isEmpty() 
                      && st.peek()>0 
                      && st.peek() < -a){

                    st.pop();

                }



                // Left asteroid survives
                if(st.isEmpty() || st.peek()<0){

                    st.push(a);

                }



                // Both destroy each other
                if(!st.isEmpty() && st.peek()==-a){

                    st.pop();

                }

            }

        }


        int[] res = new int[st.size()];

        int i = st.size()-1;


        while(!st.isEmpty()){

            res[i--]=st.pop();

        }


        return res;
    }
}
```
