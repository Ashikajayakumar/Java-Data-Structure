# LeetCode 394: Decode String

## Problem Statement

You are given an encoded string `s`.

The encoding rule is:

```
k[encoded_string]
```

means:

> Repeat `encoded_string` exactly `k` times.

The string can contain nested patterns.

### Example 1

```
Input:
s = "3[a]"

Output:
"aaa"
```

Explanation:

```
3[a]

a + a + a

= aaa
```

---

### Example 2

```
Input:
s = "3[a2[c]]"

Output:
"accaccacc"
```

Explanation:

First decode inner:

```
2[c]

= cc
```

Then:

```
a + cc

= acc
```

Then:

```
3[acc]

= accaccacc
```


Example:

```
3[a2[c]]
```

When we see:

```
3[
```

we must remember:

```
repeat = 3
previous string = ""
```

When we see:

```
a2[
```

we must remember:

```
repeat = 2
previous string = "a"
```

After closing brackets, we rebuild the string.

---

# Why Two Stacks?

We need two different information.

## 1. Count Stack

Stores repeat numbers.

Example:

```
3[a2[c]]
```

When reading:

```
3[
```

store:

```
countStack = [3]
```

When reading:

```
2[
```

store:

```
countStack = [3,2]
```

---

## 2. String Stack

Stores previous strings.

Example:

Before inner bracket:

```
3[a2[c]]
```

Before `2[c]`

Current string:

```
"a"
```

Store it:

```
stringStack=["", "a"]
```

# Complete Code Explanation

```java
class Solution {
    public String decodeString(String s) {

        Stack<Integer> count=new Stack<>();

        Stack<StringBuilder> strst=new Stack<>();

        int num=0;

        StringBuilder curr=new StringBuilder();


        for(char ch:s.toCharArray()){


            // Building number
            if(Character.isDigit(ch)){

                num=num*10+(ch-'0');

            }


            // Opening bracket
            else if(ch=='['){

                count.push(num);

                strst.push(curr);


                num=0;

                curr=new StringBuilder();

            }


            // Closing bracket
            else if(ch==']'){


                int r=count.pop();

                StringBuilder prev=strst.pop();


                for(int i=0;i<r;i++){

                    prev.append(curr);

                }


                curr=prev;

            }


            // Normal character
            else{

                curr.append(ch);

            }

        }


        return curr.toString();
    }
}
```
