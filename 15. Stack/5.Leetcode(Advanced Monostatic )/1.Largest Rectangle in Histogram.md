# 🔷 Largest Rectangle in Histogram (LeetCode 84)

## 📌 Problem Statement

Given an array of integers `heights` representing the histogram bars where the width of each bar is `1`, find the area of the largest rectangle in the histogram.

---

## Example

### Input

```text
heights = [2,1,5,6,2,3]
```

### Output

```text
10
```

### Explanation

The largest rectangle is:

```
height = 5
width  = 2

Area = 5 × 2

     5 6
     █ █
     █ █
   █ █ █ █
2  1 5 6 2 3

Area = 10
```

---

# 🧠 Approach

## Monotonic Stack

We maintain a stack that stores:

```
Index of increasing height bars
```

Example:

```
height increasing:

2
2,5
2,5,6
```

When a smaller height comes:

```
5,6,2
```

we calculate rectangles for bars that cannot extend further.

---

# Why Store Index Instead of Height?

Because area needs:

```
Area = height × width
```

To calculate width, we need positions.

Example:

```
[2,5,6]

Index:
 0 1 2
```

If height `6` is removed:

Previous smaller:

```
index 0
```

Current:

```
index 3
```

Width:

```
3-0-1

=2
```



# Java Code

```java
class Solution {

    public int largestRectangleArea(int[] h) {


        Stack<Integer> st = new Stack<>();

        st.push(-1);

        int maxarea = 0;



        for(int i = 0; i < h.length; i++){


            while(st.peek()!=-1 
                  && h[i] <= h[st.peek()]){


                int hei = h[st.pop()];


                int width = i - st.peek() - 1;


                maxarea = Math.max(
                    maxarea,
                    hei * width
                );

            }


            st.push(i);

        }



        while(st.peek()!=-1){


            int hei = h[st.pop()];


            int width = h.length - st.peek() - 1;


            maxarea = Math.max(
                maxarea,
                hei * width
            );

        }


        return maxarea;

    }
}
```
