# LeetCode 2104: Sum of Subarray Ranges

## Problem Statement 

You are given an array:

```
nums = [1,2,3]
```

For every possible subarray:

```
Range = Maximum value - Minimum value
```

You need to return the **sum of all ranges**.

---

# Example

```
nums = [1,2,3]
```

All subarrays:

| Subarray | Minimum | Maximum | Range |
| -------- | ------- | ------- | ----- |
| [1]      | 1       | 1       | 0     |
| [2]      | 2       | 2       | 0     |
| [3]      | 3       | 3       | 0     |
| [1,2]    | 1       | 2       | 1     |
| [2,3]    | 2       | 3       | 1     |
| [1,2,3]  | 1       | 3       | 2     |

Total:

```
0+0+0+1+1+2 = 4
```

Answer:

```
4
```

---

# Brute Force Approach

For every subarray:

1. Find minimum
2. Find maximum
3. Calculate:

```
max-min
```

Example:

```
[1,2,3]

max = 3
min = 1

range = 3-1=2
```

Complexity:

```
O(n²)
```

For large input → Time Limit Exceeded.

---

# Optimized Idea

Observe:

```
Range = Maximum - Minimum
```

For all subarrays:

```
Sum of Ranges

= Sum of all Maximums
        -
  Sum of all Minimums
```

So:

```
Answer = Total Maximum Contribution - Total Minimum Contribution
```

---

# Main Concept: Contribution Technique

Instead of generating subarrays.

Ask:

## How many times does each element act as minimum?

Example:

```
nums=[1,2,3]
```

For element:

```
2
```

It is minimum in:

```
[2]
[1,2]
```

Contribution:

```
2 * 2

=4
```

---

Formula:

```
Contribution = value * left choices * right choices
```

---

# Understanding Left and Right

For every element:

```
nums[i]
```

Find:

### Left

Previous smaller element.

How many elements can extend towards left?

### Right

Next smaller element.

How many elements can extend towards right?

# LEFT PASS (Minimum)

Code:

```java
while(!stack.isEmpty() &&
arr[stack.peek()] >= arr[i])
{
    stack.pop();
}
```

For minimum:

We remove bigger or equal values.

# Complete Code Flow

```java
public long subArrayRanges(int[] nums){

    long sumMax=getSum(nums,false);

    long sumMin=getSum(nums,true);

    return sumMax-sumMin;
}
```

First:

```
Calculate maximum contribution
```

Second:

```
Calculate minimum contribution
```

Finally:

```
Subtract
```

---

# Why Two Different Stack Conditions?

## Minimum

Previous smaller:

```
>=
```

Next smaller:

```
>
```

Example:

```
[2,2]
```

Equal values must not count twice.

---

## Maximum

Previous greater:

```
<=
```

Next greater:

```
<
```
