# 📌 Problem Statement

You are given a stack `st`.

Reverse the stack so that the bottom element becomes the top element.

---

## Example 1

### Input

```
Top
 ↓

4
3
2
1
```

(Stack representation)

```
[1,2,3,4]
```

---

### Output

```
Top
 ↓

1
2
3
4
```

(Stack representation)

```
[4,3,2,1]
```

---

## Example 2

### Input

```
Top
 ↓

1
2
3
```

(Stack representation)

```
[3,2,1]
```

---

### Output

```
Top
 ↓

3
2
1
```

(Stack representation)

```
[1,2,3]
```

---

# 💡 Approach

Use an **extra temporary stack**.

Steps:

1. Pop every element from the original stack.
2. Push it into another stack.
3. The order becomes reversed.

---

# 🔄 Algorithm

## Step 1

Original Stack

```
Top
 ↓

4
3
2
1
```

Temporary Stack

```
EMPTY
```

---

## Step 2

Repeat until original stack becomes empty.

```
temp.push(stack.pop());
```

---

## Step 3

Return the temporary stack.

# ✅ Java Implementation

```java
import java.util.Stack;

public class ReverseStack {

    public static Stack<Integer> reverse(Stack<Integer> stack) {

        Stack<Integer> temp = new Stack<>();

        while(!stack.isEmpty()) {

            temp.push(stack.pop());

        }

        return temp;
    }

    public static void main(String[] args) {

        Stack<Integer> stack = new Stack<>();

        stack.push(1);
        stack.push(2);
        stack.push(3);
        stack.push(4);

        System.out.println("Original Stack: " + stack);

        Stack<Integer> reversed = reverse(stack);

        System.out.println("Reversed Stack: " + reversed);

    }
}
```
