# 🔷 Baseball Game (LeetCode 682)

## 📌 Problem Statement

You are keeping scores for a baseball game.

Initially, the score record is **empty**.

You are given an array of strings `operations`, where each operation is one of the following:

| Operation | Meaning |
|-----------|---------|
| Integer `x` | Add score `x` |
| `"+"` | Add a score equal to the sum of the previous two scores |
| `"D"` | Add a score equal to double the previous score |
| `"C"` | Remove the previous score |

Return the **sum of all scores** after performing all operations.

---

# 🧩 Example

### Input

```text
operations = ["5","2","C","D","+"]
```

### Output

```text
30
```

---

# 💡 Why Do We Use a Stack?

Every operation works on the **most recent score**.

Examples:

- Remove last score
- Double last score
- Sum last two scores

A **Stack** is perfect because it follows:

```
LIFO

Last In

First Out
```

The latest score is always on the top.

# ✅ Java Implementation

```java
import java.util.Stack;

class Solution {

    public int calPoints(String[] operations) {

        Stack<Integer> st = new Stack<>();


        for(int i = 0; i < operations.length; i++) {

            String current = operations[i];


            if(current.equals("C")) {

                st.pop();

            }

            else if(current.equals("D")) {

                st.push(2 * st.peek());

            }

            else if(current.equals("+")) {

                int last = st.pop();

                int secondLast = st.peek();

                st.push(last);

                st.push(last + secondLast);

            }

            else {

                st.push(Integer.parseInt(current));

            }

        }


        int sum = 0;

        for(int i = 0; i < st.size(); i++) {

            sum += st.get(i);

        }

        return sum;

    }

}
```
