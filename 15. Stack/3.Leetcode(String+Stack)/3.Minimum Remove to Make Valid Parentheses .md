## 📌 Problem Statement

Given a string `s` containing lowercase letters and parentheses `'('` and `')'`.

Remove the **minimum number of parentheses** so that the resulting string is valid.

A valid parentheses string means:

- Every opening bracket `(` has a matching closing bracket `)`.
- Every closing bracket `)` has a matching opening bracket `(`.
- Letters remain unchanged.

Return the valid string.

---

# 🧩 Example 1

## Input

```text
s = "a)b(c)d"
```

## Output

```text
"ab(c)d"
```

Explanation

```
Original

a ) b ( c ) d
  ↑

')' has no matching '('

Remove it.

Result

ab(c)d
```

---

# 🧩 Example 2

## Input

```text
s = "))(("
```

## Output

```text
""
```

Explanation

All parentheses are invalid.

---

# 🧩 Example 3

## Input

```text
s = "(a(b(c)d)"
```

## Output

```text
"a(b(c)d)"
```

Explanation

One extra `'('` is removed.


# 🔄 Algorithm

### Step 1

Create a StringBuilder.

```java
StringBuilder st = new StringBuilder();
```

---

### Step 2

Traverse from left to right.

Keep a count of unmatched `'('`.

---

### Step 3

If character is `'('`

- Increase count.
- Append it.

---

### Step 4

If character is `')'`

Append only if there is an unmatched `'('`.

Otherwise ignore it.

---

### Step 5

Letters are always appended.

---

### Step 6

Now traverse backwards.

Remove remaining unmatched `'('`.

---

### Step 7

Reverse the result.

Return it.

# ✅ Java Implementation

```java
class Solution {

    public String minRemoveToMakeValid(String s) {

        StringBuilder st = new StringBuilder();

        int open = 0;

        // Pass 1 : Remove invalid ')'
        for(int i = 0; i < s.length(); i++) {

            char ch = s.charAt(i);

            if(ch == '(') {

                open++;
                st.append(ch);

            }

            else if(ch == ')') {

                if(open > 0) {

                    open--;
                    st.append(ch);

                }

            }

            else {

                st.append(ch);

            }

        }

        StringBuilder res = new StringBuilder();

        // Pass 2 : Remove extra '('
        for(int i = st.length() - 1; i >= 0; i--) {

            char ch = st.charAt(i);

            if(ch == '(' && open > 0) {

                open--;

            }

            else {

                res.append(ch);

            }

        }

        return res.reverse().toString();

    }

}
```
