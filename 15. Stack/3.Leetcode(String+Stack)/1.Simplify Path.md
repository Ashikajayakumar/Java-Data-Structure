# 🔷 Simplify Path (LeetCode 71)

## 📌 Problem Statement

Given a Unix-style absolute file path, simplify it.

The simplified path should follow these rules:

- `/` → Root directory.
- `.` → Current directory (ignore it).
- `..` → Parent directory (go back one folder).
- Multiple slashes (`//`, `///`) are treated as a single slash.
- Any other name (including `...`, `....`) is treated as a normal folder.

Return the simplified canonical path.

# ✅ Java Implementation

```java
import java.util.Stack;

class Solution {

    public String simplifyPath(String path) {

        Stack<String> st = new Stack<>();

        String[] parts = path.split("/");

        for(int i = 0; i < parts.length; i++) {

            String current = parts[i];

            if(current.equals("") || current.equals(".")) {

                continue;

            }

            else if(current.equals("..")) {

                if(!st.isEmpty()) {

                    st.pop();

                }

            }

            else {

                st.push(current);

            }

        }

        StringBuilder result = new StringBuilder();

        for(int i = 0; i < st.size(); i++) {

            result.append("/").append(st.get(i));

        }

        return result.length() == 0 ? "/" : result.toString();

    }

}
```
