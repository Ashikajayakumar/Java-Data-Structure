# 🔷 Make The String Great (LeetCode 1544)

## 📌 Problem Statement

Given a string `s` containing uppercase and lowercase English letters.

A string is called **good** if there are **no adjacent characters** such that:

- They are the **same letter**.
- But one is lowercase and the other is uppercase.

For example,

```
aA
Aa
bB
Bb
```

are **bad pairs**.

Keep removing bad adjacent pairs until no more exist.

Return the final string.

# 💡 ASCII Trick

Uppercase and lowercase letters differ by **32** in ASCII.

Examples

| Character | ASCII |
|-----------|------:|
| A | 65 |
| a | 97 |
| B | 66 |
| b | 98 |
| C | 67 |
| c | 99 |

Difference

```
97 - 65 = 32

98 - 66 = 32

99 - 67 = 32
```

So,

```java
Math.abs(ch1 - ch2) == 32
```

means

```
Same letter

Different case
```

# ✅ Java Implementation

```java
import java.util.Stack;

class Solution {

    public String makeGood(String s) {

        if(s.length() == 0)
            return s;

        Stack<Character> st = new Stack<>();

        StringBuilder sb = new StringBuilder();

        for(char ch : s.toCharArray()) {

            if(!st.isEmpty() &&
               Math.abs(st.peek() - ch) == 32) {

                st.pop();

            }

            else {

                st.push(ch);

            }

        }

        while(!st.isEmpty()) {

            sb.insert(0, st.pop());

        }

        return sb.toString();

    }

}
```
