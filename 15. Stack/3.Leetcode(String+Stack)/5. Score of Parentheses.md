# 🔷 Score of Parentheses (LeetCode 856)

## 📌 Problem Statement

Given a balanced parentheses string `s`, compute its score.

The scoring rules are:

```
"()" = 1

AB = A + B

(A) = 2 × A
```

where `A` and `B` are balanced parentheses strings.

Return the score of the given parentheses string.

Create a stack.

Push `0`.

```java

st.push(0);
```

---

### Step 2

Traverse the string.

---

### Step 3

If character is `'('`

Push another `0`.

This starts a new score level.

---

### Step 4

If character is `')'`

Pop the current score.

```
v = st.pop()
```

If

```
v == 0
```

then

```
()

Score = 1
```

Otherwise

```
(A)

Score = 2 × A
```

Finally,

add this score to its parent level.

# ✅ Java Implementation

```java
import java.util.Stack;

class Solution {

    public int scoreOfParentheses(String s) {

        Stack<Integer> st = new Stack<>();

        st.push(0);

        for(int i = 0; i < s.length(); i++) {

            char ch = s.charAt(i);

            if(ch == '(') {

                st.push(0);

            }

            else {

                int v = st.pop();

                int score = (v == 0) ? 1 : 2 * v;

                st.push(score + st.pop());

            }

        }

        return st.pop();

    }

}
```
