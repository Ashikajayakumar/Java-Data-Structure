## 📌 Problem Statement

Given a string `s` containing:

- Non-negative integers
- Operators:
    - `+`
    - `-`
    - `*`
    - `/`
- Spaces

Evaluate the expression and return the result.

---

## Rules

- Multiplication and division have higher priority than addition and subtraction.
- Division truncates toward zero.
- Expression is always valid.

---

# Example 1

## Input

```
s = "3+2*2"
```

## Output

```
7
```

### Explanation

```
3 + (2 * 2)

= 3 + 4

= 7
```

# 🧠 Main Idea

We use:

```
Stack<Integer>
```

Because:

- `+` and `-` can be stored directly.
- `*` and `/` need immediate calculation.

Example:

```
3+2*2
```

When we see:

```
+
```

Store 3.

Stack:

```
[3]
```

When we see:

```
*
```

Calculate immediately:

```
2*2 = 4
```

Store:

```
[3,4]
```

Finally:

```
3+4 = 7
```

# Java Implementation

```java
import java.util.Stack;

class Solution {

    public int calculate(String s) {

        Stack<Integer> st = new Stack<>();

        int num = 0;

        char sign = '+';


        for(int i = 0; i < s.length(); i++){

            char ch = s.charAt(i);


            // Build number

            if(Character.isDigit(ch)){

                num = num * 10 + (ch-'0');

            }


            // Operator or last character

            if((!Character.isDigit(ch) && ch!=' ')
                    || i == s.length()-1){


                if(sign == '+'){

                    st.push(num);

                }

                else if(sign == '-'){

                    st.push(-num);

                }

                else if(sign == '*'){

                    st.push(st.pop()*num);

                }

                else if(sign == '/'){

                    st.push(st.pop()/num);

                }


                sign = ch;

                num = 0;

            }

        }


        int result = 0;


        while(!st.isEmpty()){

            result += st.pop();

        }


        return result;

    }
}
```
