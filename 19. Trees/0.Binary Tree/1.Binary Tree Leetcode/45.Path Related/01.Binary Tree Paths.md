## Problem

Given the root of a binary tree, return all root-to-leaf paths.

A **leaf node** is a node with no left and right child.

---

## Example

### Input

```text
        1
       / \
      2   3
       \
        5
```

### Output

```text
["1->2->5", "1->3"]
```

### Explanation

Paths from root to leaf:

```text
1 → 2 → 5

1 → 3
```


## Simple Java Code

```java
class Solution {

    public List<String> binaryTreePaths(TreeNode root) {

        List<String> result = new ArrayList<>();

        dfs(root, "", result);

        return result;
    }

    private void dfs(TreeNode node, String path, List<String> result) {

        if (node == null)
            return;

        path += node.val;

        if (node.left == null && node.right == null) {
            result.add(path);
            return;
        }

        path += "->";

        dfs(node.left, path, result);
        dfs(node.right, path, result);
    }
}
```

## Dry Run

Input:

```text
        1
       / \
      2   3
       \
        5
```

### Visit 1

```text
path = "1->"
```

---

### Go Left (2)

```text
path = "1->2->"
```

---

### Go Right (5)

```text
path = "1->2->5"
```

Leaf node found.

Add:

```text
["1->2->5"]
```

---

### Back to 1 → Go Right (3)

```text
path = "1->3"
```

Leaf node found.

Add:

```text
["1->2->5", "1->3"]
```



**Key Insight:**
Build the path while traversing the tree and save it whenever a leaf node is reached. This is a classic **Root-to-Leaf DFS** pattern.
