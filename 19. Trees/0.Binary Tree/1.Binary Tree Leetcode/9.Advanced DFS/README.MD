# Advanced DFS

## Prerequisites

Before starting this section, make sure you are comfortable with:

* Binary Tree DFS Traversals
* Recursion
* Tree Height and Depth Calculations
* BFS Traversal
* Backtracking
* Path-Based Tree Problems
