## Problem

Given the root of a binary tree, flatten it into a linked list **in-place**.

The linked list should follow the same order as **Preorder Traversal**:

```text
Root → Left → Right
```

After flattening:

* `left` child should be `null`
* `right` child acts as the next pointer

---

## Example

### Input

```text
        1
       / \
      2   5
     / \   \
    3   4   6
```

### Output

```text
1
 \
  2
   \
    3
     \
      4
       \
        5
         \
          6
```

Preorder Traversal:

```text
1 → 2 → 3 → 4 → 5 → 6
```



## Simple Java Code

```java
class Solution {

    public void flatten(TreeNode root) {

        if (root == null)
            return;

        flatten(root.left);
        flatten(root.right);

        TreeNode left = root.left;
        TreeNode right = root.right;

        root.left = null;
        root.right = left;

        TreeNode curr = root;

        while (curr.right != null) {
            curr = curr.right;
        }

        curr.right = right;
    }
}
```

## Optimized Java Code (Recommended)

Uses reverse preorder traversal:

```java
class Solution {

    TreeNode prev = null;

    public void flatten(TreeNode root) {

        if (root == null)
            return;

        flatten(root.right);
        flatten(root.left);

        root.right = prev;
        root.left = null;

        prev = root;
    }
}
```
