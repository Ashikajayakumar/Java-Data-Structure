
# Same Tree(Leetcode No-100)

## Problem

Given two binary trees `p` and `q`:

Return:

* `true` → if both trees are identical
* `false` → otherwise

Trees are same when:

1. Structure is same
2. Node values are same

# Main Logic

At every node check:

1. Both are null → true
2. One is null → false
3. Values different → false
4. Check left subtree
5. Check right subtree

---

# Recursive Formula

Same(p,q)=Same(p.left,q.left)\land Same(p.right,q.right)

---

# Easy LeetCode Java Solution

```java id="njlwm2"
class Solution {

    public boolean isSameTree(TreeNode p, TreeNode q) {

        // Both null
        if (p == null && q == null) {
            return true;
        }

        // One is null
        if (p == null || q == null) {
            return false;
        }

        // Values different
        if (p.val != q.val) {
            return false;
        }

        // Check left and right
        return isSameTree(p.left, q.left) &&
               isSameTree(p.right, q.right);
    }
}
```
