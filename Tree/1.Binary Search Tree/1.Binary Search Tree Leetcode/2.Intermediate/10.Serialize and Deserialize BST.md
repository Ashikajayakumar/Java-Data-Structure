# LeetCode 449: Serialize and Deserialize BST

## Problem

Design an algorithm to:

### Serialize

Convert a BST into a string.

```text
BST → String
```

### Deserialize

Convert the string back into the original BST.

```text
String → BST
```

---

## Example

### BST

```text
       5
      / \
     3   7
    / \   \
   2   4   8
```

### Serialize

```text
"5,3,2,4,7,8"
```

### Deserialize

Rebuild the same BST.

---

# Key Observation

Because it is a **BST**, we don't need null markers (`#`).

### Preorder Traversal

```text
Root → Left → Right
```

For the above tree:

```text
5,3,2,4,7,8
```

This preorder sequence uniquely identifies the BST.

---

# Serialization

Perform preorder traversal and store values.

## Code

```java
public String serialize(TreeNode root) {

    StringBuilder sb = new StringBuilder();

    preorder(root, sb);

    return sb.toString();
}

private void preorder(TreeNode root, StringBuilder sb) {

    if (root == null) {
        return;
    }

    sb.append(root.val).append(",");

    preorder(root.left, sb);
    preorder(root.right, sb);
}
```

### Example

```text
       5
      / \
     3   7
    / \   \
   2   4   8
```

Preorder:

```text
5 → 3 → 2 → 4 → 7 → 8
```

Serialized:

```text
"5,3,2,4,7,8,"
```

# Deserialization

We use BST bounds.

### Idea

While reading preorder:

```text
5,3,2,4,7,8
```

* Root = 5
* Values < 5 go to left subtree
* Values > 5 go to right subtree

Maintain:

```java
lowerBound
upperBound
```

for each recursive call.

---

# Complete Code

```java
class Codec {

    int index = 0;

    public String serialize(TreeNode root) {

        StringBuilder sb = new StringBuilder();
        preorder(root, sb);

        return sb.toString();
    }

    private void preorder(TreeNode root, StringBuilder sb) {

        if (root == null) {
            return;
        }

        sb.append(root.val).append(",");

        preorder(root.left, sb);
        preorder(root.right, sb);
    }

    public TreeNode deserialize(String data) {

        if (data.isEmpty()) {
            return null;
        }

        String[] arr = data.split(",");

        List<Integer> preorder = new ArrayList<>();

        for (String s : arr) {
            if (!s.isEmpty()) {
                preorder.add(Integer.parseInt(s));
            }
        }

        index = 0;

        return build(preorder, Integer.MIN_VALUE, Integer.MAX_VALUE);
    }

    private TreeNode build(List<Integer> preorder,
                           int low,
                           int high) {

        if (index == preorder.size()) {
            return null;
        }

        int val = preorder.get(index);

        if (val < low || val > high) {
            return null;
        }

        index++;

        TreeNode root = new TreeNode(val);

        root.left = build(preorder, low, val);
        root.right = build(preorder, val, high);

        return root;
    }
}
```
