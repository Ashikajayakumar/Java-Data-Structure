# LeetCode 897: Increasing Order Search Tree

## Problem 
  
Given a BST, rearrange it so that: 
    
* The left child of every node is `null`.    
* The tree becomes a right-skewed tree. 
* Nodes appear in **increasing order**.  

---

## Example

### Input BST

```text
      5
     / \
    3   6
   / \   \
  2   4   8
 /       / \
1       7   9
```

### Output

```text
1
 \
  2
   \
    3
     \
      4
       \
        5
         \
          6
           \
            7
             \
              8
               \
                9
```

Notice:

```text
left = null for every node
```

and the nodes are in sorted order.

---

# Key Observation

For a BST:

```text
Inorder Traversal = Sorted Order
```

If we visit nodes in inorder:

```text
1 → 2 → 3 → 4 → 5 → 6 → 7 → 8 → 9
```

we can connect them using only the right pointers.

---

# Approach 1: Dummy Node + Inorder Traversal

Maintain:

```java
TreeNode dummy = new TreeNode(-1);
TreeNode current = dummy;
```

During inorder traversal:

1. Visit left subtree.
2. Connect current node to the right of `current`.
3. Set node's left child to `null`.
4. Move `current` forward.
5. Visit right subtree.

---

## Code

```java
class Solution {

    TreeNode current;

    public TreeNode increasingBST(TreeNode root) {

        TreeNode dummy = new TreeNode(-1);

        current = dummy;

        inorder(root);

        return dummy.right;
    }

    private void inorder(TreeNode root) {

        if (root == null) {
            return;
        }

        inorder(root.left);

        root.left = null;

        current.right = root;
        current = root;

        inorder(root.right);
    }
}
```

## Visit Node 1

Inorder reaches node 1 first.

```text
current = dummy
```

Set:

```java
root.left = null;

current.right = root;
```

Result:

```text
dummy -> 1
```

Move:

```java
current = root;
```

Now:

```text
current = 1
```

---

## Visit Node 2

Set:

```java
current.right = root;
```

Result:

```text
dummy -> 1 -> 2
```

Move:

```text
current = 2
```

---

## Visit Node 3

Set:

```java
current.right = root;
```

Result:

```text
dummy -> 1 -> 2 -> 3
```

Move:

```text
current = 3
```

---

## Return

```java
return dummy.right;
```

which is:

```text
1
 \
  2
   \
    3
```
