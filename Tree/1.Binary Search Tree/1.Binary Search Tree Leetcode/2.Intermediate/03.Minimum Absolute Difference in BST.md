# 530. Minimum Absolute Difference in BST

Given a BST, find the **minimum absolute difference between values of any two different nodes**.

---

## Key Observation

For a BST:

```text
Inorder Traversal = Sorted Order
```

Example:

```text
      4
     / \
    2   6
   / \
  1   3
```

Inorder:

```text
1 → 2 → 3 → 4 → 6
```

The minimum difference will always occur between **adjacent elements in the sorted order**.

For the above:

```text
|2-1| = 1
|3-2| = 1
|4-3| = 1
|6-4| = 2
```

Answer:

```text
1
```

---

# Approach 1: Inorder Traversal + List

### Idea

1. Store inorder traversal in a list.
2. Compare adjacent elements.
3. Keep track of the minimum difference.

### Code

```java
class Solution {

    public int getMinimumDifference(TreeNode root) {

        List<Integer> inorder = new ArrayList<>();

        dfs(root, inorder);

        int minDiff = Integer.MAX_VALUE;

        for (int i = 1; i < inorder.size(); i++) {
            minDiff = Math.min(
                minDiff,
                inorder.get(i) - inorder.get(i - 1)
            );
        }

        return minDiff;
    }

    private void dfs(TreeNode root, List<Integer> inorder) {

        if (root == null) {
            return;
        }

        dfs(root.left, inorder);
        inorder.add(root.val);
        dfs(root.right, inorder);
    }
}
```

## Optimized Approach (Interview Preferred)

Instead of storing all nodes:

* Keep track of the previous node visited during inorder traversal.
* Compute difference immediately.

### Variables

```java
prev = previously visited node value
ans = minimum difference
```

---

### Code

```java
class Solution {

    Integer prev = null;
    int ans = Integer.MAX_VALUE;

    public int getMinimumDifference(TreeNode root) {

        inorder(root);

        return ans;
    }

    private void inorder(TreeNode root) {

        if (root == null) {
            return;
        }

        inorder(root.left);

        if (prev != null) {
            ans = Math.min(ans, root.val - prev);
        }

        prev = root.val;

        inorder(root.right);
    }
}
```

# Iterative Inorder Solution

```java
class Solution {

    public int getMinimumDifference(TreeNode root) {

        Stack<TreeNode> st = new Stack<>();

        TreeNode curr = root;

        Integer prev = null;

        int ans = Integer.MAX_VALUE;

        while (curr != null || !st.isEmpty()) {

            while (curr != null) {
                st.push(curr);
                curr = curr.left;
            }

            curr = st.pop();

            if (prev != null) {
                ans = Math.min(ans, curr.val - prev);
            }

            prev = curr.val;

            curr = curr.right;
        }

        return ans;
    }
}
```
