# LeetCode 1382: Balance a Binary Search Tree
<img width="1024" height="1536" alt="image" src="https://github.com/user-attachments/assets/85341f11-f93d-4184-93da-56bf77596023" />

## Problem

Given a BST, return a **balanced BST** containing the same node values. 
 
A balanced BST means:  
 
```text 
For every node:
|height(left) - height(right)| <= 1 
```

---

## Example

### Input (Skewed BST)

```text
    1
     \
      2
       \
        3
         \
          4
```

This is a valid BST but not balanced.

### Output

```text
      2
     / \
    1   3
         \
          4
```

or

```text
      3
     / \
    2   4
   /
  1
```

Any balanced BST is accepted.



# Approach

### Step 1

Store inorder traversal in a list.

```java
inorder(root, list);
```

Result:

```text
[1,2,3,4]
```

---

### Step 2

Choose middle element as root.

```text
mid = (left + right) / 2
```

This keeps the tree balanced.

---

### Step 3

Recursively build:

```text
Left BST  -> left half
Right BST -> right half
```

---

# Code

```java
class Solution {

    public TreeNode balanceBST(TreeNode root) {

        List<Integer> inorder = new ArrayList<>();

        inorder(root, inorder);

        return build(inorder, 0, inorder.size() - 1);
    }

    private void inorder(TreeNode root, List<Integer> list) {

        if (root == null) {
            return;
        }

        inorder(root.left, list);
        list.add(root.val);
        inorder(root.right, list);
    }

    private TreeNode build(List<Integer> list, int left, int right) {

        if (left > right) {
            return null;
        }

        int mid = left + (right - left) / 2;

        TreeNode root = new TreeNode(list.get(mid));

        root.left = build(list, left, mid - 1);
        root.right = build(list, mid + 1, right);

        return root;
    }
}
```

# Complete Dry Run

## Input

```text
1
 \
  2
   \
    3
     \
      4
```

---

## Step 1: Inorder Traversal

Visit nodes:

```text
1 → 2 → 3 → 4
```

Store:

```text
list = [1,2,3,4]
```

---

## Step 2: Build Balanced BST

Call:

```java
build(list, 0, 3)
```

---

### First Call

```text
left = 0
right = 3

mid = 1
```

```text
root = 2
```

Tree:

```text
    2
```

---

### Left Subtree

Call:

```java
build(list, 0, 0)
```

```text
mid = 0
root = 1
```

Tree:

```text
    2
   /
  1
```

---

### Right Subtree

Call:

```java

```

```text

```

Tree:

```text
    2
   / \
  1   3
```

---

### Right of 3

Call:

```java

```


Balanced BST obtained.
