# 669. Trim a Binary Search Tree

Given a BST and two integers `low` and `high`, trim the tree so that:

```text
low <= node.val <= high
```

All nodes outside this range must be removed, while preserving the BST structure.


## Cases

### Case 1: root.val < low

```text
      3
     / \
    1   5

low = 4
```

Since:

```text
1 < 3 < 4
```

Everything in the left subtree is also less than `low`.

So:

```java
return trimBST(root.right, low, high);
```

---

### Case 2: root.val > high

```text
      8
     / \
    5   10

high = 7
```

Since:

```text
10 > 8 > 7
```

Everything in the right subtree is also greater than `high`.

So:

```java
return trimBST(root.left, low, high);
```

---

### Case 3: Node is Valid

If:

```text
low <= root.val <= high
```

Keep the node and recursively trim both children.

```java
root.left = trimBST(root.left, low, high);
root.right = trimBST(root.right, low, high);
```

---

# Recursive Solution

```java
class Solution {

    public TreeNode trimBST(TreeNode root, int low, int high) {

        if (root == null) {
            return null;
        }

        if (root.val < low) {
            return trimBST(root.right, low, high);
        }

        if (root.val > high) {
            return trimBST(root.left, low, high);
        }

        root.left = trimBST(root.left, low, high);
        root.right = trimBST(root.right, low, high);

        return root;
    }
}
```

# Dry Run

### Input

```text
        3
       / \
      0   4
       \
        2
       /
      1

low = 1
high = 3
```

---

### Node 3

```text
1 <= 3 <= 3
```

Keep it.

---

### Node 0

```text
0 < low
```

Discard node 0 and its left subtree.

Return:

```text
trimBST(2)
```

---

### Node 2

```text
1 <= 2 <= 3
```

Keep.

---

### Node 1

```text
1 <= 1 <= 3
```

Keep.

---

### Node 4

```text
4 > high
```

Discard 4 and its right subtree.

Return:

```text
null
```

---

### Result

```text
      3
     /
    2
   /
  1
```


This leads directly to the elegant recursive solution above.
