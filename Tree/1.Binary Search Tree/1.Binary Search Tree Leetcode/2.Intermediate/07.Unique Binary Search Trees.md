## LeetCode 96: Unique Binary Search Trees
<img width="1024" height="1536" alt="image" src="https://github.com/user-attachments/assets/a6434849-4b6f-4b81-bebf-37c056ebfdf3" />

### Problem

Given `n`, return the number of structurally unique BSTs that store values `1` to `n`.

### Example

For:

```text
n = 3
```

Possible BSTs:

```text
    1          1           2           3          3
     \          \         / \         /          /
      2          3       1   3       2          1
       \        /                       \         \
        3      2                         1         2
```

Answer:

```text
5
```
---

## DP Solution

```java
class Solution {
    public int numTrees(int n) {

        int[] dp = new int[n + 1];

        dp[0] = 1;
        dp[1] = 1;

        for (int nodes = 2; nodes <= n; nodes++) {

            for (int root = 1; root <= nodes; root++) {

                int left = root - 1;
                int right = nodes - root;

                dp[nodes] += dp[left] * dp[right];
            }
        }

        return dp[n];
    }
}
```


## DP Formula

Let:

```text
dp[k] = number of unique BSTs using k nodes
```

Then:

```text
dp[n] = Σ(dp[left] × dp[right])
```

where:

```text
left = i - 1
right = n - i
```

for every root `i`.

---

## Base Cases

```java
dp[0] = 1
dp[1] = 1
```

Why `dp[0] = 1`?

An empty subtree is considered one valid possibility.

---

## Dry Run for n = 3

### Initialize

```text
dp[0] = 1
dp[1] = 1
```

---

### Nodes = 2

#### Root = 1

```text
left = 0
right = 1

dp[0] × dp[1]
=
1 × 1 = 1
```

#### Root = 2

```text
left = 1
right = 0

dp[1] × dp[0]
=
1 × 1 = 1
```

Total:

```text
dp[2] = 2
```

---

### Nodes = 3

#### Root = 1

```text
left = 0
right = 2

1 × 2 = 2
```

#### Root = 2

```text
left = 1
right = 1

1 × 1 = 1
```

#### Root = 3

```text
left = 2
right = 0

2 × 1 = 2
```

Total:

```text
dp[3] = 2 + 1 + 2 = 5
```

Answer:

```text
5
```
