## LeetCode 96: Unique Binary Search Trees

### Problem

Given `n`, return the number of structurally unique BSTs that store values `1` to `n`.

### Example

For:

```text
n = 3
```

Possible BSTs:

```text
    1          1           2           3          3
     \          \         / \         /          /
      2          3       1   3       2          1
       \        /                       \         \
        3      2                         1         2
```

Answer:

```text
5
```

---

## Key Idea

For every number `i` from `1` to `n`, choose it as the root.

Then:

* Left subtree contains values `< i`
* Right subtree contains values `> i`

If:

```text
leftCount = number of BSTs in left subtree
rightCount = number of BSTs in right subtree
```

Then:

```text
total BSTs with root i
=
leftCount × rightCount
```

because every left BST can be paired with every right BST.

---

## DP Formula

Let:

```text
dp[k] = number of unique BSTs using k nodes
```

Then:

```text
dp[n] = Σ(dp[left] × dp[right])
```

where:

```text
left = i - 1
right = n - i
```

for every root `i`.

---

## Base Cases

```java
dp[0] = 1
dp[1] = 1
```

Why `dp[0] = 1`?

An empty subtree is considered one valid possibility.

### Initialize

### Nodes = 2

#### Root = 1

```text

dp[0] × dp[1]

```

#### Root = 2

```text
left = 1

dp[1] × dp[0]
=
1 × 1 = 1
```

Total:

```text
dp[2] = 2
```

---

### Nodes = 3

#### Root = 1

```text
left = 0
right = 2

1 × 2 = 2
```

#### Root = 2

```text
left = 1
right = 1

1 × 1 = 1
```

#### Root = 3

```text
left = 2
right = 0

2 × 1 = 2
```

Total:

```text
dp[3] = 2 + 1 + 2 = 5
```

```text
5
```

## DP Solution

```java
class Solution {
    public int numTrees(int n) {

        int[] dp = new int[n + 1];

        dp[0] = 1;
        dp[1] = 1;

        for (int nodes = 2; nodes <= n; nodes++) {

            for (int root = 1; root <= nodes; root++) {

                int left = root - 1;
                int right = nodes - root;

                dp[nodes] += dp[left] * dp[right];
            }
        }

        return dp[n];
    }
}
```

## Catalan Number

This problem follows the Catalan Number Sequence.

Formula:

[
C_n=\frac{(2n)!}{(n+1)!n!}
]

So:

```java
class Solution {
    public int numTrees(int n) {
        long c = 1;

        for (int i = 0; i < n; i++) {
            c = c * 2 * (2 * i + 1) / (i + 2);
        }

        return (int)c;
    }
}
```

## Interview Points

1. BST property:

   * Left subtree < Root < Right subtree.

2. Choosing each node as root divides the problem into:

   * Left subtree size = `root - 1`
   * Right subtree size = `n - root`

3. Total combinations:

```text
leftWays × rightWays
```

4. This is a classic **Catalan Number** problem.

### Complexity

**DP Solution**

* Time: `O(n²)`
* Space: `O(n)`

**Catalan Formula**

* Time: `O(n)`
* Space: `O(1)`
