## LeetCode 1305: All Elements in Two Binary Search Trees

### Problem

Given the roots of two BSTs, return a sorted list containing **all elements** from both trees.

### Example

Tree 1:

```text
    2
   / \
  1   4
```

Tree 2:

```text
    1
   / \
  0   3
```

Output:

```text
[0, 1, 1, 2, 3, 4]
```



# Approach 1: Inorder + Merge

## Step 1: Collect Sorted Elements

```java
inorder(root1) → list1
inorder(root2) → list2
```

Example:

```text
list1 = [1, 2, 4]
list2 = [0, 1, 3]
```

---

## Step 2: Merge

Use two pointers.

```text
i = 0
j = 0
```

Compare:

```text
1 vs 0
```

Take smaller:

```text
ans = [0]
j++
```

Compare:

```text
1 vs 1
```

Take either:

```text
ans = [0,1]
i++
```

Continue until both lists are exhausted.

Result:

```text
[0,1,1,2,3,4]
```

---

# Code

```java
class Solution {

    public List<Integer> getAllElements(TreeNode root1, TreeNode root2) {

        List<Integer> list1 = new ArrayList<>();
        List<Integer> list2 = new ArrayList<>();

        inorder(root1, list1);
        inorder(root2, list2);

        return merge(list1, list2);
    }

    private void inorder(TreeNode root, List<Integer> list) {

        if (root == null) {
            return;
        }

        inorder(root.left, list);
        list.add(root.val);
        inorder(root.right, list);
    }

    private List<Integer> merge(List<Integer> a, List<Integer> b) {

        List<Integer> ans = new ArrayList<>();

        int i = 0;
        int j = 0;

        while (i < a.size() && j < b.size()) {

            if (a.get(i) <= b.get(j)) {
                ans.add(a.get(i++));
            } else {
                ans.add(b.get(j++));
            }
        }

        while (i < a.size()) {
            ans.add(a.get(i++));
        }

        while (j < b.size()) {
            ans.add(b.get(j++));
        }

        return ans;
    }
}
```
