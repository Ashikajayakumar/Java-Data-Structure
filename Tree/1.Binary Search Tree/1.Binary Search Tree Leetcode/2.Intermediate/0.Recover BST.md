# Recover Binary Search Tree (LeetCode 99)

## Problem

Two nodes in a BST have been swapped by mistake.

Recover the tree **without changing its structure**.

---

## Example

Original BST:

```text
      2
     / \
    1   3
```

Suppose `2` and `3` are swapped:

```text
      3
     / \
    1   2
```

This is no longer a valid BST.

After recovery:

```text
      2
     / \
    1   3
```

---

# Key Observation

For a BST:

```text
Inorder Traversal = Sorted Order
```

Example:

```text
      4
     / \
    2   6
   / \ / \
  1  3 5  7
```

Inorder:

```text
1 2 3 4 5 6 7
```

Sorted.



## Case

Inorder:

```text
1 5 3 4 2 6
```

Violations:

```text
5 > 3
4 > 2
```

Then:

```text
first  = 5
second = 2
```

Swap them back.

# Simple Interview Code

```java
class Solution {

    TreeNode first = null;
    TreeNode second = null;
    TreeNode prev = null;

    public void recoverTree(TreeNode root) {

        inorder(root);

        int temp = first.val;
        first.val = second.val;
        second.val = temp;
    }

    private void inorder(TreeNode root) {

        if (root == null) {
            return;
        }

        inorder(root.left);

        if (prev != null && prev.val > root.val) {

            if (first == null) {
                first = prev;
            }

            second = root;
        }

        prev = root;

        inorder(root.right);
    }
}
```
