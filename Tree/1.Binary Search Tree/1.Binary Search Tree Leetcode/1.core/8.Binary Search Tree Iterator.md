# LeetCode 173 — BST Iterator

## Problem

Design an iterator over a BST.

Implement:

```java
BSTIterator(TreeNode root)
int next()
boolean hasNext()
```

### Example

BST:

```text
        7
       / \
      3   15
         /  \
        9   20
```

Inorder Traversal:

```text
3 7 9 15 20
```

The iterator should return:

```text
3 → 7 → 9 → 15 → 20
```

one by one.


# Simple Interview Solution

```java
class BSTIterator {

    Stack<TreeNode> stack = new Stack<>();

    public BSTIterator(TreeNode root) {
        pushLeft(root);
    }

    public int next() {

        TreeNode node = stack.pop();

        if (node.right != null) {
            pushLeft(node.right);
        }

        return node.val;
    }

    public boolean hasNext() {
        return !stack.isEmpty();
    }

    private void pushLeft(TreeNode node) {

        while (node != null) {
            stack.push(node);
            node = node.left;
        }
    }
}
```
