# Lowest Common Ancestor of a BST (LeetCode 235)

## What is LCA?

The **Lowest Common Ancestor (LCA)** of two nodes `p` and `q` is the lowest node in the tree that has both `p` and `q` as descendants. 

---

## Example

BST:

```text
          6
        /   \
       2     8
      / \   / \
     0   4 7   9
        / \
       3   5
```

Find LCA of:

```text
p = 2
q = 8
```

Answer:

```text
6
```

Because both nodes are in different subtrees of 6.

---

# BST Property Used

```text
Left Subtree < Root < Right Subtree
```

This allows us to avoid searching the entire tree.

---

# Key Idea

Suppose current node is `root`.

### Case 1

```text
p < root
q < root
```

Both nodes are in the left subtree.

```java
root = root.left;
```

---

### Case 2

```text
p > root
q > root
```

Both nodes are in the right subtree.

```java
root = root.right;
```

---

### Case 3

```text
p <= root <= q
```

(or vice versa)

Current node is the split point.

Current node = LCA.

---

# Simple Recursive Solution

```java
class Solution {

    public TreeNode lowestCommonAncestor(TreeNode root,
                                         TreeNode p,
                                         TreeNode q) {

        if (p.val < root.val && q.val < root.val) {
            return lowestCommonAncestor(root.left, p, q);
        }

        if (p.val > root.val && q.val > root.val) {
            return lowestCommonAncestor(root.right, p, q);
        }

        return root;
    }
}
```

---

# Iterative Solution (Interview Favorite)

```java
class Solution {

    public TreeNode lowestCommonAncestor(TreeNode root,
                                         TreeNode p,
                                         TreeNode q) {

        while (root != null) {

            if (p.val < root.val && q.val < root.val) {
                root = root.left;
            }
            else if (p.val > root.val && q.val > root.val) {
                root = root.right;
            }
            else {
                return root;
            }
        }

        return null;
    }
}
```

# Dry Run

Tree:

```text
          6
        /   \
       2     8
      / \   / \
     0   4 7   9
```

Find:

```text
p = 2
q = 8
```

---

### Iteration 1

Current:

```java
root = 6
```

Check:

```java
p.val < root.val && q.val < root.val
```

```text
2 < 6 = true
8 < 6 = false
```

False.

---

Check:

```java
p.val > root.val && q.val > root.val
```

```text
2 > 6 = false
```

False.

---

Both conditions fail.

Therefore:

```java
return root;
```

Answer:

```text
6
```

# Another Example

Find:

```text
p = 2
q = 4
```

Tree:

```text
          6
        /   \
       2     8
      / \
     0   4
```

---

### Iteration 1

```java
root = 6
```

Both values are smaller than 6.

Move left.

```java
root = root.left;
```

Now:

```java
root = 2
```

---

### Iteration 2

Check:

```text
2 < 2 = false
4 < 2 = false
```

Not both left.

```text
2 > 2 = false
```

Not both right.

Return:

```java
root
```

Answer:

```text
2
```
