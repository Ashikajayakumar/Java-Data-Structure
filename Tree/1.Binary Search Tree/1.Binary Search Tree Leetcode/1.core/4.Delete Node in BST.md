# LeetCode 450 — Delete Node in a BST

## Problem

Given the root of a BST and a key, delete the node with that key and return the updated BST.

---

## BST Property

```text
Left Subtree < Root < Right Subtree
```

---

# Three Cases of Deletion

## Case 1: Node has No Children (Leaf)

Tree:

```text
      50
     /
   30
```

Delete `30`

Result:

```text
      50
```

Simply remove the node.

---

## Case 2: Node has One Child

Tree:

```text
      50
     /
   30
   /
 20
```

Delete `30`

Result:

```text
      50
     /
   20
```

Replace the node with its child.

---

## Case 3: Node has Two Children

Tree:

```text
        50
       /  \
     30    70
          /  \
        60    80
```

Delete `70`

### Step 1

Find the inorder successor (smallest node in right subtree).

```text
Successor = 80
```

(or in a larger tree, the leftmost node of the right subtree)

### Step 2

Copy successor value.

```text
70 → 80
```

### Step 3

Delete the original successor node.

Result:

```text
        50
       /  \
     30    80
          /
        60
```

---

# Recursive Solution

```java
class Solution {

    public TreeNode deleteNode(TreeNode root, int key) {

        if (root == null) {
            return null;
        }

        if (key < root.val) {

            root.left = deleteNode(root.left, key);

        } else if (key > root.val) {

            root.right = deleteNode(root.right, key);

        } else {

            // Case 1 & 2
            if (root.left == null) {
                return root.right;
            }

            if (root.right == null) {
                return root.left;
            }

            // Case 3
            TreeNode successor = findMin(root.right);

            root.val = successor.val;

            root.right = deleteNode(root.right, successor.val);
        }

        return root;
    }

    private TreeNode findMin(TreeNode node) {

        while (node.left != null) {
            node = node.left;
        }

        return node;
    }
}
```

# Dry Run

Delete `50`

```text
          50
        /    \
      30      70
             /  \
           60    80
```

---

### Node Found

```text
key == root.val
```

Node `50` has two children.

---

### Find Successor

```text
Right subtree:

      70
     /
   60
```

Smallest node:

```text
60
```

Successor = 60

---

### Replace Value

```text
          60
        /    \
      30      70
             /  \
           60    80
```

---

### Delete Duplicate 60

Delete original successor node.

Final Tree:

```text
          60
        /    \
      30      70
                \
                 80
```
