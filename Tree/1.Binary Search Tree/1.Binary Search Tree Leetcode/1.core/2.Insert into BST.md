# LeetCode 701 — Insert into a Binary Search Tree
<img width="1024" height="1536" alt="image" src="https://github.com/user-attachments/assets/4ab04bd1-f72f-4364-9055-c9f0d8b72257" />

## Problem

Given the root of a BST and a value `val`, insert the value into the BST and return the root.

The BST property must remain valid.

---

## BST Rule

```text
Left Subtree < Root < Right Subtree
```

---

## Example

Insert `5` into:

```text
        4
       / \
      2   7
     / \
    1   3
```

Since:

```text
5 > 4  → Go Right
5 < 7  → Go Left
```

Result:

```text
        4
       / \
      2   7
     / \ /
    1  3 5
```

---

# Recursive Solution (Most Common)

```java
class Solution {

    public TreeNode insertIntoBST(TreeNode root, int val) {

        if (root == null) {
            return new TreeNode(val);
        }

        if (val < root.val) {
            root.left = insertIntoBST(root.left, val);
        } else {
            root.right = insertIntoBST(root.right, val);
        }

        return root;
    }
}
```

# Iterative Solution

```java
class Solution {

    public TreeNode insertIntoBST(TreeNode root, int val) {

        if (root == null) {
            return new TreeNode(val);
        }

        TreeNode curr = root;

        while (true) {

            if (val < curr.val) {

                if (curr.left == null) {
                    curr.left = new TreeNode(val);
                    break;
                }

                curr = curr.left;

            } else {

                if (curr.right == null) {
                    curr.right = new TreeNode(val);
                    break;
                }

                curr = curr.right;
            }
        }

        return root;
    }
}
```
