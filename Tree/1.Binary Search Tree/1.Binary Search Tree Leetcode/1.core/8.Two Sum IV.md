# LeetCode 653 — Two Sum IV: Input is a BST
<img width="1024" height="1535" alt="image" src="https://github.com/user-attachments/assets/0f844989-b91f-45e9-8d04-8501590f0130" />

## Problem

Given the root of a BST and an integer `k`, return `true` if there exist **two nodes** in the BST whose sum equals `k`.

---

## Example 1

```text
        5
       / \
      3   6
     / \   \
    2   4   7

k = 9
```

Possible pair:

```text
2 + 7 = 9
```

Output:

```text
true
```

---

## Example 2

```text
        5
       / \
      3   6
     / \   \
    2   4   7

k = 28
```

No pair exists.

Output:

```text
false
```

---

# Simple Interview Approach (HashSet)


## Code

```java
class Solution {

    HashSet<Integer> set = new HashSet<>();

    public boolean findTarget(TreeNode root, int k) {
        return dfs(root, k);
    }

    private boolean dfs(TreeNode root, int k) {

        if (root == null) {
            return false;
        }

        if (set.contains(k - root.val)) {
            return true;
        }

        set.add(root.val);

        return dfs(root.left, k) ||
               dfs(root.right, k);
    }
}
```
