# Binary Search Tree - Core

This section covers the fundamental Binary Search Tree (BST) problems required to build a strong foundation in BST operations, traversal techniques, construction, searching, insertion, deletion, and common interview patterns.

## Prerequisites

Before starting this section, you should be comfortable with:

* Binary Tree Traversals
* Recursion
* Tree Node Structure
* DFS Traversal
* Basic Time and Space Complexity Analysis
