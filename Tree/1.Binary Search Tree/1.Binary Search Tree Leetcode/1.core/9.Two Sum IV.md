# LeetCode 653 — Two Sum IV: Input is a BST

## Problem

Given the root of a BST and an integer `k`, return `true` if there exist **two nodes** in the BST whose sum equals `k`.

---

## Example 1

```text
        5
       / \
      3   6
     / \   \
    2   4   7

k = 9
```

Possible pair:

```text
2 + 7 = 9
```

Output:

```text
true
```

---

## Example 2

```text
        5
       / \
      3   6
     / \   \
    2   4   7

k = 28
```

No pair exists.

Output:

```text
false
```

---

# Simple Interview Approach (HashSet)

## Idea

While traversing:

For every node:

```text
Need = k - node.val
```

If `Need` already exists in a HashSet:

```text
Found pair
```

Return `true`.

Otherwise store current value.

---

## Code

```java
class Solution {

    HashSet<Integer> set = new HashSet<>();

    public boolean findTarget(TreeNode root, int k) {
        return dfs(root, k);
    }

    private boolean dfs(TreeNode root, int k) {

        if (root == null) {
            return false;
        }

        if (set.contains(k - root.val)) {
            return true;
        }

        set.add(root.val);

        return dfs(root.left, k) ||
               dfs(root.right, k);
    }
}
```

# Why It Works

For every node:

```java
if(set.contains(k - root.val))
```

means:

```text
previousValue + currentValue = k
```

So a valid pair exists.

---

# Complexity

| Operation | Complexity |
| --------- | ---------- |
| Time      | O(n)       |
| Space     | O(n)       |

where `n` = number of nodes.
