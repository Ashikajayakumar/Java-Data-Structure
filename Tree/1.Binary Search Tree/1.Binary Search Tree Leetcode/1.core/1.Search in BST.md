## Problem

Given the root of a BST and a value `val`, find the node whose value equals `val`.

If the node exists, return it; otherwise return `null`.

---

## BST Property Used

```text
Left Subtree  <  Root  <  Right Subtree
```

Because the tree is sorted, we don't need to search every node.

---

## Example

BST:

```text
        50
       /  \
     30    70
    / \   / \
   20 40 60 80
```

Search for:

```text
60
```

---

## Step-by-Step

### At Node 50

```text
60 > 50
```

Go right.

```text
        50
           \
            70
```

---

### At Node 70

```text
60 < 70
```

Go left.

```text
      70
     /
    60
```

---

### At Node 60

```text
60 == 60
```

Found.

Return node 60.

---

# Recursive Solution

```java
class Solution {

    public TreeNode searchBST(TreeNode root, int val) {

        if (root == null || root.val == val) {
            return root;
        }

        if (val < root.val) {
            return searchBST(root.left, val);
        }

        return searchBST(root.right, val);
    }
}
```

# Iterative Solution (Interview Favorite)

```java
class Solution {

    public TreeNode searchBST(TreeNode root, int val) {

        while (root != null) {

            if (root.val == val) {
                return root;
            }

            if (val < root.val) {
                root = root.left;
            } else {
                root = root.right;
            }
        }

        return null;
    }
}
```
