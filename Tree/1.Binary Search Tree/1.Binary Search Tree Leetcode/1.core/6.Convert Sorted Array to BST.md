# Convert Sorted Array to Binary Search Tree (LeetCode 108)

## Problem

Given a sorted array, convert it into a **height-balanced BST**.

Example:

```java
nums = [-10, -3, 0, 5, 9]
```

Output BST:

```text
        0
      /   \
    -10     5
      \      \
      -3      9
```

(Any balanced BST is accepted.)

---

# Key Idea

Since the array is already sorted:

```text
[-10, -3, 0, 5, 9]
```

The middle element should become the root.

Why?

```text
Left half  < Root < Right half
```

which matches the BST property.

# Simple Recursive Code

```java
class Solution {

    public TreeNode sortedArrayToBST(int[] nums) {
        return build(nums, 0, nums.length - 1);
    }

    private TreeNode build(int[] nums, int left, int right) {

        if (left > right) {
            return null;
        }

        int mid = (left + right) / 2;

        TreeNode root = new TreeNode(nums[mid]);

        root.left = build(nums, left, mid - 1);
        root.right = build(nums, mid + 1, right);

        return root;
    }
}
```
