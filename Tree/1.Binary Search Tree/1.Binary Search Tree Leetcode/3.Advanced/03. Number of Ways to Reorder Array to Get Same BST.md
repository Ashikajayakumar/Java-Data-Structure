# LeetCode 1569: Number of Ways to Reorder Array to Get Same BST
<img width="1024" height="1536" alt="image" src="https://github.com/user-attachments/assets/ffbe160e-e0df-47a0-9746-c27a6f6b568a" />  
  
Given an array `nums`, construct a BST by inserting elements from left to right. 
   
Return the **number of different reorderings** of `nums` that generate the **same BST**.
  
---

## Key Observation 

When building a BST:

* First element becomes the root.
* Elements smaller than root go to the left subtree.
* Elements greater than root go to the right subtree.

Example:

```text
nums = [3,1,2,5,4,6]

        3
       / \
      1   5
       \ / \
        2 4 6
```

Root must always remain first.

The relative order of elements inside:

* Left subtree
* Right subtree

must be preserved.

Only the merging of left and right elements can change.

---

## Recursive Formula

Let:

```text
leftWays  = ways(left subtree)
rightWays = ways(right subtree)

L = left.size()
R = right.size()
```

Number of ways to interleave left and right elements:

```text
C(L + R, L)
```

Choose positions of left subtree elements among total positions.

Therefore:

```text
ways =
C(L + R, L)
× leftWays
× rightWays
```

---

## Example

```text
nums = [3,1,2,5,4,6]
```

Root:

```text
3
```

Left:

```text
[1,2]
```

Right:

```text
[5,4,6]
```

### Left subtree

```text
1
 \
  2
```

Only:

```text
1 way
```

### Right subtree

```text
   5
  / \
 4   6
```

Can be:

```text
[5,4,6]
[5,6,4]
```

So:

```text
2 ways
```

### Merge

```text
L = 2
R = 3

C(5,2)=10
```

Total:

```text
10 × 1 × 2 = 20
```

## Java Solution

```java
class Solution {

    long MOD = 1_000_000_007L;
    long[][] comb;

    public int numOfWays(int[] nums) {

        int n = nums.length;

        comb = new long[n + 1][n + 1];

        for (int i = 0; i <= n; i++) {
            comb[i][0] = comb[i][i] = 1;

            for (int j = 1; j < i; j++) {
                comb[i][j] =
                    (comb[i - 1][j - 1] + comb[i - 1][j]) % MOD;
            }
        }

        List<Integer> arr = new ArrayList<>();

        for (int x : nums) {
            arr.add(x);
        }

        long ans = dfs(arr);

        return (int)((ans - 1 + MOD) % MOD);
    }

    private long dfs(List<Integer> nums) {

        if (nums.size() <= 2) {
            return 1;
        }

        int root = nums.get(0);

        List<Integer> left = new ArrayList<>();
        List<Integer> right = new ArrayList<>();

        for (int i = 1; i < nums.size(); i++) {

            if (nums.get(i) < root) {
                left.add(nums.get(i));
            } else {
                right.add(nums.get(i));
            }
        }

        long leftWays = dfs(left);
        long rightWays = dfs(right);

        long mergeWays =
            comb[left.size() + right.size()][left.size()];

        return (((mergeWays * leftWays) % MOD)
                * rightWays) % MOD;
    }
}
```
