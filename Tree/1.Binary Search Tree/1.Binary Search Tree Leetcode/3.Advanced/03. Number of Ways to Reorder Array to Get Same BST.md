If you want the **simplest accepted Java solution**, use recursion with a precomputed combination table.

```java
class Solution {

    long MOD = 1_000_000_007;
    long[][] nCr;

    public int numOfWays(int[] nums) {

        int n = nums.length;

        nCr = new long[n + 1][n + 1];

        for (int i = 0; i <= n; i++) {
            nCr[i][0] = nCr[i][i] = 1;

            for (int j = 1; j < i; j++) {
                nCr[i][j] = (nCr[i - 1][j - 1] + nCr[i - 1][j]) % MOD;
            }
        }

        List<Integer> list = new ArrayList<>();

        for (int num : nums) {
            list.add(num);
        }

        return (int) ((dfs(list) - 1 + MOD) % MOD);
    }

    private long dfs(List<Integer> nums) {

        if (nums.size() <= 2) {
            return 1;
        }

        int root = nums.get(0);

        List<Integer> left = new ArrayList<>();
        List<Integer> right = new ArrayList<>();

        for (int i = 1; i < nums.size(); i++) {
            if (nums.get(i) < root) {
                left.add(nums.get(i));
            } else {
                right.add(nums.get(i));
            }
        }

        long leftWays = dfs(left);
        long rightWays = dfs(right);

        long ways = nCr[left.size() + right.size()][left.size()];

        return (((ways * leftWays) % MOD) * rightWays) % MOD;
    }
}
```

### Easy Formula to Remember

For every subtree:

```text
Ways =
nCr(L + R, L)
× Ways(left)
× Ways(right)
```

where:

* `L = number of nodes in left subtree`
* `R = number of nodes in right subtree`

### Example

```text
[3,1,2,5,4,6]

        3
       / \
      1   5
       \ / \
        2 4 6
```

* Left ways = 1
* Right ways = 2
* Merge ways = C(5,2) = 10

```text
Total = 10 × 1 × 2 = 20
Answer = 20 - 1 = 19
```

(`-1` because the original ordering itself should not be counted.)
