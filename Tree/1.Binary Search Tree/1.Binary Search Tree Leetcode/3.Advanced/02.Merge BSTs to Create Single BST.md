# LeetCode 1932: Merge BSTs to Create Single BST   
  
## Key Observations  

### 1. Find the Final Root

Every tree root appears once.

If a root value appears as a leaf in another tree, it cannot be the final root.

Count occurrences:

```text
Roots: 2, 3, 5

Leaves: 3, 5
```

Then:

```text
2
```

is the only root that never appears as a leaf.

This must be the final root.

If there is not exactly one such root:

```java
return null;
```

---

### 2. Merge During DFS

Store:

```java
Map<Integer, TreeNode> roots
```

Example:

```text
2 -> tree1
3 -> tree2
5 -> tree3
```

While traversing:

If we reach a leaf node:

```java
node.left == null &&
node.right == null
```

and

```java
roots.containsKey(node.val)
```

then another tree can be attached here.

Example:

```text
    2
   /
  3
```

and

```text
  3
 / \
1   4
```

Merge into:

```text
      2
     /
    3
   / \
  1   4
```

---

### 3. Validate BST Simultaneously

Use BST bounds.

```java
dfs(node, min, max)
```

Condition:

```java
min < node.val < max
```

If violated:

```java
return false;
```

---

### 4. Verify All Trees Used

After DFS:

```java
roots.size() == 1
```

Only final root should remain.

Otherwise some trees were never merged.

Return `null`.

---

# Simple Code

```java
class Solution {

    Map<Integer, TreeNode> map = new HashMap<>();

    public TreeNode canMerge(List<TreeNode> trees) {

        Set<Integer> leaves = new HashSet<>();

        for (TreeNode root : trees) {

            map.put(root.val, root);

            if (root.left != null)
                leaves.add(root.left.val);

            if (root.right != null)
                leaves.add(root.right.val);
        }

        TreeNode root = null;

        for (TreeNode node : trees) {
            if (!leaves.contains(node.val)) {

                if (root != null)
                    return null;

                root = node;
            }
        }

        if (root == null)
            return null;

        if (!dfs(root, Long.MIN_VALUE, Long.MAX_VALUE))
            return null;

        return map.size() == 1 ? root : null;
    }

    private boolean dfs(TreeNode node, long min, long max) {

        if (node == null)
            return true;

        if (node.val <= min || node.val >= max)
            return false;

        if (node.left == null && node.right == null &&
            map.containsKey(node.val) &&
            map.get(node.val) != node) {

            TreeNode merge = map.get(node.val);

            node.left = merge.left;
            node.right = merge.right;

            map.remove(node.val);
        }

        return dfs(node.left, min, node.val) &&
               dfs(node.right, node.val, max);
    }
}
```

Input:

```text
Tree1:     2
          / \
         1   3

Tree2:     3
             \
              5

Tree3:     5
           /
          4
```

Root candidates:

```text
Roots = {2,3,5}
Leaves = {1,3,5,4}
```

Only:

```text
2
```

is not a leaf.

Merge:

```text
      2
     / \
    1   3
         \
          5
         /
        4
```
