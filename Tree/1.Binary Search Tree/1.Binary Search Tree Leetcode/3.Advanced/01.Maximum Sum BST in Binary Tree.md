## LeetCode 1373: Maximum Sum BST in Binary Tree

### Problem

Given a binary tree, return the **maximum sum of all keys of any subtree which is also a BST**.

A subtree is considered a BST if:

* Left subtree values < root value
* Right subtree values > root value
* Both left and right subtrees are BSTs

---

## Key Idea

For every node, we need to know:

1. Is the subtree rooted at this node a BST?
2. What is its minimum value?
3. What is its maximum value?
4. What is the sum of all nodes in this subtree?

This suggests a **postorder traversal (Left → Right → Root)** because we need information from children before processing the parent.

---

## Information Returned from Each Node

For every node return:

```java
{
    minValue,
    maxValue,
    sum
}
```

### Base Case

For a null node:

```java
min = Integer.MAX_VALUE
max = Integer.MIN_VALUE
sum = 0
```

Why?

```java
root.val > left.max
root.val < right.min
```

should automatically become true for leaf nodes.

---

## BST Validation

Suppose we are at node:

```text
       5
      / \
     3   8
```

After processing left and right:

```java
left.max = 3
right.min = 8
```

BST condition:

```java
root.val > left.max &&
root.val < right.min
```

If true:

```java
sum = left.sum + right.sum + root.val
```

Update answer:

```java
ans = Math.max(ans, sum);
```

Return:

```java
{
   min(root.val, left.min),
   max(root.val, right.max),
   sum
}
```

---

## Invalid BST Case

Example:

```text
      5
     / \
    8   6
```

Here:

```java
5 > 8   ❌
```

Not a BST.

Return values that will force ancestors to fail:

```java
{
   Integer.MIN_VALUE,
   Integer.MAX_VALUE,
   0
}
```

So parent checks become impossible.

## Java Solution

```java
class Solution {

    int ans = 0;

    public int maxSumBST(TreeNode root) {
        dfs(root);
        return ans;
    }

    private int[] dfs(TreeNode root) {

        if (root == null) {
            return new int[]{
                Integer.MAX_VALUE,
                Integer.MIN_VALUE,
                0
            };
        }

        int[] left = dfs(root.left);
        int[] right = dfs(root.right);

        if (root.val > left[1] && root.val < right[0]) {

            int sum = left[2] + right[2] + root.val;

            ans = Math.max(ans, sum);

            return new int[]{
                Math.min(root.val, left[0]),
                Math.max(root.val, right[1]),
                sum
            };
        }

        return new int[]{
            Integer.MIN_VALUE,
            Integer.MAX_VALUE,
            0
        };
    }
}
```

## Complexity Analysis

### Time Complexity

Each node visited once:

```text
O(N)
```

### Space Complexity

Recursion stack:

```text
O(H)
```

* Balanced Tree → `O(log N)`
* Skewed Tree → `O(N)`

## Pattern Recognition

This problem follows the common **Tree DP / Postorder Information Passing** pattern:

```text
Validate BST
+
Calculate Sum
+
Return min/max values upward
```

Similar problems:

* Validate BST (98)
* Largest BST Subtree
* Binary Tree Cameras
* House Robber III
* Diameter of Binary Tree
* Longest Univalue Path

The core idea is: **each node returns information about its subtree to its parent.**
