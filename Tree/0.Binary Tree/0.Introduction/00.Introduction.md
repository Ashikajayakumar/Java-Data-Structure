  ### Definition            
             
A **Binary Tree** is a non-linear data structure where each node has:             
     
* at most **2 children**      
    
  * Left child  
  * Right child 
 

---

# Types of Binary Tree
 
## 1. Full Binary Tree

Every node has either:

* 0 children
* OR 2 children

## 2. Complete Binary Tree

All levels filled except maybe last level.

## 3. Perfect Binary Tree

All internal nodes have 2 children and all leaf nodes are same level.

## 4. Balanced Binary Tree

Height difference between left and right subtree is small.

## 5. Degenerate Tree

Every node has only one child.
