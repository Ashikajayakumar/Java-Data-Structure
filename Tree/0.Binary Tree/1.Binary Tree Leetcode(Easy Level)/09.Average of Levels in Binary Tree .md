# Average of Levels in Binary Tree (LeetCode 637)

## Problem

Return the average value of nodes on each level of the binary tree.

# Level-wise Average

## Level 1

```text
3
Average = 3
```

## Level 2

```text
9, 20
Average = (9 + 20) / 2 = 14.5
```

## Level 3

```text
15, 7
Average = (15 + 7) / 2 = 11
```

Output:

```text
[3.00000, 14.50000, 11.00000]
```

# Main Idea

Use:

```text
BFS (Queue)
```

Process tree level by level.

---

# Easy Java Solution (BFS)

```java
class Solution {

    public List<Double> averageOfLevels(TreeNode root) {

        List<Double> ans = new ArrayList<>();

        Queue<TreeNode> q = new LinkedList<>();

        q.offer(root);

        while(!q.isEmpty()){

            int size = q.size();

            double sum = 0;

            for(int i = 0; i < size; i++){

                TreeNode curr = q.poll();

                sum += curr.val;

                if(curr.left != null){
                    q.offer(curr.left);
                }

                if(curr.right != null){
                    q.offer(curr.right);
                }
            }

            ans.add(sum / size);
        }

        return ans;
    }
}
```
