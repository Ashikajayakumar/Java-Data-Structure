# Invert Binary Tree(leetcode No- 226)

## Problem

Invert (mirror) the binary tree.

Meaning:

* Left child becomes right child
* Right child becomes left child

# Main Idea

At every node:

1. Swap left and right
2. Repeat for left subtree
3. Repeat for right subtree

---

# Swap Logic
```
left ↔ right
```

---

# Easy LeetCode Java Solution

```java id="2mvlq9"
class Solution {

    public TreeNode invertTree(TreeNode root) {

        // Base condition
        if(root == null){
            return null;
        }

        // Swap children
        TreeNode temp = root.left;
        root.left = root.right;
        root.right = temp;

        // Recursive calls
        invertTree(root.left);
        invertTree(root.right);

        return root;
    }
}
```
