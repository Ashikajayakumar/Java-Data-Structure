# Merge Two Binary Trees (LeetCode 617)

## Problem

Given two binary trees:

Merge them into one tree.

Rules:

* If both nodes exist → add values
* If one node is null → use other node

---

# Example

## Tree 1

```text id="7mql2x"
        1
       / \
      3   2
     /
    5
```

## Tree 2

```text id="x8m4ql"
        2
       / \
      1   3
       \   \
        4   7
```

---

# Merged Tree

```text id="0mql8z"
        3
       / \
      4   5
     / \   \
    5   4   7
```

---

# Main Idea

At every node:

1. Add values if both exist
2. Recursively merge left children
3. Recursively merge right children

---

# Merge Formula

MergedNode = Node1 + Node2

---

# Easy Java Solution

```java id="1xpl9m"
class Solution {

    public TreeNode mergeTrees(TreeNode root1, TreeNode root2) {

        // If first tree null
        if(root1 == null){
            return root2;
        }

        // If second tree null
        if(root2 == null){
            return root1;
        }

        // Merge values
        root1.val = root1.val + root2.val;

        // Merge left subtree
        root1.left = mergeTrees(root1.left, root2.left);

        // Merge right subtree
        root1.right = mergeTrees(root1.right, root2.right);

        return root1;
    }
}
```
