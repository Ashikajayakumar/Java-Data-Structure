## Problem
Return the inorder traversal of a binary tree.

# Inorder Traversal Rule
```
Inorder = Left → Root → Right
```



---

# Easy Recursive Java Solution

```java id="z8m1ql"
class Solution {

    List<Integer> ans = new ArrayList<>();

    public List<Integer> inorderTraversal(TreeNode root) {

        inorder(root);

        return ans;
    }

    public void inorder(TreeNode root){

        if(root == null){
            return;
        }

        // Left
        inorder(root.left);

        // Root
        ans.add(root.val);

        // Right
        inorder(root.right);
    }
}
```
