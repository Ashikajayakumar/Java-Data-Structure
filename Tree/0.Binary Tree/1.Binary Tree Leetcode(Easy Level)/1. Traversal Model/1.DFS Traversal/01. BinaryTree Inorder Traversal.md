# Binary Tree Inorder Traversal (LeetCode 94)

## Problem
<img width="1024" height="1536" alt="image" src="https://github.com/user-attachments/assets/9fd1e29a-6d6e-47d6-9106-0d44cbf7befd" />

Return the inorder traversal of a binary tree.

--- 

# Inorder Traversal Rule
```
Inorder = Left → Root → Right
```
---

* Left of 1 = null
* Visit 1
* Go right to 2
* Left of 2 = 3

```text id="2m8qlp"

```

# Easy Recursive Java Solution

```java id="z8m1ql"
class Solution {

    List<Integer> ans = new ArrayList<>();

    public List<Integer> inorderTraversal(TreeNode root) {

        inorder(root);

        return ans;
    }

    public void inorder(TreeNode root){

        if(root == null){
            return;
        }

        // Left
        inorder(root.left);

        // Root
        ans.add(root.val);

        // Right
        inorder(root.right);
    }
}
```
