# Preorder Traversal Rule

# Easy Recursive Java Solution

```java id="z7m2ql"
class Solution {

    List<Integer> ans = new ArrayList<>();

    public List<Integer> preorderTraversal(TreeNode root) {

        preorder(root);

        return ans;
    }

    public void preorder(TreeNode root){

        if(root == null){
            return;
        }

        // Root
        ans.add(root.val);

        // Left
        preorder(root.left);

        // Right
        preorder(root.right);
    }
}
```

# Iterative Stack Solution

```java id="l7m1qp"
class Solution {

    public List<Integer> preorderTraversal(TreeNode root) {

        List<Integer> ans = new ArrayList<>();

        if(root == null){
            return ans;
        }

        Stack<TreeNode> st = new Stack<>();

        st.push(root);

        while(!st.isEmpty()){

            TreeNode curr = st.pop();

            ans.add(curr.val);

            // Push right first
            if(curr.right != null){
                st.push(curr.right);
            }

            // Push left second
            if(curr.left != null){
                st.push(curr.left);
            }
        }

        return ans;
    }
}
```

---

# Why Push Right First?

Stack = LIFO

We want:

```text id="k1mql8"
Root → Left → Right
```

So:

1. Push right
2. Push left

Left comes out first.
