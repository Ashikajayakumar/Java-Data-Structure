# LeetCode 437: Path Sum III

## Problem Statement

Given the root of a binary tree and an integer `targetSum`, return the number of paths where the sum of the values along the path equals `targetSum`.

### Important

* The path does **not need to start at the root**.
* The path does **not need to end at a leaf**.
* The path must move **downward** (parent → child).

# Optimal Approach: Prefix Sum + DFS

This is similar to the array problem:

```text
Subarray Sum Equals K
```

Instead of an array, we use the current root-to-node path.

---

## Core Idea

Suppose current path sum is:

```text
currentSum = 18
target = 8
```

We need a previous prefix sum:

```text
18 - 8 = 10
```

If prefix sum `10` already exists,

```text
18 - 10 = 8
```

then a valid path exists.

---

# Formula

currentSum - targetSum

If this value exists in our prefix map, we found one or more valid paths.

---

# Algorithm

### Step 1

Maintain:

```java
Map<Long, Integer> prefixSumCount
```

Stores:

```text
prefixSum → frequency
```

---

### Step 2

At every node:

```java
currentSum += node.val
```

---

### Step 3

Find:

```java
currentSum - targetSum
```

If present in map:

```java
count += map.get(currentSum - targetSum)
```

---

### Step 4

Add current prefix sum:

```java
map.put(currentSum,
        map.getOrDefault(currentSum,0)+1);
```

---

### Step 5

Explore children.

---

### Step 6 (Backtracking)

Remove current prefix before returning.

```java
map.put(currentSum,
        map.get(currentSum)-1);
```

# Java Solution (Optimal)

```java
class Solution {

    int count = 0;

    public int pathSum(TreeNode root, int targetSum) {

        Map<Long, Integer> prefix = new HashMap<>();

        prefix.put(0L, 1);

        dfs(root, 0L, targetSum, prefix);

        return count;
    }

    private void dfs(TreeNode node,
                     long currentSum,
                     int target,
                     Map<Long, Integer> prefix) {

        if (node == null) {
            return;
        }

        currentSum += node.val;

        count += prefix.getOrDefault(
                    currentSum - target,
                    0);

        prefix.put(
            currentSum,
            prefix.getOrDefault(currentSum, 0) + 1
        );

        dfs(node.left, currentSum, target, prefix);
        dfs(node.right, currentSum, target, prefix);

        prefix.put(
            currentSum,
            prefix.get(currentSum) - 1
        );
    }
}
```



and is often considered the tree version of **Subarray Sum Equals K**.
