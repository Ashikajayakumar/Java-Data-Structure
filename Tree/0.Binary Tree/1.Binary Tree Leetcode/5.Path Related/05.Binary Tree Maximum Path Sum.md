# LeetCode 124: Binary Tree Maximum Path Sum

## Problem Statement
 
Given the root of a binary tree, return the **maximum path sum** of any path.

A path:

* Can start from any node.
* Can end at any node.
* Must follow parent-child connections.
* Must contain at least one node.

---

## Example 1

```text
       1
      / \
     2   3
```

Possible paths:

```text
1           = 1
2           = 2
3           = 3
2 → 1       = 3
1 → 3       = 4
2 → 1 → 3   = 6
```

Output:

```text
6
```

---

## Example 2

```text
         -10
         /  \
        9   20
           /  \
          15   7
```

Best path:

```text
15 → 20 → 7
```

Sum:

```text
15 + 20 + 7 = 42
```

Output:

```text
42
```

---

# Key Idea

At every node we calculate:

### 1. Maximum Gain

The maximum contribution this node can provide to its parent.

```text
node.val + max(leftGain, rightGain)
```

Parent can choose only **one side**.

---

### 2. Maximum Path Through Current Node

```text
leftGain + node.val + rightGain
```

This path uses:

```text
Left Subtree
     ↓
 Current Node
     ↓
Right Subtree
```

This may become the global answer.

---

# Formula

Maximum path through a node:

leftGain+node.val+rightGain

Gain returned to parent:

node.val+\max(leftGain,rightGain)

---

# Why Ignore Negative Paths?

Suppose:

```text
      10
     /
   -20
```

Taking `-20` reduces the sum.

So:

```java
leftGain = Math.max(0, dfs(node.left));
rightGain = Math.max(0, dfs(node.right));
```

Negative contributions are treated as `0`.

# Java Solution

```java
class Solution {

    private int maxSum = Integer.MIN_VALUE;

    public int maxPathSum(TreeNode root) {

        dfs(root);

        return maxSum;
    }

    private int dfs(TreeNode node) {

        if (node == null) {
            return 0;
        }

        int leftGain = Math.max(0, dfs(node.left));
        int rightGain = Math.max(0, dfs(node.right));

        int currentPath =
                leftGain +
                node.val +
                rightGain;

        maxSum = Math.max(maxSum, currentPath);

        return node.val +
               Math.max(leftGain, rightGain);
    }
}
```
