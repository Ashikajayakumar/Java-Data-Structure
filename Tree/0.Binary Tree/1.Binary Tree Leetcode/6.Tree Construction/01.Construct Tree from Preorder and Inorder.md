## Construct Binary Tree from Preorder and Inorder Traversal

**LeetCode 105** – Very Important Interview Question

### Problem

Given two arrays:

* **Preorder** = Root → Left → Right
* **Inorder** = Left → Root → Right

Construct the binary tree and return its root.

### Example

```text
preorder = [3,9,20,15,7]
inorder  = [9,3,15,20,7]
```

Constructed Tree:

```text
      3
     / \
    9   20
       /  \
      15   7
```



## Simple Java Code

```java
class Solution {

    int preIndex = 0;

    public TreeNode buildTree(int[] preorder, int[] inorder) {
        return build(preorder, inorder, 0, inorder.length - 1);
    }

    private TreeNode build(int[] preorder, int[] inorder,
                           int start, int end) {

        if (start > end)
            return null;

        TreeNode root = new TreeNode(preorder[preIndex++]);

        int pos = start;
        while (inorder[pos] != root.val) {
            pos++;
        }

        root.left = build(preorder, inorder, start, pos - 1);
        root.right = build(preorder, inorder, pos + 1, end);

        return root;
    }
}
```
