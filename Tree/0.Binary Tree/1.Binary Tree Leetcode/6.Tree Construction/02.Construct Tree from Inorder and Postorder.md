## Problem

Given:

* **Inorder** = Left → Root → Right
* **Postorder** = Left → Right → Root

Construct the binary tree and return its root.

### Example

```text
inorder   = [9,3,15,20,7]
postorder = [9,15,7,20,3]
```

Constructed Tree:

```text
      3
     / \
    9   20
       /  \
      15   7
```



# Important Difference from Preorder + Inorder

### Preorder

```text
Root Left Right
```

Build:

```text
Left first
Right second
```

### Postorder

```text
Left Right Root
```

Since we start from the end:

```text
Root Right Left
```

Build:

```text
Right first
Left second
```

⚠️ Interview Trap: Build **right subtree before left subtree**.

---

# Simple Java Solution (O(n²))

```java
class Solution {

    int postIndex;

    public TreeNode buildTree(int[] inorder, int[] postorder) {

        postIndex = postorder.length - 1;

        return build(inorder, postorder, 0, inorder.length - 1);
    }

    private TreeNode build(int[] inorder,
                           int[] postorder,
                           int start,
                           int end) {

        if (start > end)
            return null;

        TreeNode root = new TreeNode(postorder[postIndex--]);

        int pos = start;
        while (inorder[pos] != root.val) {
            pos++;
        }

        root.right = build(inorder, postorder, pos + 1, end);
        root.left = build(inorder, postorder, start, pos - 1);

        return root;
    }
}
```



This single rule helps solve both construction problems quickly in interviews.
