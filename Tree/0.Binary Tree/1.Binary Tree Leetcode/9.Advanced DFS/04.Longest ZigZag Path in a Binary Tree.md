# **LeetCode 1372 — Longest ZigZag Path in a Binary Tree**
 
---

## 🔷 Problem Idea 
 
A **ZigZag path** means:

* If you go **Left**, next must be **Right**
* If you go **Right**, next must be **Left**

We want the **maximum length** of such a path.

---

## 🔷 Key Insight

At every node, we track:

```text id="z1"
(coming direction, current length)
```

We try both:

* go left → then must come from right next
* go right → then must come from left next

---

## 🔷 Simple DFS Solution (Interview Friendly)

```java id="z2"
class Solution {

    int max = 0;

    public int longestZigZag(TreeNode root) {
        dfs(root, true, 0);  // try left
        dfs(root, false, 0); // try right
        return max;
    }

    private void dfs(TreeNode node, boolean goLeft, int len) {

        if (node == null) return;

        max = Math.max(max, len);

        if (goLeft) {
            // we came from right, now go left
            dfs(node.left, false, len + 1);

            // restart path from other direction
            dfs(node.right, true, 1);
        } else {
            // we came from left, now go right
            dfs(node.right, true, len + 1);

            // restart path
            dfs(node.left, false, 1);
        }
    }
}
```

---

## 🔷 Better Clean DFS (Most Used in Interviews)

This is the **standard optimal pattern**:

```java id="z3"
class Solution {

    int max = 0;

    public int longestZigZag(TreeNode root) {
        dfs(root);
        return max;
    }

    // returns int[]{leftZigZag, rightZigZag}
    private int[] dfs(TreeNode node) {

        if (node == null) {
            return new int[]{-1, -1};
        }

        int[] left = dfs(node.left);
        int[] right = dfs(node.right);

        int leftZig = left[1] + 1;
        int rightZig = right[0] + 1;

        max = Math.max(max, Math.max(leftZig, rightZig));

        return new int[]{leftZig, rightZig};
    }
}
```
