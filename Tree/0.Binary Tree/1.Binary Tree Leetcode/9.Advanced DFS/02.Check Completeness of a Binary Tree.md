## What is a Complete Binary Tree?

A binary tree is **complete** if:

1. Every level is completely filled except possibly the last level.
2. All nodes in the last level are as far left as possible.

### Complete Tree ✅

```text
        1
       / \
      2   3
     / \  /
    4  5 6
```

### Not Complete ❌

```text
        1
       / \
      2   3
     /     \
    4       7
```



# BFS Solution

```java
class Solution {
    public boolean isCompleteTree(TreeNode root) {

        Queue<TreeNode> q = new LinkedList<>();
        q.offer(root);

        boolean seenNull = false;

        while(!q.isEmpty()) {

            TreeNode curr = q.poll();

            if(curr == null) {
                seenNull = true;
            } else {

                if(seenNull) {
                    return false;
                }

                q.offer(curr.left);
                q.offer(curr.right);
            }
        }

        return true;
    }
}
```
