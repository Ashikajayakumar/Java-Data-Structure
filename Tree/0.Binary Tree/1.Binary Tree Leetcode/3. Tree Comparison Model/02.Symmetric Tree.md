# Symmetric Tree(Leetcode No- 101)

## Problem

Check whether a binary tree is a mirror of itself.

Return:

* `true` → symmetric
* `false` → not symmetric

# Mirror Formula

Mirror(a,b)=Mirror(a.left,b.right) ^ Mirror(a.right,b.left)

---

# Easy LeetCode Java Solution

```java id="n8k3dl"
class Solution {

    public boolean isSymmetric(TreeNode root) {

        if (root == null) {
            return true;
        }

        return mirror(root.left, root.right);
    }

    public boolean mirror(TreeNode left, TreeNode right) {

        // Both null
        if (left == null && right == null) {
            return true;
        }

        // One null
        if (left == null || right == null) {
            return false;
        }

        // Values different
        if (left.val != right.val) {
            return false;
        }

        // Mirror checking
        return mirror(left.left, right.right) &&
               mirror(left.right, right.left);
    }
}
```
