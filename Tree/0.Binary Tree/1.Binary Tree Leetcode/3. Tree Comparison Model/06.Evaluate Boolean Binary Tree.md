## Problem

You are given the root of a **full binary tree** where:

* Leaf nodes:

  * `0` → False 
  * `1` → True

* Internal nodes:

  * `2` → OR (`||`)
  * `3` → AND (`&&`)

Evaluate the tree and return the boolean result.

---

## Example

```text
        2
       / \
      1   3
         / \
        0   1
```

### Evaluation

```text
Node 3 = 0 AND 1
       = false

Node 2 = 1 OR false
       = true
```

Output:

```text
true
```



## Recursive Logic

### Leaf Node

```java
if(root.left == null && root.right == null)
```

Return:

```java
root.val == 1
```

---

### OR Node (2)

```java
left || right
```

---

### AND Node (3)

```java
left && right
```

---

## Simple Java Code

```java
class Solution {

    public boolean evaluateTree(TreeNode root) {

        if(root.left == null && root.right == null) {
            return root.val == 1;
        }

        boolean left = evaluateTree(root.left);
        boolean right = evaluateTree(root.right);

        if(root.val == 2) {
            return left || right;
        }

        return left && right;
    }
}
```
