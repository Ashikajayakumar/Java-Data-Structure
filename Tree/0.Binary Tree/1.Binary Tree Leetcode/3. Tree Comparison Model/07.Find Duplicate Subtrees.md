# LeetCode 652: Find Duplicate Subtrees

## Idea

Two subtrees are duplicates if they have:

* Same root value
* Same left subtree
* Same right subtree

We can **serialize** each subtree into a string and store its frequency in a HashMap.

When a serialization appears for the **second time**, add that root node to the answer.

---

## Easy Recursive Solution

```java
class Solution {

    Map<String, Integer> map = new HashMap<>();
    List<TreeNode> ans = new ArrayList<>();

    public List<TreeNode> findDuplicateSubtrees(TreeNode root) {
        dfs(root);
        return ans;
    }

    private String dfs(TreeNode root) {

        if (root == null) {
            return "#";
        }

        String serial = root.val + "," +
                        dfs(root.left) + "," +
                        dfs(root.right);

        map.put(serial, map.getOrDefault(serial, 0) + 1);

        if (map.get(serial) == 2) {
            ans.add(root);
        }

        return serial;
    }
}
```

---

## Example

```text
        1
       / \
      2   3
     /   / \
    4   2   4
       /
      4
```

### Serialized Subtrees

```text
4,#,#
```

appears 3 times

```text
2,4,#,#,#
```

appears 2 times

So the duplicate subtrees are:

```text
    4

    2
   /
  4
```

Output:

```text
[4, 2]
```

## Dry Run

For leaf node `4`:

```text
4,#,#
```

Map:

```text
{
  "4,#,#" : 1
}
```

Second `4`:

```text
{
  "4,#,#" : 2
}
```

Add node `4` to answer.

---

For subtree:

```text
  2
 /
4
```

Serialization:

```text
2,4,#,#,#
```

When it appears again:

```text
count = 2
```

Add node `2` to answer.
