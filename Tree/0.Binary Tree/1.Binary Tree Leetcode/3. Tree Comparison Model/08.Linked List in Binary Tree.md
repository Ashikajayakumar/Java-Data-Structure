# LeetCode 1367: Linked List in Binary Tree
<img width="1024" height="1536" alt="image" src="https://github.com/user-attachments/assets/35b1eb79-ecba-4af5-b4bb-396438c94781" />
 
## Idea

Check whether the linked list exists as a **downward path** in the binary tree.

For every tree node:

1. Try to match the linked list starting from that node.
2. If not matched, search in left subtree and right subtree.

---

## Easy Recursive Solution

```java
class Solution {

    public boolean isSubPath(ListNode head, TreeNode root) {

        if (root == null) {
            return false;
        }

        return match(head, root)
                || isSubPath(head, root.left)
                || isSubPath(head, root.right);
    }

    private boolean match(ListNode head, TreeNode root) {

        if (head == null) {
            return true;
        }

        if (root == null) {
            return false;
        }

        if (head.val != root.val) {
            return false;
        }

        return match(head.next, root.left)
                || match(head.next, root.right);
    }
}
```

---

## How It Works

### Linked List

```text
4 -> 2 -> 8
```

### Binary Tree

```text
          1
         / \
        4   4
           /
          2
         /
        8
```

Start from root:

```text
1 != 4
```

Move to children.

At node `4`:

```text
4 == 4 ✓
```

Check next node:

```text
2 == 2 ✓
```

Check next node:

```text
8 == 8 ✓
```

Linked list completely matched.

Return:

```text
true
```

## Recursive Pattern

### Function 1

```java
isSubPath(head, root)
```

Searches every tree node as a possible starting point.

### Function 2

```java
match(head, root)
```
