## Problem  

Given a binary tree and two nodes `p` and `q`, return their **Lowest Common Ancestor (LCA)**.

### Definition

The **Lowest Common Ancestor** is the lowest node in the tree that has both `p` and `q` as descendants.

---

## Example 1

```text
        3
      /   \
     5     1
    / \   / \
   6   2 0   8
      / \
     7   4
```

```text
p = 5
q = 1
```

Answer:

```text
3
```

Because both nodes are found in different subtrees of 3.

---

## Example 2

```text
        3
      /   \
     5     1
    / \
   6   2
      / \
     7   4
```

```text
p = 5
q = 4
```

Answer:

```text
5
```

Because 5 itself is an ancestor of 4.

---

# Key Observation

At every node:

1. Search in left subtree.
2. Search in right subtree.

Three possibilities:

### Case 1

Both found in left subtree.

```text
      3
     /
    5
   / \
  p   q
```

Return left result.

---

### Case 2

Both found in right subtree.

```text
      3
       \
        1
       / \
      p   q
```

Return right result.

---

### Case 3

One found on left and one on right.

```text
        3
      /   \
     p     q
```

Current node is LCA.

Return current node.

---

# Recursive Idea

For every node:

```java
left = dfs(left subtree)
right = dfs(right subtree)
```

If:

```java
left != null && right != null
```

Current node is LCA.

---

# Java Solution

```java
class Solution {

    public TreeNode lowestCommonAncestor(TreeNode root,
                                         TreeNode p,
                                         TreeNode q) {

        if (root == null)
            return null;

        if (root == p || root == q)
            return root;

        TreeNode left =
                lowestCommonAncestor(root.left, p, q);

        TreeNode right =
                lowestCommonAncestor(root.right, p, q);

        if (left != null && right != null)
            return root;

        return left != null ? left : right;
    }
}
```

### Node 7

root == p

Return:

```text
7
```

### Node 4

```java
root == q
```

Return:

```text
4
```

---

### Node 2

Receives:

```text
left = 7
right = 4
```

Both non-null:

```java
return 2;
```

Node 2 becomes LCA.

---

### Node 5

Receives:

```text
left = null
right = 2
```

Return:

```text
2
```

---

### Node 3

Receives:

```text
left = 2
right = null
```

Return:

```text
2
```

Final Answer:

```text
2
```
