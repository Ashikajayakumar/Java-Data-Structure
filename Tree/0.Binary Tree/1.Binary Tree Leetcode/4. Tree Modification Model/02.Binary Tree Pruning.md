# Binary Tree Pruning (LeetCode 814)

**Difficulty:** Medium

## Problem

Given the root of a binary tree containing only `0` and `1`:

Remove every subtree that does **not contain a `1`**.

Return the pruned tree.

---

## Example 1

### Before Pruning

```text
        1
       / \
      0   1
     / \
    0   0
```

### After Pruning

```text
        1
          \
           1
```

Explanation:

* Left subtree contains only `0`s.
* Remove the entire left subtree.

---

## Example 2

### Before Pruning

```text
          1
         / \
        0   0
       / \
      1   0
```

### After Pruning

```text
          1
         /
        0
       /
      1
```

Explanation:

* Right subtree contains only `0`s → removed.
* Left subtree contains a `1` → kept.




## Java Solution

```java
class Solution {

    public TreeNode pruneTree(TreeNode root) {

        if (root == null)
            return null;

        root.left = pruneTree(root.left);
        root.right = pruneTree(root.right);

        if (root.val == 0 &&
            root.left == null &&
            root.right == null) {
            return null;
        }

        return root;
    }
}
```
