## Problem

Given:

* Root of a binary tree
* Value `val`
* Depth `depth`

Add a row of nodes with value `val` at the given depth.

---

## Example

### Input

```text
Tree:

        4
       / \
      2   6
     / \ /
    3  1 5

val = 1
depth = 2
```

### Output

```text
          4
         / \
        1   1
       /     \
      2       6
     / \     /
    3   1   5
```

---

## Simple Idea

When we reach depth `depth - 1`:

### For Left Side

```text
newNode.left = oldLeft
root.left = newNode
```

### For Right Side

```text
newNode.right = oldRight
root.right = newNode
```

---

## Simple Java Code

```java
class Solution {

    public TreeNode addOneRow(TreeNode root, int val, int depth) {

        if (depth == 1) {
            TreeNode newRoot = new TreeNode(val);
            newRoot.left = root;
            return newRoot;
        }

        dfs(root, val, 1, depth);
        return root;
    }

    private void dfs(TreeNode node, int val, int level, int depth) {

        if (node == null)
            return;

        if (level == depth - 1) {

            TreeNode leftNode = new TreeNode(val);
            TreeNode rightNode = new TreeNode(val);

            leftNode.left = node.left;
            rightNode.right = node.right;

            node.left = leftNode;
            node.right = rightNode;

            return;
        }

        dfs(node.left, val, level + 1, depth);
        dfs(node.right, val, level + 1, depth);
    }
}
```

## Dry Run

Input:

```text
        4
       / \
      2   6

val = 1
depth = 2
```

Reach node `4` (depth-1).

Create:

```text
      1       1
```

Attach old children:

```text
leftNode.left = 2
rightNode.right = 6
```

Result:

```text
          4
         / \
        1   1
       /     \
      2       6
```



This is the core pattern used in many Binary Tree modification problems.
