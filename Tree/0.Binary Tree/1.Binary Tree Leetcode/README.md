# Binary Tree LeetCode

This section contains Binary Tree problems organized by common interview patterns. Instead of solving problems randomly, focus on understanding the underlying pattern and applying it across similar questions.

---

## Prerequisites

Before starting these problems, make sure you understand:

* Binary Tree Basics
* Tree Node Structure
* Recursion
* DFS Traversals (Inorder, Preorder, Postorder)
* BFS Traversal (Level Order)
* Queue and Stack
* Time & Space Complexity Analysis
