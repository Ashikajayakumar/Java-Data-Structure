# Binary Tree Leetcode

This section covers Binary Tree problems organized by common interview patterns. Instead of solving problems randomly, focus on mastering one pattern at a time. Most Binary Tree interview questions are variations of DFS, BFS, recursion, tree comparison, tree modification, path-related problems, serialization, and advanced tree traversals.

## Prerequisites

Before starting these problems, you should be comfortable with:

* Binary Tree Basics
* Tree Node Structure
* Recursion
* DFS Traversals
* BFS Traversals
* Queue and Stack
* Time and Space Complexity Analysis
