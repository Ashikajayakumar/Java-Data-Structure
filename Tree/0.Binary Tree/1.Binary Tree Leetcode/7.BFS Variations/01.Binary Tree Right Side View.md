# LeetCode 199 — Binary Tree Right Side View (Medium)

Given the `root` of a binary tree, imagine standing on the **right side** of it. Return the values of the nodes you can see from top to bottom.

---

## Example 1

```text
       1
      / \
     2   3
      \   \
       5   4
```

Output:

```java
[1,3,4]
```

Explanation:

* Level 0 → 1
* Level 1 → 3 (rightmost)
* Level 2 → 4 (rightmost)

---

## Approach 1: BFS (Level Order Traversal)

For each level, the **last node** processed is visible from the right side.

### Java Code

```java
class Solution {
    public List<Integer> rightSideView(TreeNode root) {

        List<Integer> ans = new ArrayList<>();

        if(root == null) {
            return ans;
        }

        Queue<TreeNode> q = new LinkedList<>();
        q.offer(root);

        while(!q.isEmpty()) {

            int size = q.size();

            for(int i = 0; i < size; i++) {

                TreeNode curr = q.poll();

                if(i == size - 1) {
                    ans.add(curr.val);
                }

                if(curr.left != null) {
                    q.offer(curr.left);
                }

                if(curr.right != null) {
                    q.offer(curr.right);
                }
            }
        }

        return ans;
    }
}
```

# Approach 2: DFS (Interview Favorite)

Traverse:

```text
Root → Right → Left
```

The first node encountered at each depth is the right-side view.

### Java Code

```java
class Solution {

    public List<Integer> rightSideView(TreeNode root) {

        List<Integer> ans = new ArrayList<>();

        dfs(root, 0, ans);

        return ans;
    }

    private void dfs(TreeNode root, int depth, List<Integer> ans) {

        if(root == null) {
            return;
        }

        if(depth == ans.size()) {
            ans.add(root.val);
        }

        dfs(root.right, depth + 1, ans);
        dfs(root.left, depth + 1, ans);
    }
}
```
