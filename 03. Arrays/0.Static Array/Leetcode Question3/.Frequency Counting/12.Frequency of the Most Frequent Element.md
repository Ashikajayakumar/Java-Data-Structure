## Problem Statement

You are given an integer array `nums` and an integer `k`.

In one operation, you can increase any element by `1`.

Return the **maximum possible frequency** of an element after performing at most `k` operations.

---

## Example 1

```java
Input: nums = [1,2,4], k = 5

Output: 3
```

### Explanation

```text
[1,2,4]

Increase 1 → 4 (3 operations)
Increase 2 → 4 (2 operations)

Total operations = 5
```

Array becomes:

```text
[4,4,4]
```

Frequency of 4:

```text
3
```

Answer:

```java
3
```

---

## Example 2

```java
Input: nums = [1,4,8,13], k = 5

Output: 2
```

### Explanation

```text
1 → 4 requires 3 operations
```

Now:

```text
[4,4,8,13]
```

Frequency of 4 = 2

Cannot make three elements equal within 5 operations.

Answer:

```java
2
```



## Sliding Window Approach

After sorting:

```java
nums = [1,2,4]
```

To make all elements equal to `4`:

```text
Need:

(4-1) + (4-2)
= 3 + 2
= 5 operations
```

Since:

```text
5 <= k
```

Window size = 3.

---

# Optimal Solution

```java
import java.util.Arrays;

class Solution {
    public int maxFrequency(int[] nums, int k) {

        Arrays.sort(nums);

        long sum = 0;
        int left = 0;
        int answer = 1;

        for(int right = 0; right < nums.length; right++) {

            sum += nums[right];

            while((long) nums[right] *
                  (right - left + 1)
                  - sum > k) {

                sum -= nums[left];
                left++;
            }

            answer = Math.max(
                answer,
                right - left + 1
            );
        }

        return answer;
    }
}
```
