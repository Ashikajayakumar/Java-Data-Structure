## Problem Statement

Given an array `nums` containing `n` distinct numbers in the range:

```text
[0, n]
```

Return the only number in the range that is missing from the array.

---

## Example 1

```java
Input: nums = [3,0,1]

Output: 2
```

### Explanation

Numbers from `0` to `3` should be:

```text
0 1 2 3
```

Given:

```text
3 0 1
```

Missing:

```text
2
```

---

## Example 2

2

# Approach 1: HashSet

### Idea

### Java Solution

```java
import java.util.*;

class Solution {
    public int missingNumber(int[] nums) {

        HashSet<Integer> set = new HashSet<>();

        for (int num : nums) {
            set.add(num);
        }

        for (int i = 0; i <= nums.length; i++) {
            if (!set.contains(i)) {
                return i;
            }
        }

        return -1;
    }
}
```

### Complexity

* Time: O(n)
* Space: O(n)

---

# Approach 2: Sum Formula

### Idea

Missing Number:

```text
Expected Sum - Actual Sum
```

### Java Solution

```java
class Solution {
    public int missingNumber(int[] nums) {

        int n = nums.length;

        int expectedSum = n * (n + 1) / 2;
        int actualSum = 0;

        for (int num : nums) {
            actualSum += num;
        }

        return expectedSum - actualSum;
    }
}
```
