## Problem Statement

The **degree** of an array is the maximum frequency of any element.

Your task is to find the length of the shortest contiguous subarray that has the same degree as the original array.

---

## Example 1

```java
Input: nums = [1,2,2,3,1]

Output: 2
```

### Explanation

Frequency:

```text
1 → 2 times
2 → 2 times
3 → 1 time
```

Degree of array:

```text
2
```

Elements with degree:

```text
1 and 2
```

Subarrays:

```text
For 1:
[1,2,2,3,1]
Length = 5

For 2:
[2,2]
Length = 2
```

Shortest length:

```text
2
```

Answer:

```java
2
```

## Example 2


### Explanation

Frequency:

```text

3 → 1
4 → 1
```

Degree:

```text
3
```

Element with degree:

```text
2
```

First occurrence of 2 = index 1
Last occurrence of 2 = index 6

````

Length:

```text

````

Answer:

```java

```

# Approach (HashMap)

We need:

1. Frequency of each number.
2. First occurrence index.
3. Last occurrence index.

Then:

```text
Length = lastIndex - firstIndex + 1
```

for every element having the maximum frequency.

---

## Java Solution

```java
import java.util.*;

class Solution {
    public int findShortestSubArray(int[] nums) {

        HashMap<Integer, Integer> count = new HashMap<>();
        HashMap<Integer, Integer> first = new HashMap<>();

        int degree = 0;
        int minLength = nums.length;

        for(int i = 0; i < nums.length; i++) {

            if(!first.containsKey(nums[i])) {
                first.put(nums[i], i);
            }

            count.put(nums[i],
                      count.getOrDefault(nums[i], 0) + 1);

            degree = Math.max(degree, count.get(nums[i]));
        }

        for(int i = 0; i < nums.length; i++) {

            if(count.get(nums[i]) == degree) {

                int length =
                    i - first.get(nums[i]) + 1;

                minLength =
                    Math.min(minLength, length);
            }
        }

        return minLength;
    }
}
```

## Better Version (Three HashMaps)

```java
import java.util.*;

class Solution {
    public int findShortestSubArray(int[] nums) {

        HashMap<Integer, Integer> freq = new HashMap<>();
        HashMap<Integer, Integer> first = new HashMap<>();
        HashMap<Integer, Integer> last = new HashMap<>();

        for(int i = 0; i < nums.length; i++) {

            freq.put(nums[i],
                    freq.getOrDefault(nums[i], 0) + 1);

            if(!first.containsKey(nums[i])) {
                first.put(nums[i], i);
            }

            last.put(nums[i], i);
        }

        int degree = 0;

        for(int count : freq.values()) {
            degree = Math.max(degree, count);
        }

        int answer = nums.length;

        for(int key : freq.keySet()) {

            if(freq.get(key) == degree) {

                answer = Math.min(
                    answer,
                    last.get(key) - first.get(key) + 1
                );
            }
        }

        return answer;
    }
}
```
