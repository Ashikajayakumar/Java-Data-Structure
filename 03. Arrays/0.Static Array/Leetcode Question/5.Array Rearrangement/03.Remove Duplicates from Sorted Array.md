## Problem Statement

Given an integer array `nums` sorted in **non-decreasing order**, remove the duplicates in-place such that each unique element appears only once.

Return the number of unique elements `k`.

The first `k` elements of `nums` should contain the unique elements in the same order.

---

## Example 1

```text
Input:
nums = [1,1,2]

Output:
2

nums = [1,2,_]
```

### Explanation

```text
Unique elements are:
[1,2]

k = 2
```

---

```text

nums = [0,0,1,1,1,2,2,3,3,4]

Output:
5

```

### Explanation

```text
Unique elements are:
[0,1,2,3,4]

k = 5
```

# Approach (Two Pointers)

Since the array is already sorted:

* Use pointer `k` to store the position of the next unique element.
* Traverse the array from index `1`.
* If current element is different from the previous unique element:

  * Place it at index `k`.
  * Increment `k`.

# Java Solution

```java
class Solution {

    public int removeDuplicates(int[] nums) {

        if (nums.length == 0) {
            return 0;
        }

        int k = 1;

        for (int i = 1; i < nums.length; i++) {

            if (nums[i] != nums[k - 1]) {

                nums[k] = nums[i];
                k++;
            }
        }

        return k;
    }
}
```
