# LeetCode 1089: Duplicate Zeros
<img width="1536" height="1024" alt="image" src="https://github.com/user-attachments/assets/a45b636a-10ce-41c9-ae91-c9ab6a54ecb9" />

## Problem Statement

Given a fixed-length integer array `arr`, duplicate each occurrence of zero, shifting the remaining elements to the right.

Elements beyond the length of the original array are not written.

Modify the input array **in-place** and do not return anything.

---

## Example 1

```text
Input:
arr = [1,0,2,3,0,4,5,0]

Output:
[1,0,0,2,3,0,0,4]
```

### Explanation

```text
Original:
[1,0,2,3,0,4,5,0]

Duplicate each 0:

[1,0,0,2,3,0,0,4,5,0,0]

Keep only first 8 elements:

[1,0,0,2,3,0,0,4]
```

---

## Example 2

```text
Input:
arr = [1,2,3]

Output:
[1,2,3]
```

---

# Approach

Instead of shifting elements repeatedly (O(n²)), use a **two-pass approach**:

1. Count how many zeros need to be duplicated.
2. Start filling from the end.
3. Copy elements backwards.
4. When a zero is encountered, write it twice.

This avoids overwriting elements.

---

# Dry Run

```text
arr = [1,0,2,3,0,4,5,0]

n = 8
zeros = 3
```

Virtual array length:

```text
8 + 3 = 11
```

Start from the end:

```text
i = 7
j = 10
```

---

### Step 1

```text
arr[i] = 0

Write 0 at j=10
Write 0 at j=9

(Both outside array size)
```

Move:

```text
i = 6
j = 8
```

---

### Step 2

```text
arr[i] = 5

Write at j=8

Outside array
```

Move:

```text
i = 5
j = 7
```

---

### Step 3

```text
arr[i] = 4

Write at index 7

[1,0,2,3,0,4,5,4]
```

---

### Step 4

```text
arr[i] = 0

Write two zeros

Index 6 = 0
Index 5 = 0
```

Array:

```text
[1,0,2,3,0,0,0,4]
```

---

### Continue

Final array becomes:

```text
[1,0,0,2,3,0,0,4]
```

# Java Solution

```java
class Solution {

    public void duplicateZeros(int[] arr) {

        int zeros = 0;
        int n = arr.length;

        for (int num : arr) {
            if (num == 0) {
                zeros++;
            }
        }

        int i = n - 1;
        int j = n + zeros - 1;

        while (i < j) {

            if (j < n) {
                arr[j] = arr[i];
            }

            if (arr[i] == 0) {

                j--;

                if (j < n) {
                    arr[j] = 0;
                }
            }

            i--;
            j--;
        }
    }
}
```




This is the approach typically expected in coding interviews.
