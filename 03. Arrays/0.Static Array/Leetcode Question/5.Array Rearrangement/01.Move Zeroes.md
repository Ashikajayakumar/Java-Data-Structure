# LeetCode 283: Move Zeroes
<img width="1024" height="1536" alt="image" src="https://github.com/user-attachments/assets/8ad81ea3-f682-4349-a800-4662b660af64" />

## Problem Statement

Given an integer array `nums`, move all `0`s to the end of it while maintaining the relative order of the non-zero elements.

You must do this **in-place** without making a copy of the array.

---

## Example 1

```text
Input:  nums = [0,1,0,3,12]

Output: [1,3,12,0,0]
```

### Explanation

```text
Original Array:
[0,1,0,3,12]

Move non-zero elements forward:

[1,3,12,_,_]

Fill remaining positions with 0:

[1,3,12,0,0]
```

---

## Example 2

```text
Input: nums = [0]

Output: [0]
```

---

## Approach (Two Pointers)

Use two pointers:

* `i` → traverses the array.
* `insertPos` → position where the next non-zero element should be placed.

### Steps

1. Traverse the array.
2. Whenever a non-zero element is found:

   * Swap it with `insertPos`.
   * Increment `insertPos`.
3. At the end, all non-zero elements remain in order and zeros move to the back.

# Java Solution

```java
class Solution {

    public void moveZeroes(int[] nums) {

        int insertPos = 0;

        for (int i = 0; i < nums.length; i++) {

            if (nums[i] != 0) {

                int temp = nums[i];
                nums[i] = nums[insertPos];
                nums[insertPos] = temp;

                insertPos++;
            }
        }
    }
}
```
