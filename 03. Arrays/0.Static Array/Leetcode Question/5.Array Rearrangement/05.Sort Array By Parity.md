# LeetCode 905: Sort Array By Parity
<img width="1402" height="1122" alt="image" src="https://github.com/user-attachments/assets/7490ad66-c564-4946-ba4e-044fc2dd7402" />

## Problem Statement

Given an integer array `nums`, move all the **even integers** to the beginning of the array followed by all the **odd integers**.

Return any array that satisfies this condition.

---

## Example 1

```text
Input:
nums = [3,1,2,4]

Output:
[2,4,3,1]
```

### Explanation

```text
Even Numbers:
2, 4

Odd Numbers:
3, 1

Result:
[2,4,3,1]
```

---

## Example 2

```text
Input:
nums = [0]

Output:
[0]
```

---

# Approach (Two Pointers)

Use two pointers:

```text
left  → Start of array
right → End of array
```

### Rules

* If left element is even → move left forward.
* If right element is odd → move right backward.
* Otherwise swap them.

---

# Dry Run

```text
nums = [3,1,2,4]

left = 0
right = 3
```

---

### Step 1

```text
nums[left] = 3 (Odd)
nums[right] = 4 (Even)

Swap
```

```text
[4,1,2,3]

left = 1
right = 2
```

---

### Step 2

```text
nums[left] = 1 (Odd)
nums[right] = 2 (Even)

Swap
```

```text
[4,2,1,3]

left = 2
right = 1
```

Loop ends.

---

## Final Answer

```text
[4,2,1,3]
```

Any valid parity ordering is accepted.

---

# Java Solution

```java
class Solution {
    public int[] sortArrayByParity(int[] nums) {
        int[] ans = new int[nums.length];
        int index = 0;

        // Add even numbers
        for (int num : nums) {
            if (num % 2 == 0) {
                ans[index++] = num;
            }
        }

        // Add odd numbers
        for (int num : nums) {
            if (num % 2 != 0) {
                ans[index++] = num;
            }
        }

        return ans;
    }
}
```
