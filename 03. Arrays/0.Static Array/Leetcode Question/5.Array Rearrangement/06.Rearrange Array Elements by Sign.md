# LeetCode 2149: Rearrange Array Elements by Sign

## Problem Statement

You are given a **0-indexed integer array** `nums` of even length consisting of an equal number of positive and negative integers.

Rearrange the elements so that:

1. Every consecutive pair has opposite signs.
2. For all integers with the same sign, their relative order is preserved.
3. The rearranged array begins with a positive integer.

Return the modified array.

---

## Example 1

```text
Input:
nums = [3,1,-2,-5,2,-4]

Output:
[3,-2,1,-5,2,-4]
```

### Explanation

```text
Positive Numbers:
[3,1,2]

Negative Numbers:
[-2,-5,-4]

Alternate them:

[3,-2,1,-5,2,-4]
```

---

## Example 2

```text
Input:
nums = [-1,1]

Output:
[1,-1]
```

---

# Approach

Since:

```text
Positive count = Negative count
```

We can place:

```text
Positive numbers at even indices
Negative numbers at odd indices
```

Use two pointers:

```text
pos = 0  → even index
neg = 1  → odd index
```

Create a result array and place elements accordingly.

# Java Solution

```java
class Solution {

    public int[] rearrangeArray(int[] nums) {

        int[] ans = new int[nums.length];

        int pos = 0;
        int neg = 1;

        for (int num : nums) {

            if (num > 0) {

                ans[pos] = num;
                pos += 2;

            } else {

                ans[neg] = num;
                neg += 2;
            }
        }

        return ans;
    }
}
```
