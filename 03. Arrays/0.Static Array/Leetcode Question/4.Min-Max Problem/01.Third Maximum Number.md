# LeetCode 414: Third Maximum Number
<img width="1024" height="1536" alt="image" src="https://github.com/user-attachments/assets/3d54002f-e4eb-44cd-b42a-2d03dddd3c8f" />

## Problem Statement

Given an integer array `nums`, return the **third distinct maximum number** in the array.

If the third distinct maximum does not exist, return the **maximum number**.

---

## Example 1

```java
Input: nums = [3,2,1]

Output: 1
```

### Explanation

Distinct numbers are:

```text
3, 2, 1
```

* 1st Maximum = 3
* 2nd Maximum = 2
* 3rd Maximum = 1

Return:

```text
1
```

---

## Example 2

```java
Input: nums = [1,2]

Output: 2
```

### Explanation

Only two distinct numbers exist:

```text
1, 2
```

Third maximum does not exist.

Return the maximum:

```text
2
```

---

## Example 3

```java
Input: nums = [2,2,3,1]

Output: 1
```

### Explanation

Distinct numbers are:

```text
3, 2, 1
```

Duplicates are ignored.

* 1st Maximum = 3
* 2nd Maximum = 2
* 3rd Maximum = 1

Return:

```text
1
```

---

## Approach 1: Using Three Variables (Optimal)

### Idea

Maintain:

* `first` → Largest number
* `second` → Second largest distinct number
* `third` → Third largest distinct number

Ignore duplicate values.

---

## Dry Run

```java
nums = [2,2,3,1]
```

### Initial

```text
first = null
second = null
third = null
```

### num = 2

```text
first = 2
```

### num = 2

Duplicate → Ignore

### num = 3

```text
third = null
second = 2
first = 3
```

### num = 1

```text
third = 1
```

Result:

```text
third = 1
```

---

## Java Solution (Optimal)

```java
class Solution {
    public int thirdMax(int[] nums) {

        Long first = null;
        Long second = null;
        Long third = null;

        for (int num : nums) {

            if ((first != null && num == first) ||
                (second != null && num == second) ||
                (third != null && num == third)) {
                continue;
            }

            if (first == null || num > first) {

                third = second;
                second = first;
                first = (long) num;

            } else if (second == null || num > second) {

                third = second;
                second = (long) num;

            } else if (third == null || num > third) {

                third = (long) num;
            }
        }

        return third == null ? first.intValue() : third.intValue();
    }
}
```

## Approach 2: Using HashSet + Sorting

### Idea

1. Store distinct elements in a `HashSet`.
2. Convert to a list.
3. Sort in descending order.
4. Return:

   * 3rd element if size ≥ 3
   * otherwise maximum element.

---

## Java Solution

```java
import java.util.*;

class Solution {
    public int thirdMax(int[] nums) {

        Set<Integer> set = new HashSet<>();

        for (int num : nums) {
            set.add(num);
        }

        List<Integer> list = new ArrayList<>(set);

        Collections.sort(list, Collections.reverseOrder());

        if (list.size() >= 3) {
            return list.get(2);
        }

        return list.get(0);
    }
}
```
