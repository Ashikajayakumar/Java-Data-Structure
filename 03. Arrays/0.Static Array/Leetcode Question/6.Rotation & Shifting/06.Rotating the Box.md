## Problem Statement

You are given an `m × n` box represented by a matrix `boxGrid` where:

* `'#'` → Stone
* `'*'` → Obstacle
* `'.'` → Empty space

Rotate the box **90° clockwise**.

After rotation, stones fall downward due to gravity until they hit:

* The bottom of the box,
* Another stone,
* An obstacle.

Return the resulting box.

# Java Solution

```java
class Solution {

    public char[][] rotateTheBox(char[][] box) {

        int m = box.length;
        int n = box[0].length;

        // Move stones to the right (gravity)
        for (int i = 0; i < m; i++) {

            int pos = n - 1;

            for (int j = n - 1; j >= 0; j--) {

                if (box[i][j] == '*') {
                    pos = j - 1;
                }
                else if (box[i][j] == '#') {

                    box[i][j] = '.';
                    box[i][pos] = '#';
                    pos--;
                }
            }
        }

        // Rotate 90 degrees clockwise
        char[][] result = new char[n][m];

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                result[j][m - 1 - i] = box[i][j];
            }
        }

        return result;
    }
}
```
