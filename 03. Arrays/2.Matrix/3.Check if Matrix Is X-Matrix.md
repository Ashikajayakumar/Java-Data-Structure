# Check if Matrix Is X-Matrix

## Problem

Given a square matrix `grid[][]`, check whether it is an **X-Matrix**.

A matrix is called an **X-Matrix** if:

1. All elements on the **main diagonal** are non-zero.
2. All elements on the **secondary diagonal** are non-zero.
3. All other elements must be **zero**.

Return:

```
true  → if it is an X-Matrix

false → otherwise
```

---

# Example 1

## Input

```text
grid =
[
 [2,0,0,1],
 [0,3,1,0],
 [0,5,2,0],
 [4,0,0,2]
]
```

## Output

```text
true
```

# X-Matrix Pattern

For a 3 × 3 matrix:

```text
X X X
X X X
X X X
```

Positions:

```
Main diagonal

↓

(0,0)
(1,1)
(2,2)


Secondary diagonal

↓

(0,2)
(1,1)
(2,0)
```

Only these positions can contain non-zero values.


# Java Solution

```java
class Solution {

    public boolean checkXMatrix(int[][] grid) {

        int n = grid.length;

        for(int i = 0; i < n; i++) {

            for(int j = 0; j < n; j++) {

                // Main diagonal OR Secondary diagonal
                if(i == j || i + j == n - 1) {

                    // Diagonal elements must be non-zero
                    if(grid[i][j] == 0) {
                        return false;
                    }

                } else {

                    // Non-diagonal elements must be zero
                    if(grid[i][j] != 0) {
                        return false;
                    }
                }
            }
        }

        return true;
    }
}
```
