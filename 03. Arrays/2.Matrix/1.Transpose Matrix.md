## Problem

Given a 2D matrix, return its **transpose**.

The **transpose** of a matrix is obtained by converting **rows into columns** and **columns into rows**.

---

## Example 1

```text
Input

matrix =
[
 [1,2,3],
 [4,5,6],
 [7,8,9]
]

Output

[
 [1,4,7],
 [2,5,8],
 [3,6,9]
]
```

---

## Example 2

```text
Input

matrix =
[
 [1,2,3],
 [4,5,6]
]

Output

[
 [1,4],
 [2,5],
 [3,6]
]
```

---

# What is a Transpose?

Suppose the matrix is

```text
1 2 3
4 5 6
```

Rows become columns.

```
Row 0

1 2 3

↓

Column 0

1
2
3
```

```
Row 1

4 5 6

↓

Column 1

4
5
6
```

Final matrix

```text
1 4
2 5
3 6
```

---

# Formula

For every element

```text
matrix[i][j]
```

Store it at

```text
res[j][i]
```

So,

```
Row Index

↓

Column Index
```

and

```
Column Index

↓

Row Index
```

---

# Why Create a New Matrix?

Suppose

```text
matrix

2 × 3
```

```text
[
 [1,2,3],
 [4,5,6]
]
```

After transpose,

the answer becomes

```text
3 × 2
```

```text
[
 [1,4],
 [2,5],
 [3,6]
]
```

Therefore,

```
rows become columns

columns become rows
```

So the new matrix size should be

```java
new int[n][m]
```

where

```
m = number of rows

n = number of columns
```

---

# Algorithm

```
Find rows (m)

Find columns (n)

Create result matrix

size = n × m

Traverse every element

Store

res[j][i] = matrix[i][j]

Return result
```

---

# Java Solution

```java
class Solution {

    public int[][] transpose(int[][] matrix) {

        int m = matrix.length;
        int n = matrix[0].length;

        int[][] res = new int[n][m];

        for (int i = 0; i < m; i++) {

            for (int j = 0; j < n; j++) {

                res[j][i] = matrix[i][j];
            }
        }

        return res;
    }
}
```
