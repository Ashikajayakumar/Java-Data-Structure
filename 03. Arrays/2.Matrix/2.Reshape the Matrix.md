## Problem

Given a matrix `mat` with dimensions:

```
m × n
```

Convert it into a new matrix with dimensions:

```
r × c
```

The reshape is possible **only when the total number of elements remains the same**.

---

# Important Condition

Number of elements before reshape:

```
rows × columns
```

Number of elements after reshape:

```
r × c
```

If

```
m × n == r × c
```

then reshape is possible.

Otherwise,

return the original matrix.

---

# Algorithm

```
Find original rows and columns

Check:

rows * columns == r * c


If false:

    return original matrix


Create new matrix r × c


k = 0
l = 0


Traverse original matrix

    copy element into new matrix

    increase column


    if column reaches c

        move to next row

        reset column


Return new matrix
```

---

# Java Solution

```java
class Solution {

    public int[][] matrixReshape(int[][] mat, int r, int c) {

        int n = mat.length;
        int m = mat[0].length;

        // Check total elements
        if (m * n != r * c) {
            return mat;
        }

        int[][] newarr = new int[r][c];

        int k = 0; // new matrix row
        int l = 0; // new matrix column

        for (int i = 0; i < n; i++) {

            for (int j = 0; j < m; j++) {

                newarr[k][l] = mat[i][j];

                l++;

                if (l == c) {
                    k++;
                    l = 0;
                }
            }
        }

        return newarr;
    }
}
```
