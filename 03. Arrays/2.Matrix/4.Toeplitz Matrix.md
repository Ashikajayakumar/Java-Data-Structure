# Toeplitz Matrix

## Problem

Given a matrix, check whether it is a **Toeplitz Matrix** or not.

A matrix is called **Toeplitz** if:

> Every diagonal from top-left to bottom-right contains the same elements.

---

# Mathematical Condition

For every valid position:

```java
matrix[i][j] == matrix[i+1][j+1]
```

or

```java
matrix[i][j] == matrix[i-1][j-1]
```

Both represent the same diagonal movement.

---

# Example

## Input

```text
matrix =

[
 [1,2,3,4],
 [5,1,2,3],
 [9,5,1,2]
]
```

## Output

```text
true
```

# Understanding Diagonals

A Toeplitz matrix follows:

```
Top-left  →  Bottom-right
```

Example:

```

5  1  2  3

9  5  1  2
```

# Logic Behind Code

Instead of checking complete diagonals separately:

We compare every element with the element diagonally down-right.

Example:

```
matrix[i][j]

compare with

matrix[i+1][j+1]
```

If any mismatch occurs:

```
Not Toeplitz
```

Return:

```
false
```

If all comparisons match:

```
Toeplitz Matrix
```

Return:

```
true
```

---

# Java Solution

```java
class Solution {

    public boolean isToeplitzMatrix(int[][] matrix) {

        int n = matrix.length;
        int m = matrix[0].length;


        for(int i = 0; i < n-1; i++) {

            for(int j = 0; j < m-1; j++) {

                if(matrix[i][j] != matrix[i+1][j+1]) {

                    return false;
                }
            }
        }

        return true;
    }
}
```

# Complexity Analysis

Let:

```
n = number of rows

m = number of columns
```

## Time Complexity

We check every cell except last row and column.

Total comparisons:

```
(n-1) × (m-1)
```

Therefore:

```
O(n × m)
```

---

## Space Complexity

No extra space used.

Only variables:

```
n,m,i,j
```

Therefore:

```
O(1)
```

# Interview Points

### Toeplitz Condition

Compare:

```java
matrix[i][j]
```

with

```java
matrix[i+1][j+1]
```

---

### Why Stop at n-1 and m-1?

Because:

For last row:

```
i+1
```

will go outside matrix.

For last column:

```
j+1
```

will go outside matrix.

So:

```java
i < n-1

j < m-1
```
