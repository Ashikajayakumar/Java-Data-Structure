# Rotate Image (90 Degree Clockwise)

## Problem

Given an `n x n` matrix, rotate the matrix **90 degrees clockwise**.

The rotation should be done **in-place**.

---

# Example

## Input

```text
matrix =

[
 [1,2,3],
 [4,5,6],
 [7,8,9]
]
````

---

## Output

```text
[
 [7,4,1],
 [8,5,2],
 [9,6,3]
]
```

---

# Approach

To rotate a matrix 90° clockwise:

We use two steps:

```
Step 1 → Transpose the Matrix

Step 2 → Reverse Each Row
```

---

# Step 1: Transpose Matrix

## Meaning

Transpose means:

```
Rows become Columns

Columns become Rows
```

Formula:

```
matrix[i][j] ↔ matrix[j][i]
```

---

## Before Transpose

```text
1 2 3

4 5 6

7 8 9
```

---

## After Transpose

```text
1 4 7

2 5 8

3 6 9
```

---

# Step 2: Reverse Each Row

Reverse every row from left to right.

---

Before:

```text
1 4 7

2 5 8

3 6 9
```

---

Reverse Row 1:

```
1 4 7

↓

7 4 1
```

---

Reverse Row 2:

```
2 5 8

↓

8 5 2
```

---

Reverse Row 3:

```
3 6 9

↓

9 6 3
```

---

# Final Matrix

```text
7 4 1

8 5 2

9 6 3
```

---

# Java Code

```java
class Solution {

    public void rotate(int[][] matrix) {

        int n = matrix.length;

        int temp;


        // Step 1: Transpose Matrix

        for(int i = 0; i < n; i++) {

            for(int j = i; j < n; j++) {

                temp = matrix[i][j];

                matrix[i][j] = matrix[j][i];

                matrix[j][i] = temp;
            }
        }


        // Step 2: Reverse Each Row

        for(int i = 0; i < n; i++) {

            for(int j = 0, k = n-1; j < k; j++, k--) {

                temp = matrix[i][j];

                matrix[i][j] = matrix[i][k];

                matrix[i][k] = temp;
            }
        }
    }
}
```

# Why `j = i` in Transpose?

Code:

```java
for(int j=i; j<n; j++)
```

Reason:

The matrix is symmetric during transpose.

If we swap both sides:

Example:

```
matrix[0][1] ↔ matrix[1][0]
```

Later:

```
matrix[1][0] ↔ matrix[0][1]
```

The previous swap will be reversed.

So we only process the upper triangle.
