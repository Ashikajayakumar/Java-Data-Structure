# Two Sum - Pairs with Given Sum

## Problem

Given an integer array `arr[]` and an integer `target`.

Determine whether there exist **two distinct indices** such that

```
arr[i] + arr[j] = target
```

Return

- `true` → if such a pair exists
- `false` → otherwise

---

## Examples

### Example 1

```text
Input

arr = [0,-1,2,-3,1]

target = -2

Output

true
```

Explanation

```
-3 + 1 = -2
```

---

### Example 2

```text
Input

arr = [1,-2,1,0,5]

target = 0

Output

false
```

Explanation

No pair has a sum of `0`.

---

### Example 3

```text
Input

arr = [11]

target = 11

Output

false
```

Explanation

Only one element is present, so a pair cannot be formed.

---

# Brute Force Approach

## Idea

Check every possible pair.

If any pair equals the target,

return `true`.

Otherwise,

return `false`.

---

## Algorithm

```
for i = 0 to n-1

    for j = i+1 to n-1

        if(arr[i] + arr[j] == target)

            return true

return false
```





# Formula

```
required = target - arr[i]
```

---

## Example

```
target = 9

current = 2

required = 9 - 2 = 7
```

If `7` already exists in the set,

then

```
2 + 7 = 9
```

A pair is found.

---

# Algorithm

```
Create an empty HashSet

for each element

    required = target - currentElement

    if(required exists in set)

        return true

    add currentElement into set

return false
```

# Java Solution

```java
import java.util.HashSet;

class Solution {

    boolean twoSum(int arr[], int target) {

        HashSet<Integer> set = new HashSet<>();

        for (int i = 0; i < arr.length; i++) {

            int required = target - arr[i];

            if (set.contains(required)) {
                return true;
            }

            set.add(arr[i]);
        }

        return false;
    }
}
```
