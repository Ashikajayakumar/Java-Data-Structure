# Find Triplets with Zero Sum

## Problem

Given an integer array `arr[]`, determine whether it contains a **triplet** whose sum is equal to **0**.

Return

- `true` → if such a triplet exists.
- `false` → otherwise.

---

## Examples

### Example 1

```text
Input

arr = [0,-1,2,-3,1]

Output

true
```

Explanation

```
0 + (-1) + 1 = 0
```

---

### Example 2

```text
Input

arr = [1,2,3]

Output

false
```

Explanation

No three numbers have a sum equal to `0`.

---

# Brute Force Approach

## Idea

Try every possible combination of three numbers.

For each triplet,

check whether

```
arr[i] + arr[j] + arr[k] == 0
```

If yes,

return `true`.

Otherwise,

return `false`.

---

## Algorithm

```
for i = 0 to n-1

    for j = i+1 to n-1

        for k = j+1 to n-1

            if(arr[i] + arr[j] + arr[k] == 0)

                return true

return false
```




# Pointer Movement Rule

If

```
total < 0
```

Need a bigger sum.

Move

```
left++
```

---

If

```
total > 0
```

Need a smaller sum.

Move

```
right--
```

---

If

```
total == 0
```

Triplet found.

Return

```
true
```

---

# Algorithm

```
Sort the array

for every i

    left = i+1

    right = n-1

    while(left < right)

        total = arr[i] + arr[left] + arr[right]

        if(total == 0)

            return true

        else if(total < 0)

            left++

        else

            right--

return false
```

# Java Solution

```java
import java.util.Arrays;

class Solution {

    public boolean findTriplets(int[] arr) {

        int n = arr.length;

        Arrays.sort(arr);

        for(int i = 0; i < n - 2; i++) {

            int left = i + 1;
            int right = n - 1;

            while(left < right) {

                int total = arr[i] + arr[left] + arr[right];

                if(total == 0) {

                    return true;

                }
                else if(total < 0) {

                    left++;

                }
                else {

                    right--;
                }
            }
        }

        return false;
    }
}
```
