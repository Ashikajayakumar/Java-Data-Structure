## Problem

You are given two integer arrays `a[]` and `b[]`.

Return the **Union** of both arrays in **any order**.

The **Union** contains **only distinct elements** present in either array.

If an element appears multiple times in one or both arrays, it should appear **only once** in the final result.

---

## Examples

### Example 1

```text
Input

a = [1,2,3,2,1]

b = [3,2,2,3,3,2]

Output

[1,2,3]
```

Explanation

```
Distinct elements

= {1,2,3}
```

---

### Example 2

```text
Input

a = [1,2,3]

b = [4,5,6]

Output

[1,2,3,4,5,6]
```

Explanation

Every element is unique.

---

# What is Union?

The **Union** of two arrays contains

```
All distinct elements
```

present in either array.

Example

```text
A = [1,2,3,4]

B = [2,3,5]
```

Union

```text
[1,2,3,4,5]
```

Notice

```
2 and 3 appear only once.
```



# Algorithm

```
Create HashSet

Insert every element of array A

Insert every element of array B

Create ArrayList

Copy every element from HashSet into ArrayList

Return ArrayList
```

# Java Solution

```java
import java.util.*;

class Solution {

    public static ArrayList<Integer> findUnion(int[] a, int[] b) {

        HashSet<Integer> set = new HashSet<>();

        for (int num : a) {
            set.add(num);
        }

        for (int num : b) {
            set.add(num);
        }

        ArrayList<Integer> res = new ArrayList<>();

        for (int value : set) {
            res.add(value);
        }

        return res;
    }
}
```
