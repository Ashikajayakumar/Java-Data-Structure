## Problem

Given two **unsorted** integer arrays `a[]` and `b[]`, where **each array contains distinct elements**, return the **count of common elements** between the two arrays.

The **intersection** of two arrays is the set of elements that are present in **both** arrays.

# What is Intersection?

The **Intersection** of two arrays contains

```
Only the common elements
```

present in both arrays.

Example

```text
A = [1,2,3,4]

B = [2,3,5]
```

Intersection

```text
[2,3]
```

Count

```text
2
```



## Algorithm

```
count = 0

for every element in A

    for every element in B

        if(elementA == elementB)

            count++

return count
```



# Algorithm

```
Create HashSet

Insert every element of array A

count = 0

Traverse array B

    if(element exists in HashSet)

        count++

Return count
```

# Java Solution

```java
import java.util.HashSet;

class Solution {

    public static int intersectSize(int[] a, int[] b) {

        HashSet<Integer> set = new HashSet<>();

        int count = 0;

        for (int num : a) {
            set.add(num);
        }

        for (int num : b) {

            if (set.contains(num)) {
                count++;
            }
        }

        return count;
    }
}
```
