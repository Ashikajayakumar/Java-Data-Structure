# Remove Duplicates from Sorted Array

## Problem

You are given a **sorted** array `arr[]` containing positive integers.

Remove all duplicate elements so that **each element appears only once**.

Return an array containing only the **distinct elements** in the **same order**.

> **Note:** Since the array is already sorted, all duplicate elements appear next to each other.

---

## Examples

### Example 1

```text
Input

arr = [2,2,2,2,2]

Output

[2]
```

Explanation

Only one occurrence of `2` should remain.

---

### Example 2

```text
Input

arr = [1,2,4]

Output

[1,2,4]
```

Explanation

There are no duplicate elements.



# Why Does This Work?

Since the array is sorted,

all duplicate values appear together.

Example

```text
1 1 1 2 2 3 4 4
```

Comparing with the previous element is enough to identify duplicates.

No extra searching is required.

# Java Solution

```java
import java.util.ArrayList;

class Solution {

    ArrayList<Integer> removeDuplicates(int[] arr) {

        ArrayList<Integer> list = new ArrayList<>();

        list.add(arr[0]);

        for (int i = 1; i < arr.length; i++) {

            if (arr[i] != arr[i - 1]) {

                list.add(arr[i]);
            }
        }

        return list;
    }
}
```
