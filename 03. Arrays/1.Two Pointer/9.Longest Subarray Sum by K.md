# Longest Subarray with Sum = K

## Problem

Given an integer array `arr[]` and an integer `k`, find the **length of the longest subarray** whose sum is exactly equal to `k`.

If no such subarray exists, return `0`.

---

## Examples

### Example 1

```text
Input

arr = [10,5,2,7,1,-10]

k = 15

Output

6
```

Explanation

Subarrays having sum = 15

```
[10,5]                  → Length = 2

[5,2,7,1]               → Length = 4

[10,5,2,7,1,-10]        → Length = 6
```

Longest length

```
6
```

---

### Example 2

```text
Input

arr = [-5,8,-14,2,4,12]

k = -5

Output

5
```

Explanation

```
[-5]                    → Length = 1

[-5,8,-14,2,4]          → Length = 5
```

Longest length

```
5
```

---

### Example 3

```text
Input

arr = [10,-10,20,30]

k = 5

Output

0
```

# Why Store the First Occurrence?

We always store the **first occurrence** of a prefix sum.

Example

```
Prefix Sum 10

appears at

index 0

and later

index 5
```

Keeping

```
index 0
```

gives a longer subarray.

Hence

```java
if(!map.containsKey(sum))
    map.put(sum, i);
```

---

# Algorithm

```
Create HashMap

sum = 0

maxLen = 0

for every element

    sum += arr[i]

    if(sum == k)

        maxLen = i+1

    Store first occurrence of prefix sum

    if(sum-k exists)

        length

        = currentIndex - previousIndex

        update answer

return maxLen
```

# Java Solution

```java
import java.util.HashMap;

class Solution {

    public int longestSubarray(int[] arr, int k) {

        HashMap<Integer, Integer> map = new HashMap<>();

        int sum = 0;
        int maxLen = 0;

        for (int i = 0; i < arr.length; i++) {

            sum += arr[i];

            if (sum == k) {
                maxLen = i + 1;
            }

            if (!map.containsKey(sum)) {
                map.put(sum, i);
            }

            if (map.containsKey(sum - k)) {

                maxLen = Math.max(maxLen,
                                  i - map.get(sum - k));
            }
        }

        return maxLen;
    }
}
```
