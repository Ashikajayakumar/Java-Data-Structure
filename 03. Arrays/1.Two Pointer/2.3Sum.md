# 3Sum (LeetCode 15)

## Problem

Given an integer array `nums`, return **all the unique triplets** `[nums[i], nums[j], nums[k]]` such that

```
nums[i] + nums[j] + nums[k] = 0
```

### Conditions

- `i != j`
- `j != k`
- `i != k`
- The solution set must **not contain duplicate triplets**.

---

## Example

```text
Input:
nums = [-1,0,1,2,-1,-4]

Output:
[
 [-1,-1,2],
 [-1,0,1]
]
```

---

# Brute Force Approach

## Idea

Try every possible combination of three numbers.

For each triplet,

```
Check whether

nums[i] + nums[j] + nums[k] == 0
```

If yes,

store it (avoiding duplicates).


---

# Duplicate Handling

## Duplicate i

```java
if(i > 0 && nums[i] == nums[i-1])
    continue;
```

Skip duplicate starting elements.

---

## Duplicate j

```java
while(j < k && nums[j] == nums[j-1])
    j++;
```

---

## Duplicate k

```java
while(j < k && nums[k] == nums[k+1])
    k--;
```

This guarantees

```
Only unique triplets
```

---

# Algorithm

```
Sort array

for every i

    Skip duplicate i

    j = i+1

    k = n-1

    while(j < k)

        sum = nums[i]+nums[j]+nums[k]

        if(sum < 0)

            j++

        else if(sum > 0)

            k--

        else

            Add triplet

            j++

            k--

            Skip duplicate j

            Skip duplicate k
```

# Java Solution

```java
import java.util.*;

class Solution {

    public List<List<Integer>> threeSum(int[] nums) {

        List<List<Integer>> res = new ArrayList<>();

        Arrays.sort(nums);

        for(int i = 0; i < nums.length - 2; i++) {

            if(i > 0 && nums[i] == nums[i - 1])
                continue;

            int j = i + 1;
            int k = nums.length - 1;

            while(j < k) {

                int total = nums[i] + nums[j] + nums[k];

                if(total > 0) {

                    k--;

                }
                else if(total < 0) {

                    j++;

                }
                else {

                    res.add(Arrays.asList(nums[i], nums[j], nums[k]));

                    j++;
                    k--;

                    while(j < k && nums[j] == nums[j - 1])
                        j++;

                    while(j < k && nums[k] == nums[k + 1])
                        k--;
                }
            }
        }

        return res;
    }
}
```
