## Problem Statement

Given a sorted array `nums` and a target value `target`:

Return the **first** and **last** position of the target.

If the target is not found, return:

```text
[-1, -1]
```

You must write an algorithm with **O(log n)** runtime complexity.

---

## Example 1

**Input**

```text
nums = [5,7,7,8,8,10]
target = 8
```

**Output**

```text
[3,4]
```

### Explanation

```text
Index : 0 1 2 3 4 5
Value : 5 7 7 8 8 10

First occurrence = 3
Last occurrence  = 4
```

class Solution {
    public int[] searchRange(int[] nums, int target) {

        int first = -1;
        int last = -1;

        for (int i = 0; i < nums.length; i++) {

            if (nums[i] == target) {

                if (first == -1) {
                    first = i;
                }

                last = i;
            }
        }

        return new int[]{first, last};
    }
}
