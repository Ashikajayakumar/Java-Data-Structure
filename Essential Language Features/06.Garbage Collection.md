# 🚀 Garbage Collection (GC) in Java

## What is Garbage Collection?

**Garbage Collection (GC)** is a JVM process that automatically removes unused objects from Heap Memory.

👉 Purpose: Free memory and avoid memory leaks.

---

## Simple Example

```java
class Student {
}

public class Main {
    public static void main(String[] args) {

        Student s = new Student();

        s = null;   // Object becomes unreachable
    }
}
```

### Memory Before

```text
STACK                    HEAP
-----                    -----
s ---------------------> Student Object
```

### Memory After

```text
STACK                    HEAP
-----                    -----
s = null                 Student Object
                          (No Reference)
```

Now the object is **eligible for Garbage Collection**.

---

# When Does an Object Become Eligible for GC?

## 1. Nullifying Reference

```java
Student s = new Student();
s = null;
```

Object has no reference.

✅ Eligible for GC

---

## 2. Reassigning Reference

```java
Student s = new Student();

s = new Student();
```

First object loses reference.

```text
Old Object  ← Eligible for GC
New Object  ← Active
```

---

## 3. Local Variable Goes Out of Scope

```java
public static void display() {
    Student s = new Student();
}
```

After method execution:

```text
display() ends
     ↓
Reference removed
     ↓
Object eligible for GC
```

---

## 4. Anonymous Object

```java
new Student();
```

No reference is stored.

Immediately eligible for GC.

---

# How GC Works

```text
Object Created
      ↓
Heap Memory
      ↓
Reference Removed
      ↓
Object Becomes Unreachable
      ↓
Garbage Collector Runs
      ↓
Memory Released
```

---

# Requesting Garbage Collection

You can request JVM to run GC.

```java
System.gc();
```

or

```java
Runtime.getRuntime().gc();
```

⚠️ JVM may ignore the request.

It is only a suggestion.

---

# Example

```java
public class Main {

    public static void main(String[] args) {

        Student s = new Student();

        s = null;

        System.gc();
    }
}
```

---

# finalize() Method

Before removing an object, JVM may call `finalize()`.

```java
class Student {

    protected void finalize() {
        System.out.println("Object Removed");
    }
}
```
---


➡️ JVM eventually removes it from Heap Memory.
