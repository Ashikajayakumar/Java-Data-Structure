String manipulation in Java means working with and modifying text using the `String` class and related classes like `StringBuilder` and `StringBuffer`.

## Common String Operations in Java

### 1. Creating Strings

```java
String str1 = "Hello";
String str2 = new String("World");
```

---

### 2. Concatenation (Joining Strings)

```java
String first = "Hello";
String second = "Java";

String result = first + " " + second;
System.out.println(result);
```

Output:

```text
Hello Java
```

Using `concat()`:

```java
String result = first.concat(second);
```

---

### 3. Finding Length

```java
String text = "Programming";
System.out.println(text.length());
```

Output:

```text
11
```

---

### 4. Accessing Characters

```java
String text = "Java";
System.out.println(text.charAt(1));
```

Output:

```text
a
```

---

### 5. Converting Case

```java
String text = "Java Programming";

System.out.println(text.toUpperCase());
System.out.println(text.toLowerCase());
```

---

### 6. Comparing Strings

```java
String a = "Java";
String b = "java";

System.out.println(a.equals(b));          // false
System.out.println(a.equalsIgnoreCase(b)); // true
```

---

### 7. Substring Extraction

```java
String text = "Programming";

System.out.println(text.substring(0, 6));
```

Output:

```text
Progra
```

---

### 8. Replacing Characters or Words

```java
String text = "I like Java";

String newText = text.replace("Java", "Python");

System.out.println(newText);
```

Output:

```text
I like Python
```

---

### 9. Searching in Strings

```java
String text = "Java Programming";

System.out.println(text.contains("Java"));
System.out.println(text.indexOf("Pro"));
```

---

### 10. Splitting Strings

```java
String text = "Apple,Banana,Mango";

String[] fruits = text.split(",");

for(String fruit : fruits){
    System.out.println(fruit);
}
```

---

### 11. Removing Extra Spaces

```java
String text = "   Java   ";

System.out.println(text.trim());
```

---

## Mutable Strings

### StringBuilder (Fast and Recommended)

```java
StringBuilder sb = new StringBuilder("Hello");

sb.append(" Java");
sb.insert(5, " World");
sb.replace(0, 5, "Hi");
sb.delete(2, 8);

System.out.println(sb);
```
