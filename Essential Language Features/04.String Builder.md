## What is StringBuilder?

`StringBuilder` is a class used to create and modify strings efficiently.

Unlike `String`, `StringBuilder` is **mutable** (changeable).

### String Example (Immutable)

```java
String str = "Java";
str = str + " Programming";
```

A new String object is created each time.

---

### StringBuilder Example (Mutable)

```java
StringBuilder sb = new StringBuilder("Java");
sb.append(" Programming");

System.out.println(sb);
```

Output:

```text
Java Programming
```

The same object is modified.

---

# Why Use StringBuilder?

When performing many string operations inside loops:

❌ String (Slow)

```java
String str = "";

for(int i = 1; i <= 5; i++) {
    str = str + i;
}
```

Creates multiple objects.

---

✅ StringBuilder (Fast)

```java
StringBuilder sb = new StringBuilder();

for(int i = 1; i <= 5; i++) {
    sb.append(i);
}

System.out.println(sb);
```

Output:

```text
12345
```

---

# Common Methods

## 1. append()

Adds data at the end.

```java
StringBuilder sb = new StringBuilder("Java");

sb.append(" Programming");

System.out.println(sb);
```

Output:

```text
Java Programming
```

---

## 2. insert()

Inserts data at a specific position.

```java
StringBuilder sb = new StringBuilder("Jva");

sb.insert(1, "a");

System.out.println(sb);
```

Output:

```text
Java
```

---

## 3. replace()

Replaces characters.

```java
StringBuilder sb = new StringBuilder("Java");

sb.replace(0, 4, "Python");

System.out.println(sb);
```

Output:

```text
Python
```

---

## 4. delete()

Deletes characters.

```java
StringBuilder sb = new StringBuilder("Java Programming");

sb.delete(4, 16);

System.out.println(sb);
```

Output:

```text
Java
```

---

## 5. reverse()

Reverses the string.

```java
StringBuilder sb = new StringBuilder("Java");

sb.reverse();

System.out.println(sb);
```

Output:

```text
avaJ
```

---

## 6. charAt()

Gets a character.

```java
StringBuilder sb = new StringBuilder("Java");

System.out.println(sb.charAt(2));
```

Output:

```text
v
```

---

## 7. length()

Returns length.

```java
StringBuilder sb = new StringBuilder("Java");

System.out.println(sb.length());
```

Output:

```text
4
```

---

## 8. setCharAt()

Changes a character.

```java
StringBuilder sb = new StringBuilder("Java");

sb.setCharAt(0, 'K');

System.out.println(sb);
```

Output:

```text
Kava
```

---

# Reverse a String Using StringBuilder

```java
public class Main {
    public static void main(String[] args) {

        String str = "Hello";

        String reversed =
                new StringBuilder(str)
                .reverse()
                .toString();

        System.out.println(reversed);
    }
}
```

Output:

```text
olleH
```

# String vs StringBuilder

| Feature      | String | StringBuilder |
| ------------ | ------ | ------------- |
| Mutable      | ❌ No   | ✅ Yes         |
| Memory Usage | More   | Less          |
| Performance  | Slower | Faster        |
| Thread Safe  | Yes    | No            |

---

### Difference between StringBuilder and StringBuffer?

| StringBuilder                             | StringBuffer                        |
| ----------------------------------------- | ----------------------------------- |
| Not Thread Safe                           | Thread Safe                         |
| Faster                                    | Slower                              |
| Preferred in single-threaded applications | Used in multi-threaded applications |
