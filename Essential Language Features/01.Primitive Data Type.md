## 📌 What are Primitive Data Types?

Primitive data types are the <b> **basic built-in data types**</b> provided by Java.

* They store **actual values**, not objects.
* They are faster and use less memory.
* Java has **8 primitive data types**.

---

# 📊 Types of Primitive Data Types

| Data Type | Size          | Default Value | Example             |
| --------- | ------------- | ------------- | ------------------- |
| byte      | 1 byte        | 0             | `byte a = 10;`      |
| short     | 2 bytes       | 0             | `short b = 100;`    |
| int       | 4 bytes       | 0             | `int c = 1000;`     |
| long      | 8 bytes       | 0L            | `long d = 10000L;`  |
| float     | 4 bytes       | 0.0f          | `float e = 10.5f;`  |
| double    | 8 bytes       | 0.0d          | `double f = 20.5;`  |
| char      | 2 bytes       | '\u0000'      | `char g = 'A';`     |
| boolean   | JVM dependent | false         | `boolean h = true;` |

---

# 1️⃣ byte

* Stores small integer values.
* Range: **-128 to 127**

### Example

```java
public class ByteDemo {
    public static void main(String[] args) {
        byte age = 25;
        System.out.println(age);
    }
}
```

---

# 2️⃣ short

* Used when values exceed byte range.
* Range: **-32,768 to 32,767**

### Example

```java
public class ShortDemo {
    public static void main(String[] args) {
        short marks = 500;
        System.out.println(marks);
    }
}
```

---

# 3️⃣ int

* Most commonly used integer type.
* Range: **-2³¹ to 2³¹-1**

### Example

```java
public class IntDemo {
    public static void main(String[] args) {
        int salary = 50000;
        System.out.println(salary);
    }
}
```

---

# 4️⃣ long

* Used for very large numbers.
* Must end with **L**

### Example

```java
public class LongDemo {
    public static void main(String[] args) {
        long population = 8000000000L;
        System.out.println(population);
    }
}
```

---

# 5️⃣ float

* Stores decimal numbers.
* Must end with **f**

### Example

```java
public class FloatDemo {
    public static void main(String[] args) {
        float price = 99.99f;
        System.out.println(price);
    }
}
```

---

# 6️⃣ double

* More precise than float.
* Default type for decimal values.

### Example

```java
public class DoubleDemo {
    public static void main(String[] args) {
        double pi = 3.14159;
        System.out.println(pi);
    }
}
```

---

# 7️⃣ char

* Stores a single character.
* Uses single quotes `' '`.

### Example

```java
public class CharDemo {
    public static void main(String[] args) {
        char grade = 'A';
        System.out.println(grade);
    }
}
```

---

# 8️⃣ boolean

* Stores true or false.

### Example

```java
public class BooleanDemo {
    public static void main(String[] args) {
        boolean isPassed = true;
        System.out.println(isPassed);
    }
}
```
