# 🔷 Remove Nth Node From End of List (LeetCode 19)
<img width="1024" height="1536" alt="image" src="https://github.com/user-attachments/assets/50b7e4a9-5fec-4682-9e30-be92423cf47f" />

## 📌 Problem Statement

Given the head of a linked list, remove the `nth` node from the end of the list and return its head.

---

## Example

### Input

```
head = [1,2,3,4,5]

n = 2
```

Linked List:

```
1 → 2 → 3 → 4 → 5
```

The 2nd node from the end is:

```
4
```

After removing:

```
1 → 2 → 3 → 5
```

### Output

```
[1,2,3,5]
```

---

# 💡 Approach

We use:

## Two Pointer Technique

Use two pointers:

```
slow

fast
```

The idea:

1. Move `fast` pointer `n + 1` steps ahead.
2. Move both pointers together.
3. When `fast` reaches the end:
   - `slow` will be just before the node to delete.
4. Remove the node.

---

# Why Use Dummy Node?

Consider removing the first node:

Example:

```
1 → 2 → 3

n = 3
```

The head itself must be deleted.

To handle this case easily, create:

```
dummy → head
```

Example:

```
dummy → 1 → 2 → 3 → 4 → 5
```

Now every deletion happens using:

```java
slow.next = slow.next.next;
```

---

# Algorithm

## Step 1️⃣: Create Dummy Node

```java
ListNode dummy = new ListNode(0);

dummy.next = head;
```

Initial:

```
dummy → 1 → 2 → 3 → 4 → 5
```

Initialize:

```
slow = dummy

fast = dummy
```

# Step 2️⃣: Move Fast Pointer n+1 Steps

Code:

```java
for(int i = 0; i <= n; i++){

    fast = fast.next;

}
```

For:

```
n = 2
```

Move:

```
n + 1 = 3 steps
```

# Step 4️⃣: Delete Node

```

        ↑
       slow
```

```java

```

```

```

After:

```
slow.next = 5
```

```

```

# ✅ Java Implementation

```java
class Solution {

    public ListNode removeNthFromEnd(ListNode head, int n) {


        ListNode dummy = new ListNode(0);

        dummy.next = head;


        ListNode fast = dummy;

        ListNode slow = dummy;



        // Move fast n+1 steps ahead

        for(int i = 0; i <= n; i++){

            fast = fast.next;

        }



        // Move both pointers

        while(fast != null){

            fast = fast.next;

            slow = slow.next;

        }



        // Remove nth node

        slow.next = slow.next.next;



        return dummy.next;
    }
}
```

# 🧠 Code Explanation

## Dummy Node Creation

```java
ListNode dummy = new ListNode(0);

dummy.next = head;
```

Creates:

```
dummy → original list
```

Helps handle deleting the head node.

---

## Fast Pointer Movement

```java
for(int i=0;i<=n;i++)
{
    fast = fast.next;
}
```

Creates a gap of:

```
n nodes
```

between slow and fast.

---

## Move Together

```java
while(fast != null)
{
    fast = fast.next;
    slow = slow.next;
}
```

When fast reaches the end:

```
slow.next
```

is the node to remove.

---

## Delete Node

```java
slow.next = slow.next.next;
```

Skips the target node.
