# 🔷 Linked List Cycle II (LeetCode 142)

## 📌 Problem Statement

Given the head of a linked list, return the node where the cycle begins.

If there is no cycle, return:

```
null
```

---

## 🧩 Example

### Input

```
3 → 2 → 0 → -4
    ↑       |
    |_______|
```

Cycle starts at:

```
Node value = 2
```

### Output

```
2
```

---

# 💡 Approach

We use:

## Floyd's Cycle Detection Algorithm

Also known as:

```
Slow and Fast Pointer Algorithm
```

It works in two phases:

---

# Phase 1️⃣: Detect Cycle

Use two pointers:

| Pointer | Movement |
|---------|----------|
| slow | moves 1 step |
| fast | moves 2 steps |

Example:

```
slow = slow.next

fast = fast.next.next
```

### Condition

If:

```
slow == fast
```

then a cycle exists.

---

# Phase 2️⃣: Find Cycle Starting Node

After finding the meeting point:

1. Move `slow` back to `head`.
2. Keep `fast` at the meeting point.
3. Move both pointers one step at a time.

The point where they meet again is the starting node of the cycle.

---

# ✅ Java Code

```java
public class Solution {

    public ListNode detectCycle(ListNode head) {

        if(head == null || head.next == null)
            return null;


        ListNode slow = head;
        ListNode fast = head;


        // Phase 1: Detect Cycle

        while(fast != null && fast.next != null){

            slow = slow.next;
            fast = fast.next.next;


            if(slow == fast){

                // Cycle found

                slow = head;


                // Phase 2: Find starting node

                while(slow != fast){

                    slow = slow.next;
                    fast = fast.next;

                }

                return slow;
            }
        }


        // No cycle

        return null;
    }
}
```

# 🔍 Dry Run

## Linked List

```
3 → 2 → 0 → -4
    ↑       |
    |_______|

```

Cycle starts at:


---

# Phase 1: Detect Cycle

Initial:

```
slow = 3
fast = 3
```

---

## Iteration 1

Move pointers:

```
slow = 2

fast = 0
```

Position:

```
3 → 2 → 0 → -4
    ↑    ↑
   slow fast
```

---

## Iteration 2

Move pointers:

```
slow = 0

fast = 2
```

Position:

```
3 → 2 → 0 → -4
    ↑       ↑
   fast    slow
```

---

## Iteration 3

Move pointers:

```
slow = -4

fast = 0
```

Position:

```
3 → 2 → 0 → -4
        ↑   ↑
       fast slow
```

---

## Iteration 4

Move pointers:

```
slow = 2

fast = 2
```

Now:

```
slow == fast
```

Cycle detected ✅

---

# Phase 2: Find Starting Node

Move slow back to head:

```
slow = 3
fast = 2
```

Now move both one step.

---

## Iteration 1

```
slow = 2

fast = 0
```

---

## Iteration 2

```
slow = 0

fast = -4
```

---

## Iteration 3

```
slow = -4

fast = 2
```

---

## Iteration 4

```
slow = 2

fast = 2
```

Meeting point:

```
2
```

Therefore:

```
Cycle starts at node 2
```
