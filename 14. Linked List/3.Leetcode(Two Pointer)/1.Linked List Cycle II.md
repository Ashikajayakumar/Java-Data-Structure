# 🔷 Linked List Cycle II

## Problem Statement

Given the head of a linked list, return the node where the cycle begins.

If there is no cycle, return `null`.

---

## Example

### Input

```

head = 3 → 2 → 0 → -4
↑       |
|_______|

Cycle starts at node with value = 2

```

### Output

```

Node value = 2

```

---

# Approach

We use **Floyd's Cycle Detection Algorithm** (Slow and Fast Pointer).

It has two steps:

## Step 1️⃣ Detect Whether Cycle Exists

Use two pointers:

```

slow → moves one step
fast → moves two steps

```

If slow and fast meet:

```

slow == fast

````

then a cycle exists.

---

## Step 2️⃣ Find Starting Node of Cycle

After detecting the cycle:

Move slow pointer back to head.

Keep fast pointer at meeting point.

Move both one step at a time.

The point where they meet again is the cycle starting node.

---

# Code

```java
class Solution {
    public ListNode detectCycle(ListNode head) {

        if(head == null || head.next == null)
            return null;

        ListNode slow = head;
        ListNode fast = head;

        boolean hasCycle = false;


        // Step 1: Detect Cycle

        while(fast != null && fast.next != null){

            slow = slow.next;
            fast = fast.next.next;


            if(slow == fast){
                hasCycle = true;
                break;
            }
        }


        // No Cycle

        if(!hasCycle)
            return null;



        // Step 2: Find Cycle Starting Node

        slow = head;


        while(slow != fast){

            slow = slow.next;
            fast = fast.next;

        }


        return slow;
    }
}
````

# Dry Run

## Given Linked List

```
3 → 2 → 0 → -4
    ↑         |
    |_________|

Cycle starts at:

---

# Step 1: Detect Cycle

Initial:

```
slow
 ↓
3 → 2 → 0 → -4
          ↑
         fast
```

---

## Iteration 1

```
slow = slow.next
slow = 2


fast = fast.next.next
fast = 0

```

Position:

```
3 → 2 → 0 → -4
    ↑    ↑
   slow fast

```

---

## Iteration 2

```
slow = slow.next

slow = 0


fast = fast.next.next

fast = 2

```

Position:

```
3 → 2 → 0 → -4
    ↑       ↑
   fast    slow

```

---

## Iteration 3

```
slow = slow.next

slow = -4


fast = fast.next.next

fast = 0

```

Position:

```
3 → 2 → 0 → -4
        ↑    ↑
       fast slow

```

---

## Iteration 4

```
slow = slow.next

slow = 2


fast = fast.next.next

fast = 2

```

Now:

```
slow == fast
```

Cycle detected ✅

---

# Step 2: Find Cycle Starting Point

Move slow to head:

```
slow = head
```

Now:

```
slow
 ↓
3 → 2 → 0 → -4
          ↑
         fast

```

Move both one step:

---

Iteration 1

```
slow = 2

fast = -4

```

Position:

```
3 → 2 → 0 → -4
    ↑       ↑
   slow    fast

```

---

Iteration 2

```
slow = 0

fast = 2

```

---

Iteration 3

```
slow = -4

fast = 0

```

---

Iteration 4

```
slow = 2

fast = 2

```

They meet at:

```
2
```

Therefore:

```
Cycle starts at node 2
