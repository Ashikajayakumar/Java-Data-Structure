```md
# 🔷 Intersection of Two Linked Lists

## Problem Statement

Given the heads of two singly linked lists `headA` and `headB`, return the node at which the two linked lists intersect.

If the two linked lists do not intersect, return `null`.

---

## Example

### Input

```

headA = [4,1,8,4,5]

headB = [5,6,1,8,4,5]

```

Visual Representation:

```

List A:

4 → 1

8 → 4 → 5
/
5 → 6 → 1

List B:

5 → 6 → 1

8 → 4 → 5

```

The two linked lists share the same node:

```

8 → 4 → 5

```

---

## Output

```

Intersection Node = 8

```

---

# Approach

## Two Pointer Technique

Use two pointers:

```

ptrA → starts from headA

ptrB → starts from headB

```

### Idea

Both pointers travel the same distance.

When one pointer reaches the end:

- Move it to the other linked list head.
- Continue moving.

After switching:

```

Distance travelled by ptrA = Distance travelled by ptrB

```

Therefore, they meet at the intersection node.

---

# Algorithm

1. Initialize:

```

ptrA = headA
ptrB = headB

```

2. Traverse both lists.

3. If `ptrA` reaches null:

```

ptrA = headB

```

4. If `ptrB` reaches null:

```

ptrB = headA

```

5. When:

```

ptrA == ptrB

```

return that node.

---
```

````
# Java Implementation

```java
public class Solution {

    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {

        if(headA == null || headB == null){
            return null;
        }


        ListNode ptrA = headA;
        ListNode ptrB = headB;


        while(ptrA != ptrB){

            if(ptrA != null){
                ptrA = ptrA.next;
            }
            else{
                ptrA = headB;
            }


            if(ptrB != null){
                ptrB = ptrB.next;
            }
            else{
                ptrB = headA;
            }
        }


        return ptrA;
    }
}
````

```
null
```

```java

    ptrB = ptrB.next;
}
else{
    ptrB = headA;
}
```

If pointer B reaches the end:

## Meeting Condition

```java
while(ptrA != ptrB)
```

When both pointers point to the same node:

```
ptrA == ptrB
```

The loop stops.

That node is the intersection point.

# Why This Works?

Example:

Length of List A:

```
A = a + c
```

Length of List B:

```
B = b + c
```

Where:

```
c = common part length
```

Pointer A travels:

```
a + c + b
```

Pointer B travels:

```
b + c + a
```

Both travel equal distance:

```
a + b + c
```

So they meet at the intersection node.
