# 🔷 Intersection of Two Linked Lists (LeetCode 160)

## 📌 Problem Statement

Given the heads of two singly linked lists `headA` and `headB`, return the node at which the two linked lists intersect.

If the two linked lists do not intersect, return:

```
null
```

---

# 🧩 Example

## Input

```
headA = [4,1,8,4,5]

headB = [5,6,1,8,4,5]
```

### Visual Representation

```
List A:

4 → 1
     \
      8 → 4 → 5
     /
5 → 6 → 1


List B:

5 → 6 → 1
          \
           8 → 4 → 5
```

The common part is:

```
8 → 4 → 5
```

Therefore:

```
Intersection Node = 8
```

---

# 💡 Approach

## Two Pointer Technique

Use two pointers:

```
ptrA → starts from headA

ptrB → starts from headB
```

---

## Idea

Both pointers travel through both linked lists.

When one pointer reaches the end:

```
null
```

move it to the beginning of the other list.

Example:

```
ptrA reaches end of List A

ptrA = headB
```

and:

```
ptrB reaches end of List B

ptrB = headA
```

After switching:

```
Distance travelled by ptrA = Distance travelled by ptrB
```

So they meet at the intersection node.

---

# 🔥 Algorithm

### Step 1

Initialize two pointers:

```
ptrA = headA

ptrB = headB
```

---

### Step 2

Traverse both linked lists.

Move:

```
ptrA = ptrA.next

ptrB = ptrB.next
```

---

### Step 3

If pointer A reaches the end:

```
ptrA = headB
```

---

### Step 4

If pointer B reaches the end:

```
ptrB = headA
```

---

### Step 5

When:

```
ptrA == ptrB
```

return that node.

# Pointer Movement


| Step | ptrA | ptrB |
|---|---|---|
| 1 | 4 | 5 |
| 2 | 1 | 6 |
| 3 | 8 | 1 |
| 4 | 4 | 8 |
| 5 | 5 | 4 |
| 6 | null → headB | 5 |
| 7 | 5 | 6 |
| 8 | 6 | 1 |
| 9 | 1 | 8 |
| 10 | 8 | 4 |
| 11 | 4 | 5 |
| 12 | 5 | null → headA |
| 13 | 8 | 8 |

---

At step 13:

```
ptrA == ptrB
```

Both point to:

```
8
```

Therefore:

```
Intersection Node = 8
```

# ✅ Java Implementation

```java
public class Solution {

    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {


        if(headA == null || headB == null){
            return null;
        }


        ListNode ptrA = headA;
        ListNode ptrB = headB;


        while(ptrA != ptrB){


            if(ptrA != null){

                ptrA = ptrA.next;

            }
            else{

                ptrA = headB;

            }



            if(ptrB != null){

                ptrB = ptrB.next;

            }
            else{

                ptrB = headA;

            }

        }


        return ptrA;
    }
}
```
