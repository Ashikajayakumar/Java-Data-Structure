# 🔷 Swap Nodes in Pairs (LeetCode 24)

## 📌 Problem Statement

Given a linked list, swap every two adjacent nodes and return the modified linked list.

You must solve the problem without changing the values inside the nodes.

Only node links should be modified.

---

## Example

### Input

```
head = [1,2,3,4]
```

Linked List:

```
1 → 2 → 3 → 4
```

Swap every pair:

```
(1,2) → (3,4)
```

After swapping:

```
2 → 1 → 4 → 3
```

### Output

```
[2,1,4,3]
```

---

# 💡 Approach

We use:

## Dummy Node + Pointer Manipulation

The idea:

1. Create a dummy node before the head.
2. Take two nodes at a time.
3. Swap their connections.
4. Move pointer to the next pair.

---

# Why Use Dummy Node?

Without dummy node:

Example:

```
1 → 2 → 3 → 4
```

The head changes after swapping:

```
2 → 1 → 4 → 3
```

Using:

```
dummy → 1 → 2 → 3 → 4
```

allows easy modification of the head.


# Step 3️⃣: Store Two Nodes

```java

ListNode second = curr.next.next;

```

```
          ↑    ↑
        first second
```

# Step 4️⃣: Swap Nodes

## Before Swapping

```
curr → first → second → next
```

Example:

```
dummy → 1 → 2 → 3 → 4
```

---

## 1. Connect first to next node

```java

```

Result:

```
1 → 3
```

---

## 2. Connect second to first

```java

```

Result:

```
2 → 1
```

---

## 3. Connect previous node to second

```java
curr.next = second;
```

Result:

```

```

---

## 4. Move curr forward

```java

```

```

```

# ✅ Java Implementation

```java
class Solution {

    public ListNode swapPairs(ListNode head) {


        ListNode dummy = new ListNode(0);

        dummy.next = head;


        ListNode curr = dummy;



        while(curr.next != null && curr.next.next != null){


            ListNode first = curr.next;

            ListNode second = curr.next.next;



            // Swap nodes

            first.next = second.next;

            second.next = first;

            curr.next = second;



            // Move to next pair

            curr = first;

        }


        return dummy.next;
    }
}
```

# 🧠 Code Explanation

## First Node

```java

```

Stores the first node of the pair.

Example:

```

```

---

## Second Node

```java

```

Stores the second node.

Example:

```

```

## Reverse Pair

Before:

```

```

```

```

---

## Move Pointer

```java
curr = first;
```

```
dummy → 2 → 1 → 3 → 4

              ↑
             curr
```

Moves to the end of the swapped pair.
