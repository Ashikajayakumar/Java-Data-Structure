## рЯУМ Problem Statement

Given the head of a linked list, reverse the nodes of the list `k` at a time and return the modified list.

If the number of nodes is not a multiple of `k`, the remaining nodes should stay unchanged.

---

## Example

### Input

```
head = [1,2,3,4,5]

k = 2
```

Linked List:

```
1 вЖТ 2 вЖТ 3 вЖТ 4 вЖТ 5
```

Reverse every 2 nodes:

```
(1,2) вЖТ (3,4)
```

After reversing:

```
2 вЖТ 1 вЖТ 4 вЖТ 3 вЖТ 5
```

### Output

```
[2,1,4,3,5]
```

## Step 1пЄПвГ£: Check K Nodes Availability

```

```

Check first two nodes exist.

If fewer than `k` nodes remain:

```

```

# Step 2пЄПвГ£: Reverse K Nodes

Example:

Before:

```

```

Use three pointers:

After reversing:


The old head becomes the tail:

```

```

# вЬЕ Java Implementation

```java
class Solution {

    public ListNode reverseKGroup(ListNode head, int k) {


        if(head == null){

            return null;

        }


        // Check if k nodes exist

        ListNode temp = head;

        int count = 0;


        while(temp != null && count < k){

            temp = temp.next;

            count++;

        }


        // Remaining nodes less than k

        if(count < k){

            return head;

        }



        // Reverse k nodes

        ListNode prev = null;

        ListNode current = head;

        ListNode next = null;


        count = 0;


        while(current != null && count < k){


            next = current.next;

            current.next = prev;

            prev = current;

            current = next;

            count++;

        }


        // Recursively reverse remaining groups

        head.next = reverseKGroup(next, k);



        return prev;
    }
}
```

becomes the end of the reversed group.

It connects to the next processed group.
