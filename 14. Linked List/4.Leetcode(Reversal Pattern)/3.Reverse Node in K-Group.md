# 🔷 Reverse Nodes in K-Group (LeetCode 25)

## 📌 Problem Statement

Given the head of a linked list, reverse the nodes of the list `k` at a time and return the modified list.

If the number of nodes is not a multiple of `k`, the remaining nodes should stay unchanged.

---

## Example

### Input

```
head = [1,2,3,4,5]

k = 2
```

Linked List:

```
1 → 2 → 3 → 4 → 5
```

Reverse every 2 nodes:

```
(1,2) → (3,4)
```

After reversing:

```
2 → 1 → 4 → 3 → 5
```

### Output

```
[2,1,4,3,5]
```



# Algorithm

## Step 1️⃣: Check K Nodes Availability

Before reversing:

```
1 → 2 → 3 → 4 → 5
```

For:

```
k = 2
```

Check first two nodes exist.

If fewer than `k` nodes remain:

```
return head
```

because they should not be reversed.

---

# Step 2️⃣: Reverse K Nodes

Example:

Before:

```
1 → 2 → 3
```

Reverse first `k = 2` nodes:

```
2 → 1 → 3
```

Use three pointers:

```
prev

current

next
```

---

# Step 3️⃣: Connect Remaining List

After reversing:

```
2 → 1
```

The old head becomes the tail:

```
1.next
```

should point to the next reversed group.

# ✅ Java Implementation

```java
class Solution {

    public ListNode reverseKGroup(ListNode head, int k) {


        if(head == null){

            return null;

        }


        // Check if k nodes exist

        ListNode temp = head;

        int count = 0;


        while(temp != null && count < k){

            temp = temp.next;

            count++;

        }


        // Remaining nodes less than k

        if(count < k){

            return head;

        }



        // Reverse k nodes

        ListNode prev = null;

        ListNode current = head;

        ListNode next = null;


        count = 0;


        while(current != null && count < k){


            next = current.next;

            current.next = prev;

            prev = current;

            current = next;

            count++;

        }


        // Recursively reverse remaining groups

        head.next = reverseKGroup(next, k);



        return prev;
    }
}
```

---

# 🧠 Code Explanation

## Check K Nodes

```java
while(temp != null && count < k)
```

Counts whether enough nodes exist for reversal.

---

## Reverse Current Group

```java
current.next = prev;
```

Changes the direction of links.

Example:

Before:

```
1 → 2
```

After:

```
2 → 1
```

---

## Connect Next Group

```java
head.next = reverseKGroup(next,k);
```

After reversing:

```
head
```

becomes the end of the reversed group.

It connects to the next processed group.
