## 📌 Problem Statement

Given the head of a linked list, rotate the list to the right by `k` places.

Return the new head of the rotated linked list.

---

## Example

### Input

```
head = [1,2,3,4,5]

k = 2
```

Linked List:

```
1 → 2 → 3 → 4 → 5
```

Rotate right by **2**:

```
4 → 5 → 1 → 2 → 3
```

### Output

```
[4,5,1,2,3]
```



# 🧠 Step-by-Step Explanation

## 🔹 Step 1: Handle Edge Case

If the list is empty, nothing needs to be rotated.

```java
if(head == null)
    return head;
```

---

## 🔹 Step 2: Find Length and Last Node

Initialize:

```java
int length = 1;

ListNode dummy = head;
```

Traverse until the last node.

```java
while(dummy.next != null){

    dummy = dummy.next;

    length++;

}
```

---

### Dry Run

Initial:

```
1 → 2 → 3 → 4 → 5
```

After traversal:

```
length = 5

dummy → 5
```

Here,

```
dummy
```

points to the last node.

---

## 🔹 Step 3: Reduce k Using Modulo

Sometimes `k` is larger than the length.

Example:

```
length = 5

k = 7
```

Instead of rotating **7** times:

```java
position = k % length;
```

Calculation:

```
7 % 5 = 2
```

So rotating **7** times is the same as rotating **2** times.

Code:

```java
int position = k % length;
```

If:

```
position == 0
```

the list remains unchanged.

```java
if(position == 0)
    return head;
```

---

## 🔹 Step 4: Find the New Tail

The new tail is located at:

```
length - position - 1
```

For:

```
length = 5

position = 2
```

Calculation:

```
5 - 2 - 1 = 2
```

Move **2** steps from the head.

```java
ListNode current = head;

for(int i = 0; i < length - position - 1; i++){

    current = current.next;

}
```

---

### Dry Run

Initial:

```
1 → 2 → 3 → 4 → 5
↑
current
```

Move:

```
Step 1

current → 2
```

```
Step 2

current → 3
```

Now:

```
1 → 2 → 3 → 4 → 5
        ↑
      current
```

Current is the **new tail**.

---

## 🔹 Step 5: Break and Rotate

The new head will be:

```java
ListNode newHead = current.next;
```

So:

```
newHead = 4
```

Break the list:

```java
current.next = null;
```

Result:

```
1 → 2 → 3 → null

4 → 5
```

Now connect the old tail with the old head.

```java
dummy.next = head;
```

Result:

```
4 → 5 → 1 → 2 → 3
```

---

## 🔹 Step 6: Return New Head

```java
return newHead;
```

New head:

```
4
```

# ✅ Java Implementation

```java
class Solution {

    public ListNode rotateRight(ListNode head, int k) {

        if(head == null)
            return head;


        int length = 1;

        ListNode dummy = head;


        // Find length and last node

        while(dummy.next != null){

            dummy = dummy.next;

            length++;

        }


        // Reduce k

        int position = k % length;

        if(position == 0)
            return head;


        // Find new tail

        ListNode current = head;

        for(int i = 0; i < length - position - 1; i++){

            current = current.next;

        }


        // Rotate

        ListNode newHead = current.next;

        current.next = null;

        dummy.next = head;


        return newHead;
    }
}
```
