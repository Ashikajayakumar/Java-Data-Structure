# 🔷 Reverse Linked List II (LeetCode 92)

## 📌 Problem Statement

Given the head of a singly linked list and two positions `left` and `right`, reverse the nodes of the linked list from position `left` to position `right`.

Return the modified linked list.

---

## Example

### Input

```
head = [1,2,3,4,5]

left = 2

right = 4
```

Linked List:

```
1 → 2 → 3 → 4 → 5
```

Reverse nodes between position `2` and `4`:

```
2 → 3 → 4
```

After reversing:

```
4 → 3 → 2
```

### Output

```
[1,4,3,2,5]
```

---

# 💡 Approach

We use:

## In-place Linked List Reversal

The idea:

1. Create a dummy node.
2. Move `prev` pointer to the node before `left`.
3. Reverse nodes between `left` and `right`.
4. Connect the reversed part back to the original list.

---

# Why Use Dummy Node?

Example:

```
1 → 2 → 3
```

If:

```
left = 1
```

the head itself needs to change.

Using:

```
dummy → 1 → 2 → 3
```

makes the reversal logic simpler.

---

# Algorithm

## Step 1️⃣: Create Dummy Node

```java
ListNode dummy = new ListNode(0);

dummy.next = head;
```

Initial:

```
dummy → 1 → 2 → 3 → 4 → 5
```

Initialize:

```
prev = dummy
```

---

# Step 2️⃣: Move prev to (left - 1)

Given:

```
left = 2
```

Move:

```
left - 1 = 1 step
```

Code:

```java
for(int i=1; i<left; i++){

    prev = prev.next;

}
```

After moving:

```
dummy → 1 → 2 → 3 → 4 → 5
        ↑
       prev
```

Now:

```
curr = prev.next
```

So:

```
curr = 2
```

---

# Step 3️⃣: Reverse Nodes Between left and right

We need to reverse:

```
2 → 3 → 4
```

Number of reversals:

```
right - left
```

For:

```
left = 2

right = 4
```

Iterations:

```
4 - 2 = 2
```

---

# 🔁 Iteration 1

Before:

```
prev → 1

curr → 2


1 → 2 → 3 → 4 → 5
```

Store next node:

```java
next = curr.next;
```

Now:

```
next → 3
```

Remove next node and insert after prev:

```java
curr.next = next.next;

next.next = prev.next;

prev.next = next;
```

Result:

```
1 → 3 → 2 → 4 → 5
```

---

# 🔁 Iteration 2

Current:

```
1 → 3 → 2 → 4 → 5
```

Store:

```java
next = curr.next;
```

Now:

```
next → 4
```

Re-link:

```java
curr.next = next.next;

next.next = prev.next;

prev.next = next;
```

Result:

```
1 → 4 → 3 → 2 → 5
```

---

# ✅ Final Linked List

```
1 → 4 → 3 → 2 → 5
```

---

# ✅ Java Implementation

```java
class Solution {

    public ListNode reverseBetween(ListNode head, int left, int right) {


        if(head == null || left == right){

            return head;

        }


        ListNode dummy = new ListNode(0);

        dummy.next = head;


        ListNode prev = dummy;


        // Move prev to node before left

        for(int i = 1; i < left; i++){

            prev = prev.next;

        }


        ListNode curr = prev.next;


        ListNode next = null;



        // Reverse nodes between left and right

        for(int i = 0; i < right - left; i++){


            next = curr.next;


            curr.next = next.next;


            next.next = prev.next;


            prev.next = next;


        }


        return dummy.next;
    }
}
```

---

# 🧠 Code Explanation

## Finding Previous Node

```java
prev = prev.next;
```

Moves `prev` before the reversal section.

Example:

```
dummy → 1 → 2 → 3 → 4 → 5

        ↑
       prev
```

---

## Current Node

```java
curr = prev.next;
```

`curr` is the first node that needs reversal.

Example:

```
curr = 2
```

---

## Store Next Node

```java
next = curr.next;
```

Save the next node before changing links.

Example:

```
next = 3
```

---

## Move curr Forward

```java
curr.next = next.next;
```

Removes `next` from its original position.

Before:

```
2 → 3 → 4
```

After:

```
2 → 4
```

---

## Insert next Before curr

```java
next.next = prev.next;

prev.next = next;
```

Moves the node to the front of the reversed section.

Before:

```
1 → 2 → 4
```

After:

```
1 → 3 → 2 → 4
```
