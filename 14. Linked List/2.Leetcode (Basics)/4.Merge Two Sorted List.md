# 🔷 Merge Two Sorted Lists

## Problem Statement

You are given the heads of two sorted linked lists:

```
l1
```

and

```
l2
```

Merge the two lists into one sorted linked list and return its head.

---

## Example

### Input

```
l1 = 1 → 2 → 4 → NULL

l2 = 1 → 3 → 4 → NULL
```

### Output

```
1 → 1 → 2 → 3 → 4 → 4 → NULL
```

---

# Approach

We use:

1. **Dummy Node**
2. **Current Pointer**

### Why Dummy Node?

Without dummy node, we need special handling for the first node.

Dummy node gives us a fixed starting point.

```
dummy → NULL
```

After merging:

```
return dummy.next
```

because the actual list starts after dummy.

---

# Initial State

Lists:

```
l1:

1 → 2 → 4 → NULL


l2:

1 → 3 → 4 → NULL
```

Create dummy node:

```
dummy
 ↓
0 → NULL
```

Current pointer:

```
current = dummy
```

---

# Step-by-Step Dry Run

## Iteration 1

Compare:

```
l1.val = 1

l2.val = 1
```

Condition:

```java
if(l1.val < l2.val)
```

```
1 < 1 ❌
```

Take node from `l2`.

```
current.next = l2
```

Move l2:

```
l2 = l2.next
```

Current:

```
dummy → 1
          ↑
       current
```

l2:

```
3 → 4 → NULL
```

l1:

```
1 → 2 → 4 → NULL
```

```

```

```

                            ↑
                         current
```

---

# Remaining Nodes

Loop condition:

```java
while(l1 != null && l2 != null)
```

fails because:

```
l2 == NULL
```

Remaining:

```
l1 = 4 → NULL
```

Attach remaining list:

```java
current.next = l1;
```

Final:

```
dummy → 1 → 1 → 2 → 3 → 4 → 4 → NULL
```

---

# Return Answer

```java
return dummy.next;
```

Skip dummy node.

Final Output:

```
1 → 1 → 2 → 3 → 4 → 4 → NULL
```

---

# Java Implementation

```java
class Solution {
    public ListNode mergeTwoLists(ListNode l1, ListNode l2) {

        ListNode dummy = new ListNode(0);

        ListNode current = dummy;


        while(l1 != null && l2 != null){

            if(l1.val < l2.val){

                current.next = l1;

                l1 = l1.next;

            }
            else{

                current.next = l2;

                l2 = l2.next;

            }

            current = current.next;
        }


        if(l1 != null){

            current.next = l1;

        }
        else{

            current.next = l2;

        }


        return dummy.next;
    }
}
```
