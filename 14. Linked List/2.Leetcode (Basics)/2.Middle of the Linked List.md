# 🔷 Middle of the Linked List

## Problem Statement

Given the head of a singly linked list, return the **middle node** of the linked list.

If there are two middle nodes (even length), return the **second middle node**.

---

## Example

### Input

```
1 → 2 → 3 → 4 → 5 → NULL
```

### Output

```
3
```

Explanation:

The middle node of the linked list is `3`.

---

# Approach

We use the **Slow and Fast Pointer Technique**.

### Two Pointers:

| Pointer | Movement |
|---|---|
| slow | moves one step |
| fast | moves two steps |

When `fast` reaches the end of the linked list, `slow` will be at the middle node.

---

# Algorithm

1. Initialize both pointers at head.

```
slow = head
fast = head
```

2. Move pointers:

```
slow = slow.next

fast = fast.next.next
```

3. When fast reaches the end:

```
slow = middle node
```

4. Return slow.

# Java Implementation

```java
class Solution {
    public ListNode middleNode(ListNode head) {

        if(head == null)
            return null;

        ListNode slow = head;
        ListNode fast = head;

        while(fast != null && fast.next != null){

            fast = fast.next.next;
            slow = slow.next;

        }

        return slow;
    }
}
```
