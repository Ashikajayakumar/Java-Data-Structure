# 🔷 Reverse Linked List

## Problem Statement

Given the head of a singly linked list, reverse the linked list and return the new head.

### Example

Input:

```
head → 1 → 2 → 3 → 4 → NULL
```

Output:

```
head → 4 → 3 → 2 → 1 → NULL
```

---

# Approach

We use **3 pointers**:

1. `prev` → stores previous node
2. `current` → current node we are processing
3. `next` → stores next node before breaking the link


Initial State:

```
prev = NULL
current = head
next = NULL
```

Linked List:

```
NULL ← 1 → 2 → 3 → 4 → NULL
       ↑
    current
```

---

# Algorithm

1. Store the next node.

```
next = current.next;
```

2. Reverse the current node link.

```
current.next = prev;
```

3. Move `prev` forward.

```
prev = current;
```

4. Move `current` forward.

```
current = next;
```

Repeat until:

```
current == NULL
```

# Stop Condition

Loop stops because:

```
current == NULL
```

Now:

```
head = prev
```

Return:

```
4 → 3 → 2 → 1 → NULL
```

# Java Implementation

```java
class Solution {
    public ListNode reverseList(ListNode head) {

        ListNode prev = null;
        ListNode current = head;
        ListNode next = null;

        while(current != null){

            // Store next node
            next = current.next;

            // Reverse current node link
            current.next = prev;

            // Move prev forward
            prev = current;

            // Move current forward
            current = next;
        }

        head = prev;

        return head;
    }
}
```
