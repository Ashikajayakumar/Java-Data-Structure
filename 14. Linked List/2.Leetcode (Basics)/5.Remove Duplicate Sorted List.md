# üî∑ Remove Duplicates from Sorted List

## Problem Statement

Given the head of a **sorted linked list**, remove all duplicate elements so that each value appears only once.

Return the head of the modified linked list.

---

## Example

### Input

```
head = 1 ‚Üí 1 ‚Üí 2 ‚Üí 3 ‚Üí 3 ‚Üí NULL
```

### Output

```
1 ‚Üí 2 ‚Üí 3 ‚Üí NULL
```

---

# Approach

Since the linked list is already **sorted**, duplicate values will always be next to each other.

We use one pointer:

```
current
```

### Logic:

Compare:

```
current.val
```

with

```
current.next.val
```

If both are equal:

```
current.next = current.next.next
```

Remove duplicate node.

Otherwise:

```
current = current.next
```

Move to next node.

---

# Algorithm

1. Initialize:

```java
current = head;
```

2. Traverse the linked list:

```java
while(current != null && current.next != null)
```

3. Compare current node with next node.

### Duplicate found:

```
current.val == current.next.val
```

Remove next node:

```java
current.next = current.next.next;
```

### Different values:

Move forward:

```java
current = current.next;
```

4. Return head.

# Java Implementation

```java
class Solution {
    public ListNode deleteDuplicates(ListNode head) {

        ListNode current = head;

        while(current != null && current.next != null){

            if(current.val == current.next.val){

                current.next = current.next.next;

            }
            else{

                current = current.next;

            }
        }

        return head;
    }
}
```
