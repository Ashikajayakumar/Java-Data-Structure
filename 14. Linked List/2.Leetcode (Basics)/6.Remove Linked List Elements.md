# ЁЯФ╖ Remove Linked List Elements

## Problem Statement

Given the head of a linked list and an integer `val`, remove all nodes from the linked list whose value is equal to `val`.

Return the head of the modified linked list.

---

## Example

### Input

```
head = 1 тЖТ 2 тЖТ 6 тЖТ 3 тЖТ 4 тЖТ 5 тЖТ 6 тЖТ NULL

val = 6
```

### Output

```
1 тЖТ 2 тЖТ 3 тЖТ 4 тЖТ 5 тЖТ NULL
```

---

# Approach

We use a **Dummy Node Technique**.

### Why Dummy Node?

If the first node itself has the value to remove, handling becomes difficult.

Example:

```
head = 6 тЖТ 1 тЖТ 2
```

If we remove `6`, head needs to change.

Dummy node avoids this special case.

---

# Dummy Node Creation

Code:

```java
ListNode dummy = new ListNode(0);
dummy.next = head;
```

Initial structure:

```
dummy
  тЖУ
 0 тЖТ 1 тЖТ 2 тЖТ 6 тЖТ 3 тЖТ 4 тЖТ NULL
```

We use:

```
current = dummy
```

---

# Algorithm

1. Create dummy node.
2. Point dummy to head.
3. Traverse using `current`.
4. Check the next node value.

### If next node contains target value:

Remove it:

```java
current.next = current.next.next;
```

### Otherwise:

Move forward:

```java
current = current.next;
```

5. Return:

```java
dummy.next
```

# Java Implementation

```java
class Solution {

    public ListNode removeElements(ListNode head, int val) {

        ListNode dummy = new ListNode(0);

        dummy.next = head;

        ListNode current = dummy;


        while(current.next != null){

            if(current.next.val == val){

                current.next = current.next.next;

            }
            else{

                current = current.next;

            }
        }

        return dummy.next;
    }
}
```
